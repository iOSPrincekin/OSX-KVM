Intel-आधारित Mac पर macOS रिकवरी का उपयोग करें

macOS रिकवरी आपके Mac का बिल्टइन रिकवरी सिस्टम है।

आप Intel-आधारित Mac पर macOS रिकवरी के ऐप्स का उपयोग अपनी आंतरिक डिस्क की मरम्मत करने, macOS को फिर इंस्टॉल करने, Time Machine बैकअप से अपनी फ़ाइलें रिस्टोर करने, सुरक्षा विकल्प सेट करने आदि के लिए कर सकते हैं।

macOS को फिर इंस्टॉल करने के लिए इंटरनेट कनेक्शन आवश्यक है। इंटरनेट से कनेक्ट होने के लिए, आप वायरलेस या वायर युक्त नेटवर्क कनेक्शन का उपयोग कर सकते हैं। वाई-फ़ाई उपयोग करके इंटरनेट से कनेक्ट करेंदेखें।

नोट : यदि आप कैप्टिव पोर्टल (उदाहरण के लिए कॉफ़ी शॉप, लाइब्रेरी या होटल में) या किसी एंटरप्राइज़ नेटवर्क से कनेक्ट होने की कोशिश कर रहे हैं, तो ऐसा हो सकता है कि आप macOS रिकवरी में इंटरनेट को ऐक्सेस नहीं कर पाएँ।

Intel-आधारित Mac पर macOS रिकवरी में उपलब्ध ऐप्स

Intel-आधारित Mac पर macOS रिकवरी में निम्नलिखित ऐप्स उपलब्ध हैं :

  • रिकवरी : रिकवरी ऐप उस समय खुलता है जब आपका कंप्यूटर macOS रिकवरी में शुरू होता है। macOS रिकवरी में अपना Intel-आधारित Mac शुरू करें और macOS रिकवरी को बंद करेंदेखें।

  • Time Machine सिस्टम रीस्टोर : Time Machine बैकअप से अपना डेटा रीस्टोर करें। रिकवरी ऐप में, Time Machine से रीस्टोर करें चुनें, फिर जारी रखें पर क्लिक करें। रिकवरी ऐप पर वापस आने के लिए, Time Machine सिस्टम रीस्टोर > Time Machine सिस्टम रीस्टोर बंद करें चुनें। अपना सिस्टम रीस्टोर करने के लिए Time Machine का उपयोग करें देखें।

  • macOS इंस्टॉल करें :अपने कंप्यूटर पर macOS फिर इंस्टॉल करें। रिकवरी ऐप में, macOS फिर इंस्टॉल करें चुनें, फिर जारी रखें पर क्लिक करें। रिकवरी ऐप पर वापस आने के लिए, macOS Big Sur इंस्टॉल करें > इंस्टॉल बंद करें चुनें। macOS फिर इंस्टॉल करेंदेखें।

  • Safari: रिकवरी ऐप में, Safari चुनें, फिर जारी रखें पर क्लिक करें। रिकवरी ऐप पर वापस आने के लिए, Safari > Safari बंद करें चुनें। यदि आप इंटरनेट से कनेक्ट हैं, तो आप Apple सहायता वेबसाइट पर अधिक जानकारी खोजने के लिए Safari का उपयोग कर सकते हैं। नोट करें कि Safari ब्राउज़र प्लग-इन और एक्सटेंशन को macOS रिकवरी में अक्षम किया गया है।

  • डिस्क यूटिलिटी : अपनी डिस्क की मरम्मत करें या उसे मिटाएँ। रिकवरी ऐप में, डिस्क यूटिलिटी चुनें, फिर जारी रखें पर क्लिक करें। रिकवरी ऐप पर वापस आने के लिए, डिस्क यूटिलिटी > डिस्क यूटिलिटी बंद करें चुनें। macOS मिटाएँ और फिर इंस्टॉल करें देखें।

  • स्टार्टअप सुरक्षा यूटिलिटी : अपने Mac के लिए सुरक्षा नीतियाँ सेट करें। रिकवरी ऐप में, यूटिलिटी > स्टार्टअप सुरक्षा यूटिलिटी चुनें। रिकवरी ऐप में वापस आने के लिए, स्टार्टअप सुरक्षा यूटिलिटी > स्टार्टअप सुरक्षा यूटिलिटी बंद करें चुनें। फ़र्मवेयर पासवर्ड सेट करने के लिए स्टार्टअप सुरक्षा यूटिलिटी का उपयोग करें और सुरक्षित बूट विकल्प सेट करने के लिए स्टार्टअप सुरक्षा यूटिलिटी का उपयोग करें देखें।

    स्टार्टअप सुरक्षा यूटिलिटी के उपलब्ध विकल्प आपके Mac मॉडल के आधार पर भिन्न होते हैं। Apple सहायता आलेख स्टार्टअप सुरक्षा यूटिलिटी का परिचय और Apple T2 सिक्योरिटी चिप वाले Mac मॉडलदेखें।

  • स्टार्टअप डिस्क : अपने Mac के लिए स्टार्टअप डिस्क सेट करें। Apple मेनू > स्टार्टअप डिस्क चुनें। रिकवरी ऐप पर वापस आने के लिए, स्टार्टअप डिस्क > स्टार्टअप डिस्क बंद करें चुनें।

  • टर्मिनल : रिकवरी ऐप में, यूटिलिटी > टर्मिनल चुनें। रिकवरी ऐप पर वापस आने के लिए, टर्मिनल > टर्मिनल बंद करें चुनें।

आप निम्नलिखित करने के लिए macOS रिकवरी का भी उपयोग कर सकते हैं :

  • रिकवरी लॉग देखें

  • रिकवरी डायग्नॉस्टिक फ़ाइल बनाएँ

macOS रिकवरी में अपना Intel-आधारित Mac कंप्यूटर शुरू करें

  1. अपना Mac रीस्टार्ट करें।

  2. निम्नलिखित कुंजी संयोजन को तत्काल उस समय तक दबाएँ रखें जब तक कि आपको स्टार्टअप स्क्रीन नहीं दिखती है :

    • कमांड-R: बिल्टइन macOS रिकवरी सिस्टम से शुरू करें। उस नवीनतम macOS को फिर इंस्टॉल करने के लिए इस कुंजी संयोजन का उपयोग करें जो आपके सिस्टम पर इंस्टॉल था या macOS रिकवरी में अन्य ऐप्स का उपयोग करें।

    • ऑप्शन-कमांड-R: इंटरनेट पर macOS रिकवरी से शुरू करें। macOS को फिर इंस्टॉल करने के लिए इस कुंजी संयोजन का उपयोग करें और macOS के उस नवीनतम संस्करण पर अपग्रेड करें जो आपके Mac के साथ संगत है।

    • ऑप्शन-शिफ़्ट-कमांड-R: इंटरनेट पर macOS रिकवरी से शुरू करें। macOS के उस संस्करण को जो आपके Mac के साथ आया था या अभी भी उपलब्ध सबसे निकटतम संस्करण को फिर इंस्टॉल करने के लिए, इस कुंजी संयोजन का उपयोग करें।

  3. यदि आपको लॉक दिखता है, तो अपने Mac का पासवर्ड दर्ज करें।

  4. यदि आपकी डिस्क पर एकाधिक वॉल्यूम है, तो वह वॉल्यूम चुनें जिसे आप रिकवर करना चाहते हैं, फिर अगला पर क्लिक करें।

  5. यदि अनुरोध किया गया है, तो व्यवस्थापक खाता चुनें, अगला पर क्लिक करें, खाते का पासवर्ड दर्ज करें, फिर जारी रखें पर क्लिक करें।

वाई-फ़ाई उपयोग करके इंटरनेट से कनेक्ट करें

यदि वाई-फ़ाई मेनू आइकॉन (स्क्रीन के ऊपर-दाएँ कोने में) में एक से अधिक बार हैं, तो इसका मतलब वाई-फ़ाई चालू है और वायरलेस नेटवर्क से जुड़ा हुआ है। यदि वाई-फ़ाई मेनू आइकॉन में कोई बार नहीं है , तो वाई-फ़ाई चालू है लेकिन वायरलेस नेटवर्क से जुड़ा हुआ नहीं है। यदि वाई-फ़ाई मेनू आइकॉन पर स्लैश का चिह्न है , तो वाई-फ़ाई बंद है। किसी वायरलेस नेटवर्क से जुड़ने से पहले आपको वाई-फ़ाई को चालू करना होता है।

अपने Mac पर, निम्नलिखित में से कोई एक कार्य करें :

  • वाई-फ़ाई को चालू करें : वाई-फ़ाई मेनू आइकॉन पर क्लिक करें, फिर वाई-फ़ाई चालू करें चुनें।

  • वाई-फ़ाई को बंद करें : वाई-फ़ाई मेनू आइकॉन पर क्लिक करें, फिर वाई-फ़ाई बंद करें चुनें।

  • किसी नेटवर्क से जुड़ें : वाई-फ़ाई मेनू आइकॉन पर क्लिक करें, फिर कोई नेटवर्क चुनें। पासवर्ड माँगे जाने पर, नेटवर्क का पासवर्ड टाइप करें।

  • किसी बंद नेटवर्क से जुड़ें : वाई-फ़ाई मेनू आइकॉन पर क्लिक करें, फिर अन्य नेटवर्क से जुड़ें चुनें। नेटवर्क का नाम और अनुरोध किए जाने पर नेटवर्क का पासवर्ड टाइप करें।

अपनी आंतरिक स्टोरेज डिवाइस की मरम्मत करें

यदि आपके कंप्यूटर में समस्या है या आप अपना कंप्यूटर शुरू करने पर macOS रिकवरी विंडो देखते हैं, तो आपको अपने कंप्यूटर के आंतरिक स्टोरेज डिवाइस (जैसे सॉलिड-स्टेट ड्राइव (SSD)) की मरम्मत करनी होगी।

  1. रिकवरी ऐप में, डिस्क यूटिलिटी चुनें, फिर जारी रखें पर क्लिक करें।

  2. डिस्क यूटिलिटी ऐप में, दृश्य > सभी डिवाइस दिखाएँ चुनें।

  3. साइडबार में, कोई वॉल्यूम चुनें, फिर प्राथमिक उपचार बटन पर क्लिक करें ।

  4. प्राथमिक उपचार डायलॉग में, रन पर क्लिक करें, फिर स्क्रीन पर दिए गए निर्देशों का पालन करें।

  5. प्राथमिक उपचार प्रक्रिया पूरी होने पर, पूर्ण पर क्लिक करें।

  6. स्टोरेज डिवाइस पर प्रत्येक वॉल्यूम के लिए चरण 3 से लेकर 5 तक दोहराएँ।

  7. साइडबार में, कोई कंटेनर चुनें, फिर प्राथमिक उपचार बटन पर क्लिक करें ।

  8. प्राथमिक उपचार डायलॉग में, रन पर क्लिक करें, फिर स्क्रीन पर दिए गए निर्देशों का पालन करें।

  9. प्राथमिक उपचार प्रक्रिया पूरी होने पर, पूर्ण पर क्लिक करें।

  10. स्टोरेज डिवाइस पर प्रत्येक कंटेनर के लिए चरण 7 से लेकर 9 तक दोहराएँ।

  11. साइडबार में, स्टोरेज डिवाइस चुनें, फिर प्राथमिक उपचार बटन पर क्लिक करें ।

  12. प्राथमिक उपचार डायलॉग में, रन पर क्लिक करें, फिर स्क्रीन पर दिए गए निर्देशों का पालन करें।

  13. प्राथमिक उपचार प्रक्रिया पूरी होने पर, पूर्ण पर क्लिक करें।

यदि डिस्क यूटिलिटी आपकी डिस्क की मरम्मत नहीं कर पाती है, तो जितना संभव हो सके अधिकतम उतनी जानकारी का बैकअप लें, फिर macOS मिटाएँ और फिर इंस्टॉल करें के निर्देशों का पालन करें।

macOS रिकवरी ऐप पर वापस जाने के लिए डिस्क यूटिलिटी > डिस्क यूटिलिटी बंद करें चुनें।

अपना सिस्टम रीस्टोर करने के लिए Time Machine का उपयोग करें

यदि आपने पहले Time Machine बैकअप बनाया है, तो अपने कंप्यूटर पर हर चीज को इसकी पिछली स्थिति में रीस्टोर करने के लिए Time Machine बैकअप से रीस्टोर करें का उपयोग करें।

महत्वपूर्ण : अपने Time Machine बैकअप से केवल उस कंप्यूटर पर रीस्टोर करने की कोशिश करें जो बैकअप का स्रोत है। यदि आप इस Mac से नए कंप्यूटर पर जानकारी ट्रांसफ़र करना चाहते हैं तो macOS रिकवरी बंद करें, अपना Mac रीस्टार्ट करें और macOS में माइग्रेशन सहायक ऐप उपयोग करें।

  1. यदि आपका बैकअप Time Capsule पर है, तो सुनिश्चित करें कि आप इंटरनेट से कनेक्टेड हैं।

  2. रिकवरी ऐप विंडो में, Time Machine बैकअप से रीस्टोर करें चुनें, फिर जारी रखें पर क्लिक करें।

  3. Time Machine सिस्टम रीस्टोर ऐप में, स्क्रीन पर दिए गए निर्देशों का पालन करें।

फ़र्मवेयर पासवर्ड सेट करने के लिए स्टार्टअप यूटिलिटी का उपयोग करें

फ़र्मवेयर पासवर्ड सेट करने पर आपका Mac आपके द्वारा चुनी गई स्टार्टअप डिस्क के अलावा किसी आंतरिक या बाहरी स्टोरेज डिवाइस को तब तक शुरू नहीं कर पाता है जब तक कि फ़र्मवेयर पासवर्ड नहीं डाला जाता है। आप स्टार्टअप सुरक्षा यूटिलिटी का उपयोग करके फ़र्मवेयर पासवर्ड को चालू, बंद कर सकते हैं या बदल सकते हैं।

फ़र्मवेयर पासवर्ड के चालू होने पर, प्रत्येक बार macOS रिकवरी या स्टार्टअप डिस्क प्राथमिकता में चुने गए स्टोरेज डिवाइस के अतिरिक्त अन्य स्टोरेज डिवाइस से शुरू करने पर, आपको फ़र्मवेयर पासवर्ड दर्ज करना होता है।

अपना Mac रीस्टार्ट करने के बाद, फ़र्मवेयर पासवर्ड के परिवर्तन प्रभावी हो जाते हैं।

  1. रिकवरी ऐप में, यूटिलिटी > स्टार्टअप सुरक्षा यूटिलिटी चुनें।

  2. अनुरोध किए जाने पर, व्यवस्थापक का यूज़रनेम और पासवर्ड दर्ज करें।

  3. निम्नलिखित में से कोई कार्य करें :

    • फ़र्मवेयर पासवर्ड को चालू करें : फ़र्मवेयर पासवर्ड चालू करें पर क्लिक करें, नया पासवर्ड दर्ज करें, पासवर्ड को सत्यापित करने के लिए इसे फिर से दर्ज करें, फिर पासवर्ड सेट करें पर क्लिक करें।

    • फ़र्मवेयर पासवर्ड को बंद करें : फ़र्मवेयर पासवर्ड बंद करें पर क्लिक करें, पासवर्ड दर्ज करें, फिर पासवर्ड बंद करें पर क्लिक करें।

    • फ़र्मवेयर पासवर्ड को बदलें : पासवर्ड बदलें पर क्लिक करें, पुराना पासवर्ड दर्ज करें, नया पासवर्ड दर्ज करें और इसे सत्यापित करें, फिर पासवर्ड बदलें पर क्लिक करें।

  4. स्टार्टअप सुरक्षा यूटिलिटी बंद करें पर क्लिक करें।

सुरक्षित बूट विकल्प सेट करने के लिए स्टार्टअप सुरक्षा यूटिलिटी का उपयोग करें

सुरक्षित बूट विकल्प केवल Apple T2 सिक्योरिटी चिप वाले Mac कंप्यूटरों पर ही उपलब्ध हैं।

अपना Mac रीस्टार्ट करने के बाद, सुरक्षित बूट विकल्प के परिवर्तन प्रभावी होते हैं।

  1. रिकवरी ऐप में, यूटिलिटी > स्टार्टअप सुरक्षा यूटिलिटी चुनें।

  2. व्यवस्थापक का यूज़रनेम और पासवर्ड दर्ज करें।

  3. सुरक्षित बूट पेन में, निम्नलिखित में से कोई सुरक्षा विकल्प चुनें :

    • पूर्ण सुरक्षा : सुनिश्चित करें कि केवल आपका वर्तमान OS या हस्ताक्षरित ऑपरेटिंग सिस्टम सॉफ़्टवेयर जिसे वर्तमान में Apple द्वारा विश्वसनीय माना गया है, ही चलाया जाए। इस मोड के लिए सॉफ़्टवेयर इंस्टॉलेशन के समय पर नेटवर्क कनेक्शन होना आवश्यक है।

    • मध्यम सुरक्षा : हस्ताक्षरित ऑपरेटिंग सिस्टम सॉफ़्टवेयर का कोई भी संस्करण चलाने की अनुमति देता है जिसे कभी Apple द्वारा विश्वसनीय माना गया था।

    • कोई सुरक्षा नहीं : बूट करने योग्य ऑपरेटिंग सिस्टम पर कोई आवश्यकता लागू नहीं करता है।

  4. अनुमति प्राप्त बूट मीडिया पेन में, निम्नलिखित बूट विकल्पों में से कोई चुनें :

    • बाहरी या हटाने योग्य मीडिया से बूट करने की अनुमति नहीं दें : बूट करने की क्षमता पर रोक लगाता है जो T2 से सुरक्षित नहीं है, जैसे USB और Thunderbolt ड्राइव या आंतरिक रूप से कनेक्ट PCIe या SATA ड्राइव।

    • बाहरी या हटाने योग्य मीडिया से बूट करने की अनुमति दें : किसी भी डिवाइस से बूट करने की क्षमता पर रोक नहीं लगाता है।

  5. स्टार्टअप सुरक्षा यूटिलिटी > स्टार्टअप सुरक्षा यूटिलिटी बंद करें चुनें।

macOS फिर इंस्टॉल करें

कुछ स्थितियों में, आपको macOS फिर इंस्टॉल करना पड़ सकता है। आप अपनी फ़ाइलों और यूज़र सेटिंग्ज़ को बरकरार रखते हुए macOS फिर इंस्टॉल कर सकते हैं।

  1. सुनिश्चित करें कि आप इंटरनेट से कनेक्टेड हैं।

  2. रिकवरी ऐप में, macOS Big Sur फिर इंस्टॉल करें पर क्लिक करें, फिर जारी रखें पर क्लिक करें।

    महत्वपूर्ण : आपने जिस पेन में डिस्क को चुना है, अपनी वर्तमान macOS डिस्क चुनें (अधिकांश मामलों में, केवल यही उपलब्ध होती है)।

  3. स्क्रीन पर दिए गए निर्देशों का पालन करें।

macOS मिटाएँ और फिर इंस्टॉल करें

यदि आप macOS को मिटाते और फिर इंस्टॉल करते हैं, तो Mac पर आपके यूज़र खाते, नेटवर्क सेटिंग्ज़ और आपकी सभी फ़ाइलें व फ़ोल्डर डिलीट हो जाएँगे। macOS मिटाने से पहले नीचे दिए गए निर्देशों का पालन करके अपने सिस्टम में फिर लॉग इन करने की कोशिश करें, फिर जिन फ़ाइलों को आप रखना चाहते हैं उन्हें अन्य डिस्क या स्टोरेज डिवाइस पर कॉपी करके उनका बैकअप रखें। साथ ही macOS को फिर इंस्टॉल करने के बाद फिर से आसानी से कनेक्ट होने के लिए नेटवर्क प्राथमिकता से अपनी नेटवर्क सेटिंग्ज़ को लिख लें।

  1. सुनिश्चित करें कि आप इंटरनेट से कनेक्टेड हैं।

  2. रिकवरी ऐप में, डिस्क यूटिलिटी पर क्लिक करें, फिर जारी रखें पर क्लिक करें।

  3. डिस्क यूटिलिटी में, वह वॉल्यूम चुनें जिसे आप साइडबार में मिटाना चाहते हैं, फिर टूलबार में मिटाएँ बटन पर क्लिक करें।

  4. नाम फ़ील्ड में वॉल्यूम के लिए कोई नाम टाइप करें।

  5. फ़ॉर्मैट पॉप-अप मेनू पर क्लिक करें, APFS चुनें, फिर वॉल्यूम समूह मिटाएँ पर क्लिक करें।

  6. मिटाने की प्रक्रिया पूरी होने पर, पूर्ण पर क्लिक करें, फिर डिस्क यूटिलिटी > डिस्क यूटिलिटी बंद करें चुनें।

  7. रिकवरी ऐप में, macOS Big Sur फिर इंस्टॉल करें चुनें, जारी रखें पर क्लिक करें, फिर स्क्रीन पर दिए गए निर्देशों का पालन करें।

रिकवरी लॉग देखें

रिकवरी लॉग में macOS रिकवरी द्वारा लॉग किए गए संदेश शामिल होते है। आप प्रदर्शित करने के लिए संदेशों का प्रकार निर्दिष्ट कर सकते हैं, किसी ख़ास टेक्स्ट के लिए संदेश फ़िल्टर कर सकते हैं और रिकवरी लॉग को फ़ाइल में सहेज सकते हैं।

  1. रिकवरी ऐप में, विंडो > रिकवरी लॉग चुनें।

  2. विस्तार स्तर पॉप मेनू पर क्लिक करें फिर दिखाने के लिए संदेश चुनें :

    • केवल त्रुटियाँ दिखाएँ : केवल त्रुटि वाले संदेश दिखाने के लिए यह विकल्प चुनें।

    • त्रुटियाँ और प्रगति दिखाएँ : केवल त्रुटियों और प्रगति वाले संदेश दिखाने के लिए यह विकल्प चुनें।

    • सभी लॉग दिखाएँ : सभी संदेश दिखाने के लिए यह विकल्प चुनें।

  3. संदेश फ़िल्टर करने के लिए फ़िल्टर फ़ील्ड में खोजा जाने वाला टेक्स्ट टाइप करें।

  4. रिकवरी लॉग सहेजने के लिए सहेजें बटन पर क्लिक करें, फ़ाइल का स्थान चुनें फिर सहेजें पर क्लिक करें।

  5. रिकवरी लॉग देख लेने के बाद विंडो के ऊपरी बाएँ कोने में बंद करें बटन पर क्लिक करें।

रिकवरी डायग्नॉस्टिक फ़ाइल बनाएँ

यदि आप फ़ीडबैक सहायक का उपयोग करके समस्या की रिपोर्ट करते हैं, तो आपसे रिकवरी डायग्नॉस्टिक फ़ाइल देने के लिए कहा जा सकता है। आप फ़ाइल बना सकते हैं और उसे बाहरी स्टोरेज डिवाइस में सहेज सकते हैं।

  1. अपने Mac से बाहरी स्टोरेज डिवाइस कनेक्ट करें।

  2. कंट्रोल-ऑप्शन-शिफ़्ट-कमांड-पीरियड दबाएँ।

  3. स्क्रीन चमकने के बाद डायलॉग दिखाई देने तक कुछ मिनट रुकें।

  4. इस डायलॉग का उपयोग रिकवरी डायग्नॉस्टिक फ़ाइल को अपने बाहरी स्टोरेज डिवाइस के किसी स्थान पर सहेजने के लिए करें।

macOS रिकवरी बंद करें

आपके सिस्टम को Time Machine से रीस्टोर करते समय या macOS फिर इंस्टॉल करते समय आपका Mac ऑटोमैटिकली रीस्टार्ट हो जाएगा। अन्य स्थितियों में, आपको macOS रिकवरी को मैनुअली बंद करना पड़ता है (उदाहरण के लिए, यदि आप macOS को फिर इंस्टॉल करने से पहले अपनी फ़ाइलों का बैकअप लेना चाहते हैं या यदि आपने स्टार्टअप सुरक्षा यूटिलिटी का उपयोग करके सेटिंग को अभी बदला है)।

अपने Mac पर, निम्नलिखित में से कोई एक कार्य करें :

  • अपना Mac रीस्टार्ट करें : Apple मेनू > रीस्टार्ट करें चुनें।

  • अपना Mac रीस्टार्ट करने से पहले कोई भिन्न स्टार्टअप डिस्क चुनें : Apple मेनू > स्टार्टअप डिस्क चुनें, फिर रीस्टार्ट करें पर क्लिक करें।

  • अपना Mac शटडाउन करें : Apple मेनू > शटडाउन करें चुनें।
