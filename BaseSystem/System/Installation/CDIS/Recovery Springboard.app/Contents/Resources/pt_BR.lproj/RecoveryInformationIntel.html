Usar a Recuperação do macOS em um Mac com processador Intel

A Recuperação do macOS é o sistema de recuperação integrado do Mac.

Você pode usar os apps da Recuperação do macOS em um Mac com processador Intel para reparar o disco interno, reinstalar o macOS, restaurar os arquivos de um backup do Time Machine, definir as opções de segurança e muito mais.

A reinstalação do macOS requer conexão à internet. Para se conectar à internet, você pode usar uma conexão de rede sem fio ou com fio. Consulte Conectar-se à internet por Wi-Fi.

Observação: caso esteja tentando se conectar a um portal cativo (por exemplo, em uma cafeteria, biblioteca ou hotel) ou uma rede corporativa, talvez você não consiga acessar a internet na Recuperação do macOS.

Apps disponíveis na Recuperação do macOS em um Mac com processador Intel

Os seguintes apps estão disponíveis na Recuperação do macOS em um Mac com processador Intel:

  • Recovery: o app Recovery é aberto quando o computador é inicializado na Recuperação do macOS. Consulte Iniciar o Mac com processador Intel na Recuperação do macOS e Encerrar a Recuperação do macOS.

  • Restauração de Sistema do Time Machine: restaure os dados a partir de um backup do Time Machine. No app Recovery, selecione “Restaurar do Time Machine” e clique em Continuar. Para retornar ao app Recovery, selecione Restauração de Sistema do Time Machine > Encerrar Restauração de Sistema do Time Machine. Consulte Usar o Time Machine para restaurar o sistema.

  • Instalação do macOS: reinstale o macOS no computador. No app Recovery, selecione “Reinstalar macOS” e clique em Continuar. Para retornar ao app Recovery, selecione Instalação do macOS Big Sur > Encerrar Instalação do macOS. Consulte Reinstalar o macOS.

  • Safari: no app Recovery, selecione Safari e clique em Continuar. Para retornar ao app Recovery, selecione Safari > Encerrar Safari. Caso esteja conectado à internet, você pode usar o Safari para pesquisar mais informações no site de Suporte da Apple. Observe que os plug-ins e as extensões do Safari estão desativados na Recuperação do macOS.

  • Utilitário de Disco: repara ou apaga o disco. No app Recovery, selecione “Utilitário de Disco” e clique em Continuar. Para retornar ao app Recovery, selecione Utilitário de Disco > Encerrar Utilitário de Disco. Consulte Apagar e reinstalar o macOS.

  • Utilitário de Segurança da Inicialização: define as políticas de segurança do Mac. No app Recovery, selecione Utilitários > Utilitário de Segurança da Inicialização. Para retornar ao app Recovery, selecione Utilitário de Segurança da Inicialização > Encerrar Utilitário de Segurança da Inicialização. Consulte Usar o Utilitário de Segurança da Inicialização para definir uma senha de firmware e Usar o Utilitário de Segurança da Inicialização para definir opções de inicialização segura.

    As opções disponíveis no Utilitário de Segurança da Inicialização variam, dependendo do modelo do Mac. Consulte os artigos de Suporte da Apple Sobre o Utilitário de Segurança da Inicialização e Modelos de Mac com chip Apple T2 Security.

  • Disco de Inicialização: define o disco de inicialização do Mac. Selecione menu Apple > Disco de Inicialização. Para retornar ao app Recovery, selecione Disco de Inicialização > Encerrar Disco de Inicialização.

  • Terminal: no app Recovery, selecione Utilitários > Terminal. Para retornar ao app Recovery, selecione Terminal > Encerrar Terminal.

Você também pode usar a Recuperação do macOS para fazer o seguinte:

  • Visualizar o registro de recuperação

  • Criar um arquivo de diagnóstico de recuperação

Iniciar o Mac com processador Intel na Recuperação do macOS

  1. Reinicie o Mac.

  2. Imediatamente, mantenha pressionada uma das seguintes combinações de teclas até ver a tela inicial:

    • Comando + R: inicie a partir do sistema de recuperação macOS integrado. Use esta combinação de teclas para reinstalar a versão do macOS mais recente instalada no sistema ou para usar os outros apps na Recuperação do macOS.

    • Opção + Comando + R: inicie a partir da Recuperação do macOS pela internet. Use esta combinação de teclas para reinstalar o macOS e atualizar para a versão mais recente do macOS compatível com o Mac.

    • Opção + Shift + Comando + R: inicie a partir da Recuperação do macOS pela internet. Use esta combinação de teclas para reinstalar a versão do macOS que veio com o Mac ou a versão mais próxima ainda disponível.

  3. Caso veja um cadeado, insira a senha do Mac.

  4. Se você tiver vários volumes no disco, selecione o que deseja recuperar e clique em Seguinte

  5. Se solicitado, escolha uma conta de administrador, clique em Seguinte, digite a senha da conta e clique em Continuar.

Conectar-se à internet por Wi-Fi

Se o ícone de menu Wi-Fi (no canto superior direito da tela) exibir uma ou mais barras , o Wi-Fi está ativado e conectado a uma rede sem fio. Se o ícone de menu Wi-Fi não exibir nenhuma barra , o Wi-Fi está ativado, mas não está conectado a uma rede sem fio. Se houver um risco sobre o ícone de menu Wi-Fi , o Wi-Fi está desativado. O Wi-Fi precisa estar ativo para que você possa se conectar a uma rede sem fio.

No Mac:

  • Para ativar o Wi-Fi: clique no ícone de menu Wi-Fi e selecione “Ativar Wi-Fi”.

  • Para desativar o Wi-Fi: clique no ícone de menu Wi-Fi e selecione “Desativar Wi-Fi”.

  • Para conectar-se a uma rede: clique no ícone de menu Wi-Fi e selecione uma rede. Se solicitado, digite a senha da rede.

  • Para conectar-se a uma rede fechada: clique no ícone de menu Wi-Fi e selecione “Conectar-se a Outra Rede”. Digite o nome da rede e, se solicitado, a senha.

Reparar o dispositivo de armazenamento interno

Se você tiver problemas com o computador ou se o iniciou e viu a janela da Recuperação do macOS, pode ser necessário reparar o dispositivo de armazenamento interno do computador (por exemplo, uma unidade de estado sólido (SSD)).

  1. No app Recovery, selecione “Utilitário de Disco” e clique em Continuar.

  2. No app Utilitário de Disco, escolha Visualizar > Mostrar Todos os Dispositivos.

  3. Na barra lateral, selecione um volume e clique no botão Reparar .

  4. No diálogo Reparar, clique em Executar e siga as instruções na tela.

  5. Quando o processo do Reparador tiver terminado, clique em Concluído.

  6. Repita os passos de 3 a 5 para cada volume do dispositivo de armazenamento.

  7. Na barra lateral, selecione um contêiner e clique no botão Reparar .

  8. No diálogo Reparar, clique em Executar e siga as instruções na tela.

  9. Quando o processo do Reparador tiver terminado, clique em Concluído.

  10. Repita os passos de 7 a 9 para cada contêiner do dispositivo de armazenamento.

  11. Na barra lateral, selecione o dispositivo de armazenamento e clique no botão Reparar .

  12. No diálogo Reparar, clique em Executar e siga as instruções na tela.

  13. Quando o processo do Reparador tiver terminado, clique em Concluído.

Se o Utilitário de Disco não conseguir reparar o disco, faça backup do máximo possível de informações e siga as instruções em Apagar e reinstalar o macOS.

Para retornar ao app Recovery, selecione Utilitário de Disco > Encerrar Utilitário de Disco.

Usar o Time Machine para restaurar o sistema

Se você já criou um backup do Time Machine antes, use “Restaurar do Time Machine” para restaurar tudo em seu computador à condição anterior.

Importante: use o backup do Time Machine apenas para restaurar as informações do computador que é a fonte do backup. Se quiser transferir informações desse Mac para um computador novo, feche a Recuperação do macOS, reinicie o Mac e use o app Assistente de Migração no macOS.

  1. Se o backup estiver em um Time Capsule, verifique se você está conectado à internet.

  2. No app Recovery, selecione “Restaurar do Time Machine” e clique em Continuar.

  3. No app Restauração de Sistema do Time Machine, siga as instruções na tela.

Usar o Utilitário de Segurança da Inicialização para definir uma senha de firmware

Definir uma senha de firmware impede que o Mac inicialize a partir de qualquer dispositivo de armazenamento interno ou externo que não seja o disco de inicialização que você selecionou, a menos que a senha de firmware seja inserida. Você pode usar o Utilitário de Segurança da Inicialização para ativar, desativar ou alterar a senha de firmware.

Quando a senha de firmware está ativada, é necessário inseri-la sempre que você inicializar o computador a partir da Recuperação do macOS ou de um dispositivo de armazenamento diferente do selecionado nas preferências Disco de Inicialização.

Depois de reiniciar o Mac, as alterações da senha de firmware entram em vigor.

  1. No app Recovery, selecione Utilitários > Utilitário de Segurança da Inicialização.

  2. Se solicitado, insira o nome de usuário e a senha de um administrador.

  3. Execute uma das ações a seguir:

    • Para ativar a senha de firmware: clique em “Ativar Senha de Firmware”, digite uma senha nova, digite-a novamente para confirmá-la e clique em Definir Senha.

    • Para desativar a senha de firmware: clique em “Desativar Senha de Firmware”, insira a senha e clique em Desativar Senha.

    • Para alterar a senha de firmware: clique em Alterar Senha, insira a senha antiga, insira a senha nova, confirme-a e clique em Alterar Senha.

  4. Clique em “Encerrar o Utilitário de Segurança da Inicialização”.

Usar o Utilitário de Segurança da Inicialização para definir opções de inicialização segura

As opções de inicialização segura estão disponíveis apenas nos computadores Mac com o chip Apple T2 Security.

Depois de reiniciar o Mac, as alterações da opção de inicialização segura entram em vigor.

  1. No app Recovery, selecione Utilitários > Utilitário de Segurança da Inicialização.

  2. Insira o nome de usuário e a senha de um administrador.

  3. No painel Inicialização Segura, escolha uma das seguintes opções de segurança:

    • Segurança Total: garante que apenas o seu sistema operacional atual ou um software de sistema operacional verificado e considerado confiável pela Apple naquele momento possa ser executado. Este modo requer conexão de rede no momento da instalação do software.

    • Segurança Média: permite a execução de qualquer versão de software de sistema operacional verificado e alguma vez considerado confiável pela Apple.

    • Sem Segurança: não impõe nenhum requisito ao sistema operacional inicializável.

  4. No painel “Permitir Mídia de Inicialização”, escolha uma das seguintes opções de inicialização:

    • Não permitir inicialização a partir de mídia externa ou removível: restringe a capacidade de inicialização a partir de qualquer dispositivo não protegido pelo T2, como unidades USB e Thunderbolt ou unidades PCIe ou SATA conectadas internamente.

    • Permitir inicialização a partir de mídia externa ou removível: não restringe a capacidade de inicialização a partir de outros dispositivos.

  5. Selecione Utilitário de Segurança da Inicialização > Encerrar o Utilitário de Segurança da Inicialização.

Reinstalar o macOS

Em algumas situações, pode ser necessário reinstalar o macOS. Você pode reinstalar o macOS e manter os arquivos e ajustes de usuário intactos.

  1. Certifique-se de estar conectado à internet.

  2. No app Recovery, clique em “Reinstalar macOS Big Sur” e clique em Continuar.

    Importante: no painel de seleção de disco, selecione o disco macOS atual (na maioria dos casos, é o único disponível).

  3. Siga as instruções na tela.

Apagar e reinstalar o macOS

Se você apagar e reinstalar o macOS, as contas de usuário, os ajustes de rede e todos os arquivos e pastas do Mac serão excluídos. Antes de apagar o macOS, tente iniciar sessão novamente no sistema e faça o backup de todos os arquivos que deseja manter copiando-os para outro dispositivo de armazenamento. Além disso, anote os ajustes de rede nas preferências Rede para que fique mais fácil refazer a conexão após a reinstalação do macOS.

  1. Certifique-se de estar conectado à internet.

  2. No app Recovery, clique em “Utilitário de Disco” e clique em Continuar.

  3. No Utilitário de Disco, selecione o volume que deseja apagar na barra lateral e clique no botão Apagar na barra de ferramentas.

  4. Digite um nome para o volume no campo Nome.

  5. Clique no menu local Formato, escolha APFS e clique em “Apagar Grupo de Volumes”.

  6. Quando o processo de apagamento terminar, clique em Concluído e escolha Utilitário de Disco > Encerrar Utilitário de Disco.

  7. No app Recovery, clique em “Reinstalar macOS Big Sur”, clique em Continuar e siga as instruções na tela.

Visualizar o registro de recuperação

O registro de recuperação contém mensagens registradas pela Recuperação do macOS. Você pode especificar os tipos de mensagens exibidas, filtrar mensagens para buscar um texto específico e salvar o registro de recuperação em um arquivo.

  1. No app Recovery, escolha Janela > Registro de Recuperação.

  2. Clique no menu local “Nível de Detalhe” e escolha quais mensagens deseja mostrar:

    • Mostrar Apenas os Erros: escolha esta opção para mostrar apenas mensagens de erro.

    • Mostrar Erros e Progresso: escolha esta opção para mostrar mensagens de erro e mensagens de progresso.

    • Mostrar Todos os Registros: escolha esta opção para mostrar todas as mensagens.

  3. Para filtrar mensagens, digite o texto que deseja buscar no campo Filtro.

  4. Para salvar o registro de recuperação, clique no botão Salvar, escolha um local para o arquivo e clique em Salvar.

  5. Quando terminar de visualizar o registro de recuperação, clique no botão Fechar no canto superior esquerdo da janela.

Criar um arquivo de diagnóstico de recuperação

Se você protocolar um relatório de problema usando o Assistente de Feedback, talvez será solicitado a fornecer um arquivo de diagnóstico de recuperação. É possível criar o arquivo e salvá-lo em um dispositivo de armazenamento externo.

  1. Conecte o dispositivo de armazenamento externo ao Mac.

  2. Pressione Control + Option + Shift + Command + Ponto.

  3. Após a tela piscar, espere alguns instantes até que um diálogo apareça.

  4. Use o diálogo para salvar o arquivo de diagnóstico de recuperação em um local no dispositivo de armazenamento externo.

Encerrar a Recuperação do macOS

O Mac é reiniciado automaticamente enquanto restaura o sistema do Time Machine ou reinstala o macOS. Em outros casos, você precisa encerrar a Recuperação do macOS manualmente (por exemplo, se quiser fazer backup dos arquivos antes de reinstalar o macOS ou se acabou de alterar uma configuração usando o Utilitário de Segurança da Inicialização).

No Mac:

  • Para reiniciar o Mac: selecione menu Apple > Reiniciar.

  • Para escolher um disco de inicialização diferente antes de reiniciar o Mac: selecione menu Apple > Disco de Inicialização, selecione um disco de inicialização e clique em Reiniciar.

  • Para desligar o Mac: selecione menu Apple > Desligar.
