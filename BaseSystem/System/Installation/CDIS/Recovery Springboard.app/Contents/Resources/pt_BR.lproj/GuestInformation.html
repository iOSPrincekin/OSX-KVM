Informações para usuários convidados no Mac

Ao iniciar sessão como convidado no Mac com o FileVault ativo, é possível usar apenas o Safari. Não é possível usar outros apps.

Ao finalizar a sessão, todos os arquivos criados são apagados.

Conectar-se à internet

Se o ícone de menu Wi-Fi (no canto superior direito da tela) exibir uma ou mais barras , o Wi-Fi está ativado e conectado a uma rede sem fio. Se o ícone de menu Wi-Fi não exibir nenhuma barra , o Wi-Fi está ativado, mas não está conectado a uma rede sem fio. Se houver um risco sobre o ícone de menu Wi-Fi , o Wi-Fi está desativado. O Wi-Fi precisa estar ativo para que você possa se conectar a uma rede sem fio.

No Mac:

  • Para ativar ou desativar o Wi-Fi: clique no ícone de menu Wi-Fi e clique na opção para ativar ou desativar o Wi-Fi.

  • Para conectar-se a uma rede: clique no ícone de menu Wi-Fi e selecione uma rede. Se solicitado, digite a senha da rede.

Finalizar sessão de uma conta de Usuário Convidado

Quando terminar de usar o Mac, finalize a sessão da conta do Usuário Convidado.

  1. No Mac, selecione menu Apple > Finalizar Sessão do Usuário Convidado.

  2. Leia as informações do diálogo de aviso e:

    • Para finalizar a sessão: clique em Apagar Arquivos e selecione Finalizar Sessão.

    • Para copiar os arquivos: clique em Cancelar e copie os arquivos para outro meio de armazenamento, como uma unidade flash externa.
