Usar a Recuperação do macOS num Mac com Apple Silicon

Recuperação do macOS é o sistema integrado de recuperação do Mac.

Pode usar as aplicações na Recuperação do macOS num Mac com Apple Silicon para reparar o disco interno, reinstalar o macOS, restaurar os ficheiros a partir de uma cópia de segurança do Time Machine, definir a política de segurança para diferentes discos, transferir ficheiros entre dois computadores Mac, arrancar no modo de segurança, etc.

Para reinstalar o macOS é necessária uma ligação à internet. Para estabelecer ligação à internet, pode usar uma ligação de rede com ou sem fios. Consulte Estabelecer ligação à internet através de Wi‑Fi.

Nota: se estiver a tentar estabelecer ligação a um portal cativo (por exemplo, num café, numa biblioteca ou num hotel) ou a uma rede empresarial, pode não conseguir aceder à internet na Recuperação do macOS.

Aplicações disponíveis na Recuperação do macOS num Mac com Apple Silicon

As aplicações seguintes estão disponíveis na Recuperação do macOS num Mac com Apple Silicon:

  • Recuperação: a aplicação Recuperação fornece acesso a aplicações adicionais na Recuperação do macOS. Consulte Iniciar o computador com a Recuperação do macOS, Iniciar o computador com a Recuperação do macOS e emparelhar dispositivos de entrada Bluetooth e Sair da Recuperação do macOS.

  • Restauro do Sistema do Time Machine: restaure os seus dados a partir de uma cópia de segurança do Time Machine. Na aplicação Recuperação, selecione “Restaurar a partir do Time Machine” e, depois, clique em “Continuar”. Para voltar à aplicação Recuperação, selecione Restauro do Sistema do Time Machine > Sair do Restauro do Sistema do Time Machine. Consulte Usar o Time Machine para restaurar o sistema.

  • Instalar o macOS: reinstale o macOS no computador. Na aplicação Recuperação, selecione “Reinstalar o macOS” e, depois, clique em “Continuar”. Para voltar à aplicação Recuperação, selecione Instalar o macOS Big Sur > Sair de Instalar o macOS. Consulte Reinstalar o macOS.

  • Safari: na aplicação Recuperação, selecione “Safari” e, depois, clique em “Continuar”. Para voltar à aplicação Recuperação, selecione Safari > Sair do Safari. Se tiver uma ligação à internet, pode usar o Safari para pesquisar informação no site do Suporte Apple. Tenha em atenção que as extensões e os plug‑ins do navegador Safari estão desativados na Recuperação do macOS.

  • Utilitário de Discos: repare ou apague o disco. Na aplicação Recuperação, selecione “Utilitário de Discos” e, depois, clique em “Continuar”. Para voltar à aplicação Recuperação, selecione Utilitário de Discos > Sair do Utilitário de Discos. Consulte Apagar e reinstalar o macOS e Reparar o dispositivo de armazenamento interno.

  • Utilitário de Segurança de Arranque: defina as políticas de segurança para o Mac. Na aplicação Recuperação, selecione Utilitários > Utilitário de Segurança de Arranque. Para voltar à aplicação Recuperação, selecione Utilitário de Segurança de Arranque > Sair do Utilitário de Segurança de Arranque. Consulte Definir a política de segurança.

  • Terminal: altere as definições através da linha de comandos. Na aplicação Recuperação, selecione Utilitários > Terminal. Para voltar à aplicação Recuperação, selecione Terminal > Sair do Terminal.

  • Partilhar Disco: partilhe o disco de um Mac que tenha sido iniciado com a Recuperação do macOS. Na aplicação Recuperação, selecione Utilitários > Partilhar Disco. Consulte Usar Partilhar Disco para transferir ficheiros entre dois computadores Mac.

  • Disco de Arranque: defina o disco de arranque para o Mac. Selecione o menu Apple > Disco de Arranque. Para sair da aplicação, selecione Disco de Arranque > Sair do Disco de Arranque.

Também é possível utilizar a Recuperação do macOS para:

  • Iniciar o computador no modo de segurança

  • Definir o volume de arranque predefinido e reiniciar o computador

  • Ver o registo de recuperação

  • Criar um ficheiro de diagnóstico de recuperação

Iniciar o computador com a Recuperação do macOS

  1. Selecione o menu Apple > Desligar.

  2. Mantenha premido o botão de alimentação do Mac até ver “A carregar opções de arranque”.

  3. Clique em “Opções” e, depois, em “Continuar”.

  4. Digite a palavra‑passe de uma conta de administrador.

Iniciar o computador com a Recuperação do macOS e emparelhar dispositivos de entrada Bluetooth

Siga estas instruções se precisar de emparelhar um teclado, trackpad ou rato Bluetooth quando iniciar o Mac com a Recuperação do macOS.

  1. Selecione o menu Apple > Desligar.

  2. Mantenha premido o botão de alimentação do Mac até ver “A carregar opções de arranque”.

  3. Quando aparecer o volume do sistema e o botão Opções, prima o botão de alimentação três vezes.

    Nota: o botão de alimentação deve ser premido a intervalos de cerca de 1 segundo. Se premir o botão de alimentação demasiado rapidamente, a pressão não será registada porque as pressões rápidas estão reservadas para outras finalidades, como a utilização do VoiceOver.

  4. Use o Assistente de Configuração Bluetooth para emparelhar acessórios Bluetooth com o Mac.

    Nota: o Assistente de Configuração Bluetooth não aparece se já estiver ligado um dispositivo de entrada.

  5. Digite a palavra‑passe de uma conta de administrador.

Iniciar o computador no modo de segurança

  1. Selecione o menu Apple > Desligar.

  2. Mantenha premido o botão de alimentação do Mac até ver “A carregar opções de arranque”.

  3. Selecione um volume.

  4. Mantenha premida a tecla Shift e, depois, clique em “Continuar no modo de segurança”.

    O computador é reiniciado automaticamente.

Definir o volume de arranque predefinido e reiniciar o computador

  1. Selecione o menu Apple > Desligar.

  2. Mantenha premido o botão de alimentação do Mac até ver “A carregar opções de arranque”.

  3. Selecione um volume.

  4. Mantenha premida a tecla Opção e, depois, clique em “Usar sempre”.

    O computador é reiniciado automaticamente.

Estabelecer ligação à internet através de Wi‑Fi

Se o ícone do menu Wi‑Fi (no canto superior direito do ecrã) mostrar uma ou mais barras , o Wi‑Fi está ativado e ligado a uma rede sem fios. Se o ícone do menu Wi‑Fi não tiver nenhuma barra , o Wi‑Fi está ativado, mas não está ligado a uma rede sem fios. Se o ícone do menu tiver uma barra oblíqua sobreposta , o Wi‑Fi está desativado. É necessário ativar o Wi‑Fi para poder aceder a uma rede sem fios.

No Mac, proceda de uma das seguintes formas:

  • Ativar o Wi‑Fi: clique no ícone do menu Wi‑Fi e, depois, selecione “Ativar Wi‑Fi”.

  • Desativar o Wi‑Fi: clique no ícone do menu Wi‑Fi e, depois, selecione “Desativar Wi‑Fi”.

  • Aceder a uma rede: clique no ícone do menu Wi‑Fi e, depois, escolha uma rede. Se for solicitada a palavra‑passe da rede, digite‑a.

  • Aceder a uma rede fechada: clique no ícone do menu Wi‑Fi e, depois, escolha “Aceder a outra rede”. Digite o nome da rede e, se for solicitada, a palavra‑passe da rede.

Reparar o dispositivo de armazenamento interno

Se o computador tiver problemas ou se iniciar o computador e for apresentada a janela da Recuperação do macOS, pode ser necessário reparar o dispositivo de armazenamento interno do computador (por exemplo, uma unidade de estado sólido (SSD)).

  1. Na aplicação Recuperação, selecione “Utilitário de Discos” e, depois, clique em “Continuar”.

  2. Na aplicação Utilitário de Discos, selecione Visualização > Mostrar todos os dispositivos.

  3. Na barra lateral, selecione um volume e, depois, clique no botão “Primeiros socorros” .

  4. Na caixa de diálogo “Primeiros socorros”, clique em “Executar” e siga as instruções no ecrã.

  5. Quando o processo de “Primeiros socorros” terminar, clique em “OK”.

  6. Repita os passos 3 a 5 para cada volume no dispositivo de armazenamento.

  7. Na barra lateral, selecione um contentor e, depois, clique no botão “Primeiros socorros” .

  8. Na caixa de diálogo “Primeiros socorros”, clique em “Executar” e siga as instruções no ecrã.

  9. Quando o processo de “Primeiros socorros” terminar, clique em “OK”.

  10. Repita os passos 7 a 9 para cada contentor no dispositivo de armazenamento.

  11. Na barra lateral, selecione o dispositivo de armazenamento e, depois, clique no botão “Primeiros socorros” .

  12. Na caixa de diálogo “Primeiros socorros”, clique em “Executar” e siga as instruções no ecrã.

  13. Quando o processo de “Primeiros socorros” terminar, clique em “OK”.

Se o Utilitário de Discos não conseguir reparar o disco, efetue uma cópia de segurança de toda a informação que for possível e, depois, siga as instruções em Apagar e reinstalar o macOS.

Para voltar à aplicação Recuperação, selecione Utilitário de Discos > Sair do Utilitário de Discos.

Usar o Time Machine para restaurar o sistema

Se tiver criado anteriormente uma cópia de segurança do Time Machine, use a opção “Restaurar a partir de cópia de segurança do Time Machine” para restabelecer o estado anterior do computador.

Importante: a cópia de segurança do Time Machine só deve ser usada para restaurar informação no computador que foi a origem da cópia de segurança. Se pretender transferir informação deste Mac para um novo computador, saia da Recuperação do macOS, reinicie o Mac e use a aplicação Assistente de Migração no macOS.

  1. Se a cópia de segurança estiver num Time Capsule, certifique‑se de que o computador está ligado à internet.

  2. Na aplicação Recuperação, selecione “Restaurar a partir de cópia de segurança do Time Machine” e, depois, clique em “Continuar”.

  3. Na aplicação Restauro do Sistema do Time Machine, siga as instruções no ecrã.

Reinstalar o macOS

Em algumas circunstâncias, poderá ser necessário reinstalar o macOS. Pode reinstalar o macOS e manter intactos os seus ficheiros e as suas definições de utilizador.

  1. Certifique‑se de que o computador está ligado à internet.

  2. Na aplicação Recuperação, clique em “Reinstalar o macOS Big Sur” e, depois, clique em “Continuar”.

    Importante: no painel de seleção do disco, selecione o disco do macOS atual (na maioria dos casos, é o único disponível).

  3. Siga as instruções no ecrã.

Apagar e reinstalar o macOS

Se apagar e reinstalar o macOS, as contas de utilizador, as definições de rede e todos os ficheiros e pastas do Mac são apagados. Antes de apagar o macOS, tente voltar a iniciar sessão no sistema e, depois, efetue uma cópia de segurança dos ficheiros que pretende manter copiando‑os para outro dispositivo de armazenamento. Tome igualmente nota das definições de rede a partir do painel de preferências Rede, para ser mais fácil restabelecer as ligações após reinstalar o macOS.

  1. Certifique‑se de que o computador está ligado à internet.

  2. Na aplicação Recuperação, clique em “Utilitário de Discos” e, depois, clique em “Continuar”.

  3. No Utilitário de Discos, selecione na barra lateral o volume que pretende apagar e, depois, clique no botão “Eliminar” na barra de ferramentas.

  4. Digite um nome para o volume no campo Nome.

  5. Clique no menu pop‑up “Formato”, selecione “APFS” e, depois, clique em “Eliminar grupo de volumes”.

  6. Quando o processo de eliminação terminar, selecione “OK” e, depois, selecione Utilitário de Discos > Sair do Utilitário de Discos.

  7. Na aplicação Recuperação, selecione “Reinstalar o macOS Big Sur”, clique em “Continuar” e, depois, siga as instruções no ecrã.

Definir a política de segurança

  1. Na aplicação Recuperação, selecione Utilitários > Utilitário de Segurança de Arranque.

  2. Selecione o sistema que pretende usar para definir a política de segurança.

  3. Se o disco estiver cifrado com FileVault, clique em “Desproteger”, digite a palavra‑passe e clique em “Desproteger”.

  4. Clique em “Política de segurança”.

  5. Reveja as seguintes opções de segurança:

    • Segurança total: garante que apenas pode ser executado o SO atual ou um software de sistema operativo assinado que a Apple atualmente considere fidedigno. Este modo requer uma ligação à rede no momento da instalação do software.

    • Segurança reduzida: permite que seja executada qualquer versão do software de sistema operativo assinado que a Apple tenha considerado fidedigna.

  6. Se for necessário, selecione “Segurança reduzida”, digite o nome de utilizador e palavra‑passe de administrador e, depois, proceda de uma das seguintes formas:

    • Assinale a opção “Permitir a gestão pelos utilizadores de extensões do kernel de programadores identificados” para permitir a instalação de software que use extensões do kernel antigas.

    • Assinale a opção “Permitir a gestão remota das extensões do kernel e atualizações de software automáticas” para autorizar a gestão remota de extensões do kernel antigas e atualizações de software através de uma solução de gestão de dispositivos móveis (MDM).

  7. Clique em “OK”.

  8. Reinicie o Mac para as alterações entrarem em vigor.

Usar Partilhar Disco para transferir ficheiros entre dois computadores Mac

Pode transferir ficheiros entre um Mac iniciado com a Recuperação do macOS e outro Mac através da partilha de um volume e ligando os dois computadores com um cabo USB, USB‑C ou Thunderbolt.

  1. Ligue os dois computadores com um cabo USB, USB‑C ou Thunderbolt.

  2. Na aplicação Recuperação, selecione Utilitários > Partilhar Disco.

  3. Selecione o volume que pretende partilhar e, depois, clique em “Iniciar partilha”.

  4. No outro Mac, abra uma janela do Finder e, depois, clique em “Rede” na barra lateral, em “Locais”.

  5. Na janela “Rede”, faça duplo clique no Mac que tem o disco partilhado, clique em “Ligar como”, selecione “Convidado” na janela “Ligar como” e, depois, clique em “Ligar”.

  6. Transfira os ficheiros.

  7. Quando terminar a transferência dos ficheiros, ejete o volume no outro Mac.

Ver o registo de recuperação

O registo de recuperação contém mensagens registadas pela Recuperação do macOS. É possível especificar os tipos de mensagens a consultar, filtrar mensagens para procurar texto específico e guardar o registo de recuperação num ficheiro.

  1. Na aplicação Recuperação, selecione Janela > Registo de recuperação.

  2. Clique no menu pop‑up “Nível de detalhe” e, depois, escolha as mensagens que devem ser apresentadas:

    • Mostrar só erros: selecione esta opção para ver apenas as mensagens de erro.

    • Mostrar erros e progresso: selecione esta opção para ver as mensagens de erro e de progresso.

    • Mostrar todos os registos: selecione esta opção para ver todas as mensagens.

  3. Para filtrar as mensagens, digite o texto que pretende procurar no campo “Filtrar”.

  4. Para guardar o registo de recuperação, clique no botão “Guardar”, escolha uma localização para o ficheiro e, depois, clique em “Guardar”.

  5. Quando terminar de consultar o registo de recuperação, clique no botão “Fechar” no canto superior esquerdo da janela.

Criar um ficheiro de diagnóstico de recuperação

Se criar um relatório de problema através do Feedback Assistant, poderá ser‑lhe pedido que forneça um ficheiro de diagnóstico de recuperação. Pode criar o ficheiro e guardá‑lo num dispositivo de armazenamento externo.

  1. Ligue um dispositivo de armazenamento externo ao Mac.

  2. Prima as teclas Controlo + Opção + Shift + Comando + ponto final.

  3. Após uma intermitência do ecrã, aguarde até surgir uma caixa de diálogo.

  4. Use a caixa de diálogo para guardar o ficheiro de diagnóstico de recuperação no dispositivo de armazenamento externo.

Sair da Recuperação do macOS

O Mac é reiniciado automaticamente durante o restauro do sistema a partir do Time Machine ou durante a reinstalação do macOS. Noutros casos, pode ser necessário sair da Recuperação do macOS manualmente (por exemplo, se pretender efetuar uma cópia de segurança dos ficheiros antes de reinstalar o macOS ou se tiver acabado de alterar uma definição com o Utilitário de Segurança de Arranque).

No Mac, proceda de uma das seguintes formas:

  • Reiniciar o Mac: selecione o menu Apple > Reiniciar.

  • Selecionar um disco de arranque diferente antes de reiniciar o Mac: selecione o menu Apple > Disco de Arranque, selecione um disco de arranque e, depois, clique em “Reiniciar”.

  • Desligar o Mac: selecione o menu Apple > Desligar.
