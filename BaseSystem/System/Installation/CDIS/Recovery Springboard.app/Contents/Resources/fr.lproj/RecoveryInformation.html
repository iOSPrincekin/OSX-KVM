Utiliser la fonctionnalité de récupération de macOS sur un Mac avec puce Apple

La fonctionnalité de récupération de macOS est le système de récupération intégré de votre Mac.

Vous pouvez utiliser les apps disponibles dans la fonctionnalité de récupération de macOS sur un Mac avec puce Apple pour réparer votre disque interne, réinstaller macOS, restaurer vos fichiers à partir d’une sauvegarde Time Machine, définir le règlement de sécurité des différents disques, transférer des fichiers entre deux ordinateurs Mac, démarrer en mode sans échec, et plus encore.

La réinstallation de macOS nécessite une connexion à Internet. Pour vous connecter à Internet, vous pouvez utiliser une connexion réseau sans fil ou câblée. Consultez la rubrique Se connecter à Internet en Wi-Fi.

Remarque : si vous tentez de vous connecter à un portail captif (dans un café, une bibliothèque ou un hôtel, par exemple) ou à un réseau d’entreprise, il se peut que vous ne puissiez pas accéder à Internet dans la fonctionnalité de récupération de macOS.

Apps disponibles dans la fonctionnalité de récupération de macOS sur un Mac avec puce Apple

Les apps suivantes sont disponibles dans la fonctionnalité de récupération de macOS sur un Mac avec puce Apple :

  • Récupération : l’app Récupération permet d’accéder à des apps supplémentaires dans la fonctionnalité de récupération de macOS. Consultez les rubriques Démarrer l’ordinateur à partir de la fonctionnalité de récupération de macOS, Démarrer l’ordinateur à partir de la fonctionnalité de récupération de macOS et jumeler des périphériques d’entrée Bluetooth et Quitter la fonctionnalité de récupération de macOS.

  • Restauration système Time Machine : restaurez vos données à partir d’une sauvegarde Time Machine. Dans l’app Récupération, sélectionnez « Restaurer à partir de Time Machine », puis cliquez sur Continuer. Pour revenir à l’app Récupération, choisissez Restauration système Time Machine > Quitter la restauration système Time Machine. Consultez la rubrique Utiliser Time Machine pour restaurer le système.

  • Installer macOS : réinstallez macOS sur votre ordinateur. Dans l’app Récupération, sélectionnez « Réinstalller macOS », puis cliquez sur Continuer. Pour revenir à l’app Récupération, choisissez Installer macOS Big Sur > Quitter Installer macOS. Consultez la rubrique Réinstaller macOS.

  • Safari : dans l’app Récupération, sélectionnez Safari, puis cliquez sur Continuer. Pour revenir à l’app Récupération, choisissez Safari > Quitter Safari. Si vous êtes connecté à Internet, vous pouvez utiliser Safari pour rechercher de plus amples informations sur le site web d’assistance Apple. Notez que les modules et les extensions du navigateur Safari sont désactivés dans la fonctionnalité de récupération de macOS.

  • Utilitaire de disque : réparez ou effacez votre disque. Dans l’app Récupération, sélectionnez « Utilitaire de disque », puis cliquez sur Continuer. Pour revenir à l’app Récupération, choisissez Utilitaire de disque > Quitter Utilitaire de disque. Consultez les rubriques Effacer et réinstaller macOS et Réparer le périphérique de stockage interne.

  • Utilitaire Sécurité au démarrage : définissez les règlements de sécurité de votre Mac. Dans l’app Récupération, choisissez Utilitaires > Utilitaire Sécurité au démarrage. Pour revenir à l’app Récupération, choisissez Utilitaire Sécurité au démarrage > Quitter Utilitaire Sécurité au démarrage. Consultez la rubrique Définir le règlement de sécurité.

  • Terminal : modifiez des réglages via la ligne de commande. Dans l’app Récupération, choisissez Utilitaires > Terminal. Pour revenir à l’app Récupération, choisissez Terminal > Quitter Terminal.

  • Partager un disque : partagez le disque d’un Mac démarré à partir de la fonctionnalité de récupération de macOS. Dans l’app Récupération, choisissez Utilitaires > Partager un disque. Consultez la rubrique Utiliser « Partager un disque » pour transférer des fichiers entre deux ordinateurs Mac.

  • Disque de démarrage : définissez le disque de démarrage du Mac. Choisissez le menu Pomme  > Disque de démarrage. Pour quitter l’app, choisissez Disque de démarrage > Quitter Disque de démarrage.

Vous pouvez également utiliser la fonctionnalité de récupération de macOS pour effectuer les tâches suivantes :

  • Démarrer l’ordinateur en mode sans échec

  • Définir le volume de démarrage par défaut et redémarrer l’ordinateur

  • Consulter l’historique de récupération

  • Créer un fichier de diagnostic de récupération

Démarrer l’ordinateur à partir de la fonctionnalité de récupération de macOS

  1. Choisissez le menu Pomme  > Éteindre.

  2. Maintenez enfoncé le bouton d’alimentation de votre Mac jusqu’à ce que le message « Chargement des options de démarrage » apparaisse.

  3. Cliquez sur Options, puis sur Continuer.

  4. Saisissez le mot de passe d’un compte administrateur.

Démarrer l’ordinateur à partir de la fonctionnalité de récupération de macOS et jumeler des périphériques d’entrée Bluetooth

Procédez comme suit si vous devez jumeler un clavier, un trackpad ou une souris Bluetooth avec votre Mac lorsque vous démarrez à partir de la fonctionnalité de récupération de macOS.

  1. Choisissez le menu Pomme  > Éteindre.

  2. Maintenez enfoncé le bouton d’alimentation de votre Mac jusqu’à ce que le message « Chargement des options de démarrage » apparaisse.

  3. Lorsque le volume système et le bouton Options apparaissent, appuyez trois fois sur le bouton d’alimentation.

    Remarque : les différentes pressions sur le bouton d’alimentation doivent être espacées d’environ 1 seconde. Si vous appuyez trop rapidement sur le bouton d’alimentation, l’opération ne sera pas prise en compte, car les suites d’appuis rapides sont réservées à d’autres usages, comme VoiceOver.

  4. Utilisez Assistant réglages Bluetooth pour jumeler vos accessoires Bluetooth avec votre Mac.

    Remarque : Assistant réglages Bluetooth n’est pas disponible si un périphérique d’entrée est déjà connecté.

  5. Saisissez le mot de passe d’un compte administrateur.

Démarrer l’ordinateur en mode sans échec

  1. Choisissez le menu Pomme  > Éteindre.

  2. Maintenez enfoncé le bouton d’alimentation de votre Mac jusqu’à ce que le message « Chargement des options de démarrage » apparaisse.

  3. Sélectionnez un volume.

  4. Maintenez enfoncée la touche Maj, puis cliquez sur « Continuer en mode sans échec ».

    L’ordinateur redémarre automatiquement.

Définir le volume de démarrage par défaut et redémarrer l’ordinateur

  1. Choisissez le menu Pomme  > Éteindre.

  2. Maintenez enfoncé le bouton d’alimentation de votre Mac jusqu’à ce que le message « Chargement des options de démarrage » apparaisse.

  3. Sélectionnez un volume.

  4. Maintenez enfoncée la touche Option, puis cliquez sur « Toujours utiliser ».

    L’ordinateur redémarre automatiquement.

Se connecter à Internet en Wi-Fi

Si l’icône de menu Wi-Fi (dans le coin supérieur droit de l’écran) contient une ou plusieurs barres , cela signifie que le Wi-Fi est activé et connecté à un réseau sans fil. Si l’icône de menu Wi-Fi ne contient aucune barre , cela signifie que le Wi-Fi est activé mais n’est pas connecté à un réseau sans fil. Si l’icône de menu Wi-Fi est barrée , cela signifie que le Wi-Fi est désactivé. Le Wi-Fi doit être activé avant que vous puissiez vous connecter à un réseau sans fil.

Sur votre Mac, effectuez l’une des opérations suivantes :

  • Pour activer le Wi-Fi : cliquez sur l’icône de menu Wi-Fi, puis choisissez « Activer le Wi-Fi ».

  • Pour désactiver le Wi-Fi : cliquez sur l’icône de menu Wi-Fi, puis choisissez « Désactiver le Wi-Fi ».

  • Pour vous connecter à un réseau : cliquez sur l’icône de menu Wi-Fi, puis choisissez un réseau. Si vous y êtes invité, saisissez le mot de passe du réseau.

  • Pour vous connecter à un réseau fermé : cliquez sur l’icône de menu Wi-Fi, puis choisissez « Se connecter à un autre réseau ». Saisissez le nom du réseau et, si vous y êtes invité, le mot de passe pour y accéder.

Réparer le périphérique de stockage interne

Si vous rencontrez des problèmes avec votre ordinateur ou si la fenêtre de la fonctionnalité de récupération de macOS s’est affichée lors de son démarrage, il se peut que vous deviez réparer son périphérique de stockage interne (par exemple, un SSD).

  1. Dans l’app Récupération, sélectionnez « Utilitaire de disque », puis cliquez sur Continuer.

  2. Dans l’app Utilitaire de disque, choisissez Présentation > Afficher tous les appareils.

  3. Dans la barre latérale, sélectionnez un volume, puis cliquez sur le bouton S.O.S. .

  4. Dans la zone de dialogue S.O.S., cliquez sur Exécuter, puis suivez les instructions à l’écran.

  5. Une fois le processus S.O.S. terminé, cliquez sur Terminé.

  6. Recommencez les étapes 3 à 5 pour chaque volume du périphérique de stockage.

  7. Dans la barre latérale, sélectionnez un conteneur, puis cliquez sur le bouton S.O.S. .

  8. Dans la zone de dialogue S.O.S., cliquez sur Exécuter, puis suivez les instructions à l’écran.

  9. Une fois le processus S.O.S. terminé, cliquez sur Terminé.

  10. Recommencez les étapes 7 à 9 pour chaque conteneur du périphérique de stockage.

  11. Dans la barre latérale, sélectionnez le périphérique de stockage, puis cliquez sur le bouton S.O.S. .

  12. Dans la zone de dialogue S.O.S., cliquez sur Exécuter, puis suivez les instructions à l’écran.

  13. Une fois le processus S.O.S. terminé, cliquez sur Terminé.

Si Utilitaire de disque n’est pas en mesure de réparer votre disque, sauvegardez autant d’informations que possible, puis suivez les instructions de la rubrique Effacer et réinstaller macOS.

Pour revenir à l’app Récupération, choisissez Utilitaire de disque > Quitter Utilitaire de disque.

Utiliser Time Machine pour restaurer le système

Si vous avez déjà créé une sauvegarde Time Machine, utilisez « Restaurer à partir d’une sauvegarde Time Machine » pour rétablir l’état précédent de votre ordinateur.

Important : n’utilisez votre sauvegarde Time Machine que pour restaurer des données sur l’ordinateur qui est la source de cette sauvegarde. Si vous souhaitez transférer des données sur un nouvel ordinateur, quittez la fonctionnalité de récupération de macOS, redémarrez votre Mac, puis utilisez l’app Assistant migration dans macOS.

  1. Si votre sauvegarde se trouve sur une Time Capsule, vérifiez que vous êtes connecté à Internet.

  2. Dans l’app Récupération, sélectionnez « Restaurer à partir d’une sauvegarde Time Machine », puis cliquez sur Continuer.

  3. Dans l’app Restauration système Time Machine, suivez les instructions à l’écran.

Réinstaller macOS

Dans certains cas, il peut être nécessaire de réinstaller macOS. Cette réinstallation peut s’effectuer en gardant intacts vos fichiers et réglages utilisateur.

  1. Vérifiez que vous êtes connecté à Internet.

  2. Dans l’app Récupération, cliquez sur « Réinstalller macOS Big Sur », puis sur Continuer.

    Important : dans la sous-fenêtre permettant de sélectionner un disque, sélectionnez votre disque macOS actuel (dans la plupart des cas, il s’agit du seul disque disponible).

  3. Suivez les instructions à l’écran.

Effacer et réinstaller macOS

Si vous effacez puis réinstallez macOS, tous vos comptes utilisateur, réglages réseau, fichiers et dossiers sont supprimés de votre Mac. Avant d’effacer macOS, essayez de vous reconnecter à votre système, puis sauvegardez tous les fichiers que vous souhaitez conserver en les copiant sur un autre périphérique de stockage. Notez également les réglages réseau définis dans les préférences Réseau pour vous reconnecter plus facilement après la réinstallation de macOS.

  1. Vérifiez que vous êtes connecté à Internet.

  2. Dans l’app Récupération, cliquez sur « Utilitaire de disque », puis sur Continuer.

  3. Dans Utilitaire de disque, sélectionnez le volume que vous souhaitez effacer dans la barre latérale, puis cliquez sur le bouton Effacer dans la barre d’outils.

  4. Saisissez un nom pour le volume dans le champ Nom.

  5. Cliquez sur le menu local Format, choisissez APFS, puis cliquez sur « Effacer le groupe de volumes ».

  6. Une fois le processus d’effacement terminé, cliquez sur Terminé, puis choisissez Utilitaire de disque > Quitter Utilitaire de disque.

  7. Dans l’app Récupération, cliquez sur « Réinstalller macOS Big Sur », cliquez sur Continuer, puis suivez les instructions à l’écran.

Définir le règlement de sécurité

  1. Dans l’app Récupération, choisissez Utilitaires > Utilitaire Sécurité au démarrage.

  2. Sélectionnez le système que vous souhaitez utiliser pour définir le règlement de sécurité.

  3. Si le disque est chiffré avec FileVault, cliquez sur Déverrouiller, saisissez le mot de passe, puis cliquez sur Déverrouiller.

  4. Cliquez sur « Règlement de sécurité ».

  5. Examinez les options de sécurité suivantes :

    • Sécurité maximale : garantit que seul votre système d’exploitation actuel, ou votre logiciel de système d’exploitation signé actuellement approuvé par Apple, peut s’exécuter. Ce mode requiert une connexion réseau au moment de l’installation des logiciels.

    • Sécurité réduite : autorise l’exécution de toute version d’un logiciel de système d’exploitation signé ayant déjà été approuvé par Apple.

  6. Si nécessaire, sélectionnez « Sécurité réduite », saisissez votre nom d’utilisateur et votre mot de passe d’administrateur, puis effectuez l’une des opérations suivantes :

    • Cochez la case « Autoriser la gestion par les utilisateurs des extensions du noyau provenant de développeurs identifiés », pour autoriser l’installation de logiciels utilisant des extensions de noyau héritées.

    • Cochez la case « Autoriser la gestion à distance des extensions du noyau et des mises à jour automatiques de logiciels » pour autoriser la gestion à distance d’extensions de noyau héritées et des mises à jour de logiciels au moyen d’une solution de gestion des appareils mobiles (MDM).

  7. Cliquez sur OK.

  8. Redémarrez votre Mac pour que les modifications soient appliquées.

Utiliser « Partager un disque » pour transférer des fichiers entre deux ordinateurs Mac

Vous pouvez transférer des fichiers entre un Mac démarré à partir de la fonctionnalité de récupération de macOS et un autre Mac en partageant un volume et en connectant les deux ordinateurs à l’aide d’un câble USB, USB-C ou Thunderbolt.

  1. Connectez les deux ordinateurs à l’aide d’un câble USB, USB-C ou Thunderbolt.

  2. Dans l’app Récupération, choisissez Utilitaires > Partager un disque.

  3. Sélectionnez le volume à partager, puis cliquez sur « Démarrer le partage ».

  4. Sur l’autre Mac, ouvrez une fenêtre Finder, puis cliquez sur Réseau dans la barre latérale, sous Emplacements.

  5. Dans la fenêtre Réseau, cliquez deux fois sur le Mac contenant le disque partagé, cliquez sur « Se connecter comme », sélectionnez Invité dans la fenêtre « Se connecter comme », puis cliquez sur « Se connecter ».

  6. Transférez les fichiers.

  7. Une fois le transfert des fichiers terminé, éjectez le volume sur l’autre Mac.

Consulter l’historique de récupération

L’historique de récupération contient les messages consignés par la fonctionnalité de récupération de macOS. Vous pouvez choisir les types de messages à afficher, filtrer les messages pour rechercher un texte précis, et enregistrer l’historique de récupération sous forme de fichier.

  1. Dans l’app Récupération, choisissez Fenêtre > Historique de Récupération.

  2. Cliquez sur le menu local « Niveau de détail », puis choisissez les messages à afficher :

    • Afficher uniquement les erreurs : choisissez cette option pour n’afficher que les messages d’erreur.

    • Afficher les erreurs et la progression : choisissez cette option pour afficher les messages d’erreur et de progression.

    • Afficher tout l’historique : choisissez cette option pour afficher tous les messages.

  3. Pour filtrer les messages, saisissez le texte à rechercher dans le champ Filtre.

  4. Pour enregistrer l’historique de récupération, cliquez sur le bouton Enregistrer, choisissez un emplacement pour le fichier, puis cliquez sur Enregistrer.

  5. Lorsque vous avez terminé de consulter l’historique de récupération, cliquez sur le bouton Fermer  dans le coin supérieur gauche de la fenêtre.

Créer un fichier de diagnostic de récupération

Si vous signalez un problème à l’aide d’Assistant d’évaluation, il peut vous être demandé de fournir un fichier de diagnostic de récupération. Vous pouvez créer ce fichier et l’enregistrer sur un périphérique de stockage externe.

  1. Connectez un périphérique de stockage externe à votre Mac.

  2. Appuyez sur Contrôle + Option + Maj + Commande + Point.

  3. Après le clignotement de l’écran, patientez quelques instants jusqu’à ce qu’une zone de dialogue apparaisse.

  4. Utilisez la zone de dialogue pour enregistrer le fichier de diagnostic de récupération dans un emplacement de votre périphérique de stockage externe.

Quitter la fonctionnalité de récupération de macOS

Votre Mac redémarre automatiquement lors de la restauration de votre système depuis Time Machine ou lors de la réinstallation de macOS. Dans les autres cas, vous devez quitter manuellement la fonctionnalité de récupération de macOS (par exemple, si vous souhaitez sauvegarder vos fichiers avant la réinstallation de macOS ou si vous avez seulement modifié un réglage à l’aide d’Utilitaire Sécurité au démarrage).

Sur votre Mac, effectuez l’une des opérations suivantes :

  • Pour redémarrer votre Mac : choisissez le menu Pomme  > Redémarrer.

  • Pour choisir un autre disque de démarrage avant de redémarrer votre Mac : choisissez le menu Pomme  > Disque de démarrage, sélectionnez un disque de démarrage, puis cliquez sur Redémarrer.

  • Pour éteindre votre Mac : choisissez le menu Pomme  > Éteindre.
