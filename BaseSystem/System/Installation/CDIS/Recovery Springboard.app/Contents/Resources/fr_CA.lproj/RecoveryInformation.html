Utiliser la récupération de macOS sur un Mac avec puce Apple

La récupération de macOS est le système de récupération intégré à votre Mac.

Vous pouvez utiliser les apps de la récupération de macOS sur un Mac doté de la puce Apple pour réparer votre disque interne, réinstaller macOS, restaurer vos fichiers à partir d’une sauvegarde Time Machine, définir le règlement de sécurité pour différents disques, transférer des fichiers entre deux ordinateurs Mac, démarrer en mode sans échec et plus encore.

La réinstallation de macOS nécessite une connexion Internet. Vous pouvez vous connecter à Internet à l’aide d’une connexion réseau filaire ou sans fil. Consultez la rubrique Se connecter à Internet avec le Wi-Fi.

Remarque : Si vous essayez de vous connecter à un portail captif (comme dans un café, une bibliothèque ou un hôtel) ou à un réseau d’entreprise, vous pourriez ne pas avoir accès à Internet en mode de récupération de macOS.

Apps accessibles en mode de récupération de macOS sur un Mac avec puce Apple

Les apps suivantes sont accessibles en mode de récupération de macOS sur un Mac doté de la puce Apple :

  • Récupération : L’app Récupération donne accès à des apps supplémentaires en mode de récupération de macOS. Consultez les rubriques Démarrer l’ordinateur en mode de récupération de macOS, Démarrer l’ordinateur en mode de récupération de macOS et jumeler des appareils d’entrée Bluetooth et Quitter la récupération de macOS.

  • Restauration système Time Machine : Restaurez vos données à partir d’une sauvegarde Time Machine. Dans l’app Récupération, sélectionnez « Restaurer à partir de Time Machine », puis cliquez sur Continuer. Pour revenir à l’app Récupération, choisissez Restauration système Time Machine > Quitter la restauration système Time Machine. Consultez la rubrique Utiliser Time Machine pour restaurer le système.

  • Installer macOS : Réinstallez macOS sur votre ordinateur. Dans l’app Récupération, sélectionnez « Réinstaller macOS », puis cliquez sur Continuer. Pour revenir à l’app Récupération, choisissez Installer macOS Big Sur > Quitter Installer macOS. Consultez la rubrique Réinstaller macOS.

  • Safari : Dans l’app Récupération, sélectionnez Safari, puis cliquez sur Continuer. Pour revenir à l’app Récupération, choisissez Safari > Quitter Safari. Si vous êtes connecté à Internet, vous pouvez utiliser Safari pour rechercher plus d’informations sur le site Web de l’assistance Apple. Veuillez noter que les extensions et modules du navigateur Safari sont désactivés en mode de récupération de macOS.

  • Utilitaire de disque : Réparez ou effacez votre disque. Dans l’app Récupération, sélectionnez Utilitaire de disque, puis cliquez sur Continuer. Pour revenir à l’app Récupération, choisissez Utilitaire de disque > Quitter Utilitaire de disque. Consultez les rubriques Effacer et réinstaller macOS et Réparer le périphérique de stockage interne.

  • Utilitaire Sécurité au démarrage : Définissez les règlements de sécurité de votre Mac. Dans l’app Récupération, choisissez Utilitaires > Utilitaire Sécurité au démarrage. Pour revenir à l’app Récupération, choisissez Utilitaire Sécurité au démarrage > Quitter Utilitaire Sécurité au démarrage. Consultez la rubrique Définir le règlement de sécurité.

  • Terminal : Modifiez les réglages à l’aide de la ligne de commande. Dans l’app Récupération, choisissez Utilitaires > Terminal. Pour revenir à l’app Récupération, choisissez Terminal > Quitter Terminal.

  • Partage de disque : Partagez le disque d’un Mac démarré en mode de récupération de macOS. Dans l’app Récupération, choisissez Utilitaires > Partage de disque. Consultez la rubrique Utiliser le partage de disque pour transférer des fichiers entre deux ordinateurs Mac.

  • Disque de démarrage : Réglez le disque de démarrage du Mac. Choisissez le menu Pomme  > Disque de démarrage. Pour quitter l’app, choisissez Disque de démarrage > Quitter Disque de démarrage.

Vous pouvez aussi utiliser la récupération de macOS pour accomplir les tâches suivantes :

  • Démarrer l’ordinateur en mode sans échec

  • Régler le volume de démarrage et redémarrer l’ordinateur

  • Afficher l’historique de récupération

  • Créer un fichier de diagnostic de récupération

Démarrer l’ordinateur en mode de récupération de macOS

  1. Choisissez le menu Pomme  > Éteindre.

  2. Appuyez de façon prolongée sur le bouton d’alimentation de votre Mac jusqu’à l’affichage de « Chargement des options de démarrage ».

  3. Cliquez sur Options, puis sur Continuer.

  4. Saisissez le mot de passe d’un compte administrateur.

Démarrer l’ordinateur en mode de récupération de macOS et jumeler des appareils d’entrée Bluetooth

Suivez les instructions suivantes si vous devez jumeler un clavier, un pavé tactile ou une souris Bluetooth avec votre Mac lorsque vous démarrez en mode de récupération de macOS.

  1. Choisissez le menu Pomme  > Éteindre.

  2. Appuyez de façon prolongée sur le bouton d’alimentation de votre Mac jusqu’à l’affichage de « Chargement des options de démarrage ».

  3. Lorsque le volume système et le bouton Options s’affichent, appuyez trois fois sur le bouton d’alimentation.

    Remarque : Les appuis sur le bouton d’alimentation doivent être effectués à 1 seconde d’intervalle. Si vous appuyez trop rapidement, l’action ne sera pas détectée, car les appuis rapides sont réservés à d’autres fins, comme VoiceOver.

  4. Utilisez l’assistant réglages Bluetooth pour jumeler vos accessoires Bluetooth avec votre Mac.

    Remarque : L’assistant réglages Bluetooth n’apparaîtra pas si un appareil d’entrée est déjà connecté.

  5. Saisissez le mot de passe d’un compte administrateur.

Démarrer l’ordinateur en mode sans échec

  1. Choisissez le menu Pomme  > Éteindre.

  2. Appuyez de façon prolongée sur le bouton d’alimentation de votre Mac jusqu’à l’affichage de « Chargement des options de démarrage ».

  3. Sélectionnez un volume.

  4. Appuyez de façon prolongée sur la touche Maj, puis cliquez sur Continuer en mode sans échec.

    L’ordinateur redémarre automatiquement.

Régler le volume de démarrage et redémarrer l’ordinateur

  1. Choisissez le menu Pomme  > Éteindre.

  2. Appuyez de façon prolongée sur le bouton d’alimentation de votre Mac jusqu’à l’affichage de « Chargement des options de démarrage ».

  3. Sélectionnez un volume.

  4. Appuyez de façon prolongée sur la touche Option, puis cliquez sur Toujours utiliser.

    L’ordinateur redémarre automatiquement.

Se connecter à Internet avec le Wi-Fi

Si l’icône de menu Wi‑Fi (dans le coin supérieur droit de l’écran) contient une ou plusieurs barres , le Wi‑Fi est activé et connecté à un réseau sans fil. Si l’icône de menu Wi‑Fi ne contient aucune barre , le Wi‑Fi est activé, mais il n’est pas connecté à un réseau sans fil. Si une barre diagonale apparaît sur l’icône de menu Wi‑Fi , le Wi‑Fi est désactivé. Il faut activer celui-ci pour pouvoir accéder à un réseau sans fil.

Sur votre Mac, effectuez l’une des opérations suivantes :

  • Activer le Wi-Fi : Cliquez sur l’icône de menu Wi‑Fi, puis sélectionnez Activer le Wi‑Fi.

  • Désactiver le Wi-Fi : Cliquez sur l’icône de menu Wi‑Fi, puis sélectionnez Désactiver le Wi‑Fi.

  • Accéder à un réseau : Cliquez sur l’icône de menu Wi‑Fi, puis sélectionnez un réseau. Au besoin, saisissez le mot de passe du réseau.

  • Accéder à un réseau fermé : Cliquez sur l’icône de menu Wi‑Fi, puis sélectionnez « Se connecter à un autre réseau ». Saisissez le nom du réseau et, au besoin, son mot de passe.

Réparer le périphérique de stockage

Si vous avez des problèmes avec votre ordinateur ou si vous voyez la fenêtre Récupération de macOS après son démarrage, vous pourriez devoir réparer son périphérique de stockage interne, par exemple un disque SSD (Solid-State Drive).

  1. Dans l’app Récupération, sélectionnez Utilitaire de disque, puis cliquez sur Continuer.

  2. Dans l’app Utilitaire de disque, choisissez Présentation > Afficher tous les appareils.

  3. Dans la barre latérale, sélectionnez un volume, puis cliquez sur le bouton S.O.S. .

  4. Dans la zone de dialogue S.O.S., cliquez sur Exécuter, puis suivez les instructions à l’écran.

  5. Lorsque le processus S.O.S. est terminé, cliquez sur Terminé.

  6. Répétez les étapes 3 à 5 pour chaque volume du périphérique de stockage.

  7. Dans la barre latérale, sélectionnez un conteneur, puis cliquez sur le bouton S.O.S. .

  8. Dans la zone de dialogue S.O.S., cliquez sur Exécuter, puis suivez les instructions à l’écran.

  9. Lorsque le processus S.O.S. est terminé, cliquez sur Terminé.

  10. Répétez les étapes 7 à 9 pour chaque conteneur du périphérique de stockage.

  11. Dans la barre latérale, sélectionnez le périphérique de stockage, puis cliquez sur le bouton S.O.S. .

  12. Dans la zone de dialogue S.O.S., cliquez sur Exécuter, puis suivez les instructions à l’écran.

  13. Lorsque le processus S.O.S. est terminé, cliquez sur Terminé.

Si Utilitaire de disque ne peut pas réparer votre disque, sauvegardez vos données autant que possible, puis suivez les instructions de la rubrique Effacer et installer macOS.

Pour revenir à l’app Récupération de macOS, choisissez Utilitaire de disque > Quitter Utilitaire de disque.

Utiliser Time Machine pour restaurer le système

Si vous avez une sauvegarde Time Machine, utilisez « Restaurer à partir de Time Machine » pour rétablir l’état précédent de votre ordinateur.

Important : Utilisez uniquement votre sauvegarde Time Machine pour restaurer des données sur l’ordinateur d’où elle provient. Si vous voulez transférer des données de ce Mac à un nouvel ordinateur, quittez la récupération de macOS, redémarrez votre Mac, puis utilisez l’app Assistant migration dans macOS.

  1. Si votre sauvegarde se trouve sur une Time Capsule, vérifiez que vous êtes connecté à Internet.

  2. Dans l’app Récupération, sélectionnez « Restaurer à partir de Time Machine », puis cliquez sur Continuer.

  3. Dans l’app Restauration système Time Machine, suivez les instructions à l’écran.

Réinstaller macOS

Dans certaines circonstances, vous pourriez devoir réinstaller macOS. Vous pouvez réinstaller macOS tout en préservant les fichiers et les réglages utilisateur existants.

  1. Assurez-vous que vous êtes connecté à Internet.

  2. Dans l’app Récupération, cliquez sur « Réinstaller macOS Big Sur », puis sur Continuer.

    Important : Dans la sous-fenêtre permettant de sélectionner un disque, choisissez votre disque macOS actuel (dans la plupart des cas, c’est le seul disponible).

  3. Suivez les instructions à l’écran.

Effacer et réinstaller macOS

Si vous effacez et réinstallez macOS, votre compte d’utilisateur, vos réglages réseau ainsi que tous les fichiers et dossiers sur votre Mac seront supprimés. Avant d’effacer macOS, tentez d’ouvrir une session sur votre système, puis sauvegardez les fichiers que vous souhaitez conserver en les copiant vers un autre périphérique de stockage. Prenez aussi en note vos réglages dans les préférences Réseau afin de pouvoir vous reconnecter facilement après la réinstallation de macOS.

  1. Assurez-vous que vous êtes connecté à Internet.

  2. Dans l’app Récupération, cliquez sur Utilitaire de disque, puis sur Continuer.

  3. Dans Utilitaire de disque, sélectionnez le volume à effacer dans la barre latérale, puis cliquez sur le bouton Effacer dans la barre d’outils.

  4. Saisissez un nom pour le volume dans le champ Nom.

  5. Cliquez sur le menu contextuel Format, choisissez APFS, puis cliquez sur Supprimer le groupe de volumes.

  6. Lorsque l’effacement est terminé, cliquez sur Terminé, puis choisissez Utilitaire de disque > Quitter Utilitaire de disque.

  7. Dans l’app Récupération, cliquez sur Réinstaller macOS Big Sur, cliquez sur Continuer, puis suivez les instructions à l’écran.

Définir le règlement de sécurité

  1. Dans l’app Récupération, choisissez Utilitaires > Utilitaire Sécurité au démarrage.

  2. Sélectionnez le système à utiliser pour définir le règlement de sécurité.

  3. Si le disque est chiffré avec FileVault, cliquez sur Déverrouiller, saisissez le mot de passe, puis cliquez sur Déverrouiller.

  4. Cliquez sur Règlement de sécurité.

  5. Considérez les options de sécurité suivantes :

    • Sécurité maximale : Faites en sorte que seul votre système d’exploitation actuel, ou un logiciel de système d’exploitation signé auquel Apple fait actuellement confiance, peut s’exécuter. Ce mode requiert une connexion réseau au moment de l’installation du logiciel.

    • Sécurité réduite : Autorisez n’importe quelle version de système d’exploitation signé auxquels Apple a déjà fait confiance.

  6. Au besoin, sélectionnez Sécurité réduite, saisissez votre nom d’utilisateur et votre mot de passe d’administrateur, puis effectuez l’une des opérations suivantes :

    • Cochez la case « Autoriser la gestion par les utilisateurs des extensions du noyau de développeurs identifiés » pour autoriser l’installation d’un logiciel qui utilise d’anciennes extensions de noyau.

    • Cochez la case « Autoriser la gestion à distance des extensions du noyau et les mises à jour logicielles automatiques » pour autoriser la gestion à distance des anciennes extensions de noyau et les mises à jour logicielles au moyen d’une solution de gestion des appareils mobiles (GAM).

  7. Cliquez sur OK.

  8. Redémarrez votre Mac pour appliquer les changements.

Utiliser le partage de disque pour transférer des fichiers entre deux ordinateurs Mac

Vous pouvez transférer des fichiers entre un Mac démarré en mode de récupération de macOS et un autre en partageant un volume et en connectant les deux ordinateurs à l’aide d’un câble USB, USB‑C ou Thunderbolt.

  1. Connectez les deux ordinateurs avec un câble USB, USB‑C ou Thunderbolt.

  2. Dans l’app Récupération, choisissez Utilitaires > Partage de disque.

  3. Sélectionnez le volume que vous voulez partager, puis cliquez sur Commencer le partage.

  4. Sur l’autre Mac, ouvrez une fenêtre du Finder, puis cliquez sur Réseau dans la barre latérale sous Emplacements.

  5. Dans la fenêtre Réseau, double-cliquez sur le Mac du disque partagé, cliquez sur « Se connecter comme », sélectionnez Invité dans la fenêtre, puis cliquez sur Connecter.

  6. Transférez les fichiers.

  7. Lorsque le transfert est terminé, éjectez le volume de l’autre Mac.

Afficher l’historique de récupération

L’historique de récupération contient des messages journalisés par la récupération de macOS. Vous pouvez indiquer les types de messages à afficher, filtrer les messages pour rechercher une expression précise et enregistrer l’historique de récupération sur un fichier.

  1. Dans l’app Récupération, choisissez Fenêtre > Historique de récupération.

  2. Cliquez sur le menu contextuel Niveau de détail, puis choisissez les messages à afficher :

    • Afficher uniquement les erreurs : Choisissez cette option pour afficher uniquement les messages d’erreur.

    • Afficher les erreurs et la progression : Choisissez cette option pour afficher les messages d’erreur et les messages de progression.

    • Afficher tout l’historique : Choisissez cette option pour afficher tous les messages.

  3. Pour filtrer les messages, saisissez le texte à rechercher dans le champ de filtre.

  4. Pour enregistrer l’historique de récupération, cliquez sur le bouton Enregistrer, choisissez un emplacement pour le fichier, puis cliquez sur Enregistrer.

  5. Lorsque vous avez fini de consulter l’historique de récupération, cliquez sur le bouton Fermer  dans le coin supérieur gauche de la fenêtre.

Créer un fichier de diagnostic de récupération

Si vous signalez un problème à l’aide de l’assistant de commentaire, on pourrait vous demander de fournir un fichier de diagnostic de récupération. Vous pouvez créer le fichier et l’enregistrer sur un périphérique de stockage externe.

  1. Connectez un périphérique de stockage externe à votre Mac.

  2. Appuyez sur les touches Commande + Option + Maj + Commande + Point.

  3. Lorsque l’écran commence à clignoter, attendez quelques instants et une zone de dialogue s’affichera.

  4. Utilisez la zone de dialogue pour enregistrer le fichier de diagnostic de récupération vers un emplacement de votre périphérique de stockage externe.

Quitter la récupération de macOS

Votre Mac redémarre automatiquement pendant la restauration de votre système à partir de Time Machine ou la réinstallation de macOS. Dans les autres cas, vous devez quitter manuellement la récupération de macOS (par exemple, si vous voulez sauvegarder vos fichiers avant de réinstaller macOS ou si vous avez simplement modifié un réglage à l’aide d’Utilitaire Sécurité au démarrage).

Sur votre Mac, effectuez l’une des opérations suivantes :

  • Redémarrer le Mac : Choisissez le menu Pomme  > Redémarrer.

  • Choisir un autre disque de démarrage avant de redémarrer le Mac : Choisissez le menu Pomme  > Disque de démarrage, sélectionnez un disque de démarrage, puis cliquez sur Redémarrer.

  • Éteindre le Mac : Choisissez le menu Pomme  > Éteindre.
