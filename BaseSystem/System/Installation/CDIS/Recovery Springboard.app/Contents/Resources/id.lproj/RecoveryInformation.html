Menggunakan Pemulihan macOS di Mac dengan Apple silicon

Pemulihan macOS adalah sistem pemulihan internal Mac Anda.

Anda dapat menggunakan app di Pemulihan macOS di Mac dengan Apple silicon untuk memperbaiki disk internal Anda, menginstal ulang macOS, memulihkan file Anda dari cadangan Time Machine, mengatur kebijakan keamanan untuk beberapa disk yang berbeda, mentransfer file di antara dua komputer Mac, memulai dalam mode aman, dan lainnya.

Penginstalan ulang macOS memerlukan koneksi internet. Untuk terhubung ke internet, Anda dapat menggunakan koneksi jaringan nirkabel atau berkabel. Lihat Terhubung ke internet menggunakan Wi-Fi.

Catatan: Jika Anda mencoba terhubung ke portal terikat (misalnya, di kedai kopi, perpustakaan, atau hotel) atau jaringan perusahaan, Anda mungkin tidak dapat mengakses internet di Pemulihan macOS.

App yang tersedia di Pemulihan macOS di Mac dengan Apple silicon

App berikut tersedia di Pemulihan macOS di Mac dengan Apple silicon:

  • Pemulihan: App Pemulihan menyediakan akses ke app tambahan di Pemulihan macOS. Lihat Memulai komputer Anda di Pemulihan macOS, Memulai komputer Anda di Pemulihan macOS dan memasangkan perangkat input Bluetooth, dan Menutup Pemulihan macOS.

  • Pemulihan Sistem Time Machine: Memulihkan data Anda dari cadangan Time Machine. Di app Pemulihan, pilih Pulihkan Dari Time Machine, lalu klik Lanjutkan. Untuk kembali ke app Pemulihan, pilih Pemulihan Sistem Time Machine > Tutup Pemulihan Sistem Time Machine. Lihat Menggunakan Time Machine untuk memulihkan sistem Anda.

  • Instal macOS: Menginstal ulang macOS di komputer Anda. Di app Pemulihan, pilih Instal Ulang macOS, lalu klik Lanjutkan. Untuk kembali ke app Pemulihan, pilih Instal macOS Big Sur > Tutup Instal macOS. Lihat Menginstal ulang macOS.

  • Safari: Di app Pemulihan, pilih Safari, lalu klik Lanjutkan. Untuk kembali ke app Pemulihan, pilih Safari > Tutup Safari. Jika Anda terhubung ke internet, Anda dapat menggunakan Safari untuk mencari informasi lainnya di situs web Dukungan Apple. Harap diingat bahwa plug-in dan ekstensi browser Safari dinonaktifkan di Pemulihan macOS.

  • Utilitas Disk: Memperbaiki atau menghapus disk Anda. Di app Pemulihan, pilih Utilitas Disk, lalu klik Lanjutkan. Untuk kembali ke app Pemulihan, pilih Utilitas Disk > Tutup Utilitas Disk. Lihat Menghapus dan menginstal ulang macOS serta Memperbaiki perangkat penyimpanan internal Anda.

  • Utilitas Keamanan Mulai: Mengatur kebijakan keamanan untuk Mac Anda. Di app Pemulihan, pilih Utilitas > Utilitas Keamanan Mulai. Untuk kembali ke app Pemulihan, pilih Utilitas Keamanan Mulai > Tutup Utilitas Keamanan Mulai. Lihat Mengatur kebijakan keamanan.

  • Terminal: Mengubah pengaturan melalui baris perintah. Di app Pemulihan, pilih Utilitas > Terminal. Untuk kembali ke app Pemulihan, pilih Terminal > Tutup Terminal.

  • Bagikan Disk: Membagikan disk Mac yang dimulai di Pemulihan macOS. Di app Pemulihan, pilih Utilitas > Bagikan Disk. Lihat Menggunakan Bagikan Disk untuk mentransfer file di antara dua komputer Mac.

  • Disk Mulai: Mengatur disk mulai untuk Mac. Pilih menu Apple > Disk Mulai. Untuk menutup app, pilih Disk Mulai > Tutup Disk Mulai.

Anda juga dapat menggunakan Pemulihan macOS untuk melakukan tugas berikut:

  • Memulai komputer Anda dalam mode aman

  • Mengatur volume mulai default dan memulai ulang komputer

  • Melihat log pemulihan

  • Membuat file diagnostik pemulihan

Memulai komputer Anda dalam Pemulihan macOS

  1. Pilih menu Apple > Matikan.

  2. Tekan dan tahan tombol daya di Mac Anda hingga Anda melihat “Memuat pilihan mulai”.

  3. Klik Pilihan, lalu klik Lanjutkan.

  4. Masukkan kata sandi untuk akun administrator.

Memulai komputer Anda di Pemulihan macOS dan memasangkan perangkat input Bluetooth

Ikuti instruksi ini jika Anda perlu memasangkan papan ketik, trackpad, atau tetikus Bluetooth dengan Mac saat Anda memulai di Pemulihan macOS.

  1. Pilih menu Apple > Matikan.

  2. Tekan dan tahan tombol daya di Mac Anda hingga Anda melihat “Memuat pilihan mulai”.

  3. Setelah volume sistem dan tombol Pilihan muncul, tekan tombol daya tiga kali.

    Catatan: Setiap penekanan tombol daya harus dilakukan dalam 1 detik setelah penekanan sebelumnya. Jika Anda menekan tombol daya terlalu cepat, penekanan tidak akan terdeteksi karena penekanan cepat ditujukan untuk tujuan lain, seperti VoiceOver.

  4. Gunakan Asisten Pengaturan Bluetooth untuk memasangkan aksesori Bluetooth dengan Mac Anda.

    Catatan: Anda tidak akan melihat Asisten Pengaturan Bluetooth jika sudah memiliki perangkat input yang terhubung.

  5. Masukan kata sandi untuk akun administrator.

Memulai komputer Anda dalam mode aman

  1. Pilih menu Apple > Matikan.

  2. Tekan dan tahan tombol daya di Mac Anda hingga Anda melihat “Memuat pilihan mulai”.

  3. Pilih volume.

  4. Tekan dan tahan tombol Shift, lalu klik Lanjutkan dalam mode aman.

    Komputer dimulai ulang secara otomatis.

Mengatur volume mulai default dan memulai ulang komputer

  1. Pilih menu Apple > Matikan.

  2. Tekan dan tahan tombol daya di Mac Anda hingga Anda melihat “Memuat pilihan mulai”.

  3. Pilih volume.

  4. Tekan dan tahan tombol Option, lalu klik Selalu Gunakan.

    Komputer dimulai ulang secara otomatis.

Terhubung ke internet menggunakan Wi-Fi

Jika ikon menu Wi-Fi (di pojok kanan atas layar) berisi satu atau beberapa bar , itu berarti Wi-Fi dinyalakan dan terhubung ke jaringan nirkabel. Jika ikon menu Wi-Fi tidak berisi bar , itu berarti Wi-Fi dinyalakan tetapi tidak terhubung ke jaringan nirkabel. Jika ikon menu Wi-Fi memiliki garis miring melaluinya , itu berarti Wi-Fi dimatikan. Wi-Fi perlu dinyalakan sebelum Anda dapat bergabung dengan jaringan nirkabel.

Di Mac Anda, lakukan salah satu hal berikut:

  • Menyalakan Wi-Fi: Klik ikon menu Wi-Fi, lalu pilih Nyalakan Wi-Fi.

  • Mematikan Wi-Fi: Klik ikon menu Wi-Fi, lalu pilih Matikan Wi-Fi.

  • Bergabung dengan jaringan: Klik ikon menu Wi-Fi, lalu pilih jaringan. Jika diminta, ketik kata sandi jaringan.

  • Bergabung dengan jaringan tertutup: Klik ikon menu Wi-Fi, lalu pilih Bergabung dengan Jaringan Lain. Masukkan nama jaringan dan jika diminta, kata sandi jaringan.

Memperbaiki perangkat penyimpanan internal Anda

Jika Anda mengalami masalah dengan komputer Anda atau jika Anda memulai komputer Anda dan melihat jendela Pemulihan macOS, Anda mungkin harus memperbaiki perangkat penyimpanan internal komputer Anda (seperti solid-state drive (SSD)).

  1. Di app Pemulihan, pilih Utilitas Disk, lalu klik Lanjutkan.

  2. Di app Utilitas Disk, pilih Lihat > Tampilkan Semua Perangkat.

  3. Di bar samping, pilih volume, lalu klik tombol Pertolongan Pertama .

  4. Di dialog Pertolongan Pertama, klik Jalankan, lalu ikuti instruksi pada layar.

  5. Setelah proses Pertolongan Pertama selesai, klik Selesai.

  6. Ulangi langkah 3 hingga 5 untuk setiap volume di perangkat penyimpanan.

  7. Di bar samping, pilih wadah, lalu klik tombol Pertolongan Pertama .

  8. Di dialog Pertolongan Pertama, klik Jalankan, lalu ikuti instruksi pada layar.

  9. Setelah proses Pertolongan Pertama selesai, klik Selesai.

  10. Ulangi langkah 7 hingga 9 untuk setiap wadah di perangkat penyimpanan.

  11. Di bar samping, pilih perangkat penyimpanan, lalu klik tombol Pertolongan Pertama .

  12. Di dialog Pertolongan Pertama, klik Jalankan, lalu ikuti instruksi pada layar.

  13. Setelah proses Pertolongan Pertama selesai, klik Selesai.

Jika Utilitas Disk tidak dapat memperbaiki disk Anda, cadangkan sebanyak mungkin informasi, lalu ikuti instruksi di Menghapus dan menginstal ulang macOS.

Untuk kembali ke app Pemulihan, pilih Utilitas Disk > Tutup Utilitas Disk.

Menggunakan Time Machine untuk memulihkan sistem Anda

Jika Anda sebelumnya membuat cadangan Time Machine, gunakan Pulihkan dari Cadangan Time Machine untuk memulihkan semua hal di komputer Anda ke kondisi sebelumnya.

Penting: Hanya gunakan cadangan Time Machine Anda untuk memulihkan informasi ke komputer yang merupakan sumber cadangan. Jika Anda ingin mentransfer informasi dari Mac ini ke komputer baru, tutup Pemulihan macOS, mulai ulang Mac Anda, lalu gunakan app Asisten Migrasi di macOS.

  1. Jika cadangan Anda berada di Time Capsule, pastikan Anda terhubung ke internet.

  2. Di app Pemulihan, pilih Pulihkan Dari Cadangan Time Machine, lalu klik Lanjutkan.

  3. Di app Pemulihan Sistem Time Machine, ikuti instruksi pada layar.

Menginstal ulang macOS

Dalam beberapa situasi, Anda mungkin harus menginstal ulang macOS. Anda dapat menginstal ulang macOS sambil menjaga agar file dan pengaturan pengguna Anda tetap tersimpan.

  1. Pastikan Anda terhubung ke internet.

  2. Di app Pemulihan, klik Instal Ulang macOS Big Sur, lalu klik Lanjutkan.

    Penting: Di panel tempat Anda memilih disk, pilih disk macOS Anda saat ini (di sebagian besar kasus, hanya disk yang tersedia).

  3. Ikuti instruksi pada layar.

Menghapus dan menginstal ulang macOS

Jika Anda menghapus dan menginstal ulang macOS, akun pengguna, pengaturan jaringan, dan semua file serta folder di Mac Anda akan dihapus. Sebelum Anda menghapus macOS, coba masuk kembali ke sistem Anda, lalu cadangkan file yang ingin Anda simpan dengan menyalinnya ke perangkat penyimpanan lain. Selain itu, catat pengaturan jaringan Anda dari preferensi Jaringan agar memudahkan Anda untuk terhubung lagi setelah menginstal ulang macOS.

  1. Pastikan Anda terhubung ke internet.

  2. Di app Pemulihan, klik Utilitas Disk, lalu klik Lanjutkan.

  3. Di Utilitas Disk, pilih volume yang ingin Anda hapus di bar samping, lalu klik tombol Hapus di bar alat.

  4. Ketik nama untuk volume di bidang Nama.

  5. Klik menu pop-up Format, pilih APFS, lalu klik Hapus Grup Volume.

  6. Setelah proses penghapusan selesai, klik Selesai, lalu pilih Utilitas Disk > Tutup Utilitas Disk.

  7. Di app Pemulihan, klik Instal Ulang macOS Big Sur, klik Lanjutkan, lalu ikuti instruksi pada layar.

Mengatur kebijakan keamanan

  1. Di app Pemulihan, pilih Utilitas > Utilitas Keamanan Mulai.

  2. Pilih sistem yang ingin Anda gunakan untuk mengatur kebijakan keamanan.

  3. Jika disk dienkripsi dengan FileVault, klik Buka, masukkan kata sandi, lalu klik Buka.

  4. Klik Kebijakan Keamanan.

  5. Tinjau pilihan keamanan berikut:

    • Keamanan Penuh: Memastikan hanya OS Anda saat ini, atau perangkat lunak sistem operasi bertanda tangan yang saat ini dipercayai oleh Apple, yang dapat dijalankan. Mode ini memerlukan koneksi jaringan saat penginstalan perangkat lunak.

    • Keamanan Dikurangi: Mengizinkan semua versi perangkat lunak sistem operasi bertanda tangan yang pernah dipercayai oleh Apple untuk dijalankan.

  6. Jika perlu, pilih Keamanan Dikurangi, masukkan nama pengguna dan kata sandi administrator Anda, lalu lakukan salah satu hal berikut:

    • Pilih kotak centang “Izinkan manajemen pengguna ekstensi kernel dari pengembang yang dikenal” untuk mengizinkan penginstalan perangkat lunak yang menggunakan ekstensi kernel versi lama.

    • Pilih kotak centang “Izinkan manajemen jarak jauh ekstensi kernel dan pembaruan perangkat lunak otomatis” untuk mengesahkan manajemen jarak jauh ekstensi kernel versi lama serta pembaruan perangkat lunak menggunakan solusi mobile device management (MDM).

  7. Klik OKE.

  8. Mulai ulang Mac Anda agar perubahan diterapkan.

Menggunakan Bagikan Disk untuk mentransfer file di antara dua komputer Mac

Anda dapat mentransfer file di antara Mac yang di-boot ke Pemulihan macOS dan Mac lain dengan membagikan volume serta menyambungkan kedua komputer dengan kabel USB, USB-C, atau Thunderbolt.

  1. Sambungkan kedua komputer dengan kabel USB, USB-C, atau Thunderbolt.

  2. Di app Pemulihan, pilih Utilitas > Bagikan Disk.

  3. Pilih volume yang ingin Anda bagikan, lalu klik Mulai Berbagi.

  4. Di Mac lain, buka jendela Finder, lalu klik Jaringan di bar samping di bawah Lokasi.

  5. Di jendela Jaringan, klik dua kali Mac yang memiliki disk bersama, klik Terhubung Sebagai, pilih Tamu di jendela Terhubung Sebagai, lalu klik Hubungkan.

  6. Transfer file.

  7. Setelah selesai mentransfer file, keluarkan volume di Mac lain.

Melihat log pemulihan

Log pemulihan berisi pesan yang dicatat oleh Pemulihan macOS. Anda dapat menentukan jenis pesan yang akan ditampilkan, memfilter pesan untuk mencari teks tertentu, dan menyimpan log pemulihan ke file.

  1. Di app Pemulihan, pilih Jendela > Log Pemulihan.

  2. Klik menu pop-up Level Detail, lalu pilih pesan mana yang ditampilkan:

    • Hanya Tampilkan Kesalahan: Pilih opsi ini untuk hanya menampilkan pesan kesalahan.

    • Tampilkan Kesalahan dan Kemajuan: Pilih opsi ini untuk menampilkan pesan kesalahan dan kemajuan.

    • Tampilkan Semua Log: Pilih opsi ini untuk menampilkan semua pesan.

  3. Untuk memfilter pesan, ketik teks yang ingin Anda cari di bidang Filter.

  4. Untuk menyimpan log pemulihan, klik tombol Simpan, pilih lokasi untuk file, lalu klik Simpan.

  5. Setelah Anda selesai melihat log pemulihan, klik tombol Tutup di pojok kiri atas jendela.

Membuat file diagnostik pemulihan

Jika Anda melaporkan masalah menggunakan Asisten Umpan Balik, Anda mungkin diminta untuk menyediakan file diagnostik pemulihan. Anda dapat membuat file dan menyimpannya ke perangkat penyimpanan eksternal.

  1. Sambungkan perangkat penyimpanan eksternal ke Mac Anda.

  2. Tekan Control-Option-Shift-Command-Titik.

  3. Setelah layar berkedip, tunggu beberapa saat hingga dialog muncul.

  4. Gunakan dialog untuk menyimpan file diagnostik pemulihan ke lokasi di perangkat penyimpanan eksternal Anda.

Menutup Pemulihan macOS

Mac Anda dimulai ulang secara otomatis saat memulihkan sistem Anda dari Time Machine atau menginstal ulang macOS. Dalam kasus lain, Anda harus menutup Pemulihan macOS secara manual (misalnya, jika Anda ingin mencadangkan file Anda sebelum menginstal ulang macOS, atau jika Anda baru mengubah pengaturan menggunakan Utilitas Keamanan Mulai).

Di Mac Anda, lakukan salah satu hal berikut:

  • Memulai ulang Mac Anda: Pilih menu Apple > Mulai Ulang.

  • Memilih disk mulai berbeda sebelum memulai ulang Mac Anda: Pilih menu Apple > Disk Mulai, pilih disk mulai, lalu klik Mulai Ulang.

  • Mematikan Mac Anda: Pilih menu Apple > Matikan.
