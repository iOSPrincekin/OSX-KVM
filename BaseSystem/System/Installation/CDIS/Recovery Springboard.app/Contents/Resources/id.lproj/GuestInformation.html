Informasi mengenai pengguna tamu di Mac

Saat Anda masuk sebagai tamu di Mac yang menyalakan FileVault, Anda hanya dapat menggunakan Safari. Anda tidak akan dapat menggunakan app lain di Mac.

Saat Anda keluar, file apa pun yang Anda buat akan dihapus.

Terhubung ke internet

Jika ikon menu Wi-Fi (di pojok kanan atas layar) berisi satu atau beberapa bar , itu berarti Wi-Fi dinyalakan dan terhubung ke jaringan nirkabel. Jika ikon menu Wi-Fi tidak berisi bar , itu berarti Wi-Fi dinyalakan tetapi tidak terhubung ke jaringan nirkabel. Jika ikon menu Wi-Fi memiliki garis miring melaluinya , itu berarti Wi-Fi dimatikan. Wi-Fi perlu dinyalakan sebelum Anda dapat bergabung dengan jaringan nirkabel.

Di Mac Anda, lakukan salah satu hal berikut:

  • Menyalakan atau mematikan Wi-Fi: Klik ikon menu Wi-Fi, lalu klik pengalih Wi-Fi.

  • Bergabung dengan jaringan: Klik ikon menu Wi-Fi, lalu pilih jaringan. Jika diminta, ketik kata sandi jaringan.

Keluar dari akun Pengguna Tamu

Setelah Anda selesai menggunakan Mac, keluar dari akun Pengguna Tamu.

  1. Di Mac Anda, pilih menu Apple > Keluar dari Pengguna Tamu.

  2. Baca informasi di dialog peringatan, lalu lakukan salah satu hal berikut:

    • Keluar: Klik Hapus File & Keluar.

    • Menyalin file Anda: Klik Batalkan, lalu salin file Anda ke media penyimpanan lain, seperti flash drive eksternal.
