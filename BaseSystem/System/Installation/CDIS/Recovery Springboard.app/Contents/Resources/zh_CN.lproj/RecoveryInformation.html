在基于 Apple 芯片的 Mac 上使用 macOS 恢复

macOS 恢复是 Mac 内建的恢复系统。

您可以在基于 Apple 芯片的 Mac 上使用 macOS 恢复中的 App 来修复内置磁盘、重新安装 macOS、从时间机器备份恢复文件、为不同的磁盘设定安全策略、在两台 Mac 电脑之间传输文件以及以安全模式启动等等。

重新安装 macOS 需要互联网连接。若要接入互联网，您可以使用无线或有线网络连接。请参阅通过 Wi-Fi 接入互联网。

注：如果您尝试接入强制网络门户（例如在咖啡店、图书馆或酒店）或企业级网络，您可能无法在 macOS 恢复中访问互联网。

macOS 恢复中可用的 App（基于 Apple 芯片的 Mac）

以下 App 在 macOS 恢复中可用（基于 Apple 芯片的 Mac）：

  • 恢复：“恢复” App 可让您访问 macOS 恢复中的其他 App。请参阅在 macOS 恢复中启动您的电脑、在 macOS 恢复中启动您的电脑和配对蓝牙输入设备和退出 macOS 恢复。

  • 时间机器系统恢复：从时间机器备份恢复您的数据。在“恢复” App 中，选择“从时间机器恢复”，然后点按“继续”。若要返回“恢复” App，请选取“时间机器系统恢复”>“退出时间机器系统恢复”。请参阅使用时间机器恢复您的系统。

  • 安装 macOS：在电脑上重新安装 macOS。在“恢复” App 中，选择“重新安装 macOS”，然后点按“继续”。若要返回“恢复” App，请选取“安装 macOS Big Sur”>“退出安装 macOS”。请参阅重新安装 macOS。

  • Safari 浏览器：在“恢复” App 中，选择“Safari 浏览器”，然后点按“继续”。若要返回“恢复” App，请选取“Safari 浏览器”>“退出 Safari 浏览器”。如果接入了互联网，您可以使用 Safari 浏览器在 Apple 支持网站上搜索更多信息。请注意，Safari 浏览器插件和扩展在 macOS 恢复中被停用。

  • 磁盘工具：修复或抹掉磁盘。在“恢复” App 中，选择“磁盘工具”，然后点按“继续”。若要返回“恢复” App，请选取“磁盘工具”>“退出磁盘工具”。请参阅抹掉并重新安装 macOS 和修复内建储存设备。

  • 启动安全性实用工具：设定 Mac 的安全策略。在“恢复” App 中，选取“实用工具”>“启动安全性实用工具”。若要返回“恢复” App，请选取“启动安全性实用工具”>“退出启动安全性实用工具”。请参阅设定安全策略。

  • 终端：通过命令行更改设置。在“恢复” App 中，选取“实用工具”>“终端”。若要返回“恢复” App，请选取“终端”>“退出终端”。

  • 共享磁盘：共享在 macOS 恢复中启动的 Mac 磁盘。在“恢复” App 中，选取“实用工具”>“共享磁盘”。请参阅使用“共享磁盘”在两台 Mac 电脑之间传输文件。

  • 启动磁盘：设定 Mac 的启动磁盘。选取苹果菜单 >“启动磁盘”。若要退出该 App，请选取“启动磁盘”>“退出启动磁盘”。

您还可以使用 macOS 恢复执行以下任务：

  • 以安全模式启动您的电脑

  • 设定默认启动宗卷并重新启动您的电脑

  • 查看恢复日志

  • 创建恢复诊断文件

在 macOS 恢复中启动您的电脑

  1. 选取苹果菜单 >“关机”。

  2. 按住 Mac 上的电源按钮，直至看到“正在载入启动选项”。

  3. 点按“选项”，然后点按“继续”。

  4. 输入管理员帐户的密码。

在 macOS 恢复中启动您的电脑和配对蓝牙输入设备

在 macOS 恢复中启动时，如果想要将蓝牙键盘、触控板或鼠标与 Mac 配对，请按照以下指示操作。

  1. 选取苹果菜单 >“关机”。

  2. 按住 Mac 上的电源按钮，直至看到“正在载入启动选项”。

  3. 系统宗卷和“选项”按钮出现后，按下电源按钮三次。

    注：在前一次按下电源按钮后，后续每次按下都需要在 1 秒钟内完成。如果按下电源按钮速度太快，则不会寄存，因为快速按下用于其他用途，例如“旁白”。

  4. 使用“蓝牙设置助理”将蓝牙配件与 Mac 配对。

    注：如果已经连接了输入设备，则不会看到“蓝牙设置助理”。

  5. 输入管理员帐户的密码。

以安全模式启动您的电脑

  1. 选取苹果菜单 >“关机”。

  2. 按住 Mac 上的电源按钮，直至看到“正在载入启动选项”。

  3. 选择一个宗卷。

  4. 按住 Shift 键，然后在安全模式中点按“继续”。

    电脑会自动重新启动。

设定默认启动宗卷并重新启动您的电脑

  1. 选取苹果菜单 >“关机”。

  2. 按住 Mac 上的电源按钮，直至看到“正在载入启动选项”。

  3. 选择一个宗卷。

  4. 按住 Option 键，然后点按“始终使用”。

    电脑会自动重新启动。

通过 Wi-Fi 接入互联网

如果 Wi-Fi 菜单图标（位于屏幕右上角）包含一个或多个信号格 ，则说明 Wi-Fi 已打开且接入了无线网络。如果 Wi-Fi 菜单图标未包含任何信号格 ，则说明 Wi-Fi 已打开但未接入无线网络。如果 Wi-Fi 菜单图标上有一条斜杠 ，则说明 Wi-Fi 已关闭。首先需要打开 Wi-Fi 才能加入无线网络。

在 Mac 上，执行以下任一项操作：

  • 打开 Wi-Fi：点按 Wi-Fi 菜单图标，然后选取“打开 Wi-Fi”。

  • 关闭 Wi-Fi：点按 Wi-Fi 菜单图标，然后选取“关闭 Wi-Fi”。

  • 加入网络：点按 Wi-Fi 菜单图标，然后选取一个网络。如果需要，键入网络的密码。

  • 加入封闭网络：点按 Wi-Fi 菜单图标，然后选取“加入其他网络”。输入网络的名称，需要时再输入密码。

修复内建储存设备

如果您的电脑出现问题，或者在启动电脑时看到了 macOS 恢复窗口，您可能需要修复电脑的内建储存设备（例如固态硬盘 (SSD)）。

  1. 在“恢复” App 中，选择“磁盘工具”，然后点按“继续”。

  2. 在“磁盘工具” App 中，选取“显示”>“显示所有设备”。

  3. 在边栏中选择宗卷，然后点按“急救”按钮 。

  4. 在“急救”对话框中，点按“运行”，然后按照屏幕指示进行操作。

  5. “急救”进程完成后，点按“完成”。

  6. 针对储存设备上的每个宗卷重复步骤 3 到 5。

  7. 在边栏中选择容器，然后点按“急救”按钮 。

  8. 在“急救”对话框中，点按“运行”，然后按照屏幕指示进行操作。

  9. “急救”进程完成后，点按“完成”。

  10. 针对储存设备上的每个容器重复步骤 7 到 9。

  11. 在边栏中选择储存设备，然后点按“急救”按钮 。

  12. 在“急救”对话框中，点按“运行”，然后按照屏幕指示进行操作。

  13. “急救”进程完成后，点按“完成”。

如果“磁盘工具”无法修复您的磁盘，请备份尽可能多的信息，然后按照抹掉并重新安装 macOS 中的指示操作。

若要返回“恢复” App，请选取“磁盘工具”>“退出磁盘工具”。

使用时间机器恢复您的系统

如果此前已创建了时间机器备份，请使用“从时间机器备份进行恢复”将电脑上所有的内容恢复到其先前的状态。

重要事项：您的时间机器备份只能将信息恢复到作为备份来源的电脑上。如果您想要将这台 Mac 的信息传输到新电脑，请退出 macOS 恢复，重新启动 Mac，然后使用 macOS 中的“迁移助理” App。

  1. 如果备份位于时间返回舱上，请确保接入了互联网。

  2. 在“恢复” App 中，选择“从时间机器备份进行恢复”，然后点按“继续”。

  3. 在“时间机器系统恢复” App 中，按照屏幕指示操作。

重新安装 macOS

在某些情况下，您可能需要重新安装 macOS。您可以在重新安装 macOS 的同时保持文件和用户设置的完整。

  1. 请确保接入了互联网。

  2. 在“恢复” App 中，点按“重新安装 macOS Big Sur”，然后点按“继续”。

    重要事项：在选择磁盘的面板中，选择您当前的 macOS 磁盘（大多数情况下，这是唯一可用的磁盘）。

  3. 按照屏幕指示操作。

抹掉并重新安装 macOS

如果抹掉并重新安装 macOS，您 Mac 上的用户帐户、网络设置以及所有文件和文件夹都会被删除。抹掉 macOS 之前，请尝试重新登录您的系统，然后将要保留的任何文件拷贝到其他储存设备上进行备份。另请记下“网络”偏好设置中您网络的设置，以便在重新安装 macOS 后能轻松地重新接入网络。

  1. 请确保接入了互联网。

  2. 在“恢复” App 中，点按“磁盘工具”，然后点按“继续”。

  3. 在“磁盘工具”中，选择边栏中想要抹掉的宗卷，然后在工具栏中点按“抹掉”按钮。

  4. 在“名称”栏中键入宗卷的名称。

  5. 点按“格式”弹出式菜单，选取“APFS”，然后点按“抹掉宗卷组”。

  6. 抹掉进程完成后，点按“完成”，然后选取“磁盘工具”>“退出磁盘工具”。

  7. 在“恢复” App 中，点按“重新安装 macOS Big Sur”，点按“继续”，然后按照屏幕指示进行操作。

设定安全策略

  1. 在“恢复” App 中，选取“实用工具”>“启动安全性实用工具”。

  2. 选择要用来设定安全策略的系统。

  3. 如果磁盘已使用文件保险箱加密，请点按“解锁”，输入密码，然后点按“解锁”。

  4. 点按“安全策略”。

  5. 检查以下安全选项：

    • 完整安全性：确保只有当前的操作系统或者当前受 Apple 信任的签名操作系统软件才能运行。此模式在安装软件时需要网络连接。

    • 降低安全性：允许运行 Apple 信任过的任何版本的签名操作系统软件。

  6. 如果需要，请选择“降低安全性”，输入管理员用户名和密码，然后执行以下一项操作：

    • 选择“允许用户管理来自被认可开发者的内核扩展”复选框以允许安装使用旧版内核扩展的软件。

    • 选择“允许远程管理内核扩展和软件自动更新”复选框以授权使用移动设备管理 (MDM) 解决方案来远程管理旧版内核扩展和软件更新。

  7. 点按“好”。

  8. 重新启动您的 Mac 以使更改生效。

使用“共享磁盘”在两台 Mac 电脑之间传输文件

您可以在启动进入 macOS 恢复模式的 Mac 和另一台 Mac 之间传输文件，方法是通过共享宗卷并使用 USB、 USB-C 或雷雳线缆连接这两台电脑。

  1. 使用 USB、 USB-C 或雷雳线缆连接这两台电脑。

  2. 在“恢复” App 中，选取“实用工具”>“共享磁盘”。

  3. 选择想要共享的宗卷，然后点按“开始共享”。

  4. 在另一台 Mac 上，打开“访达”窗口，然后点按边栏中“位置”下方的“网络”。

  5. 在“网络”窗口中，连按共享了磁盘的 Mac，点按“连接身份”，在“连接身份”窗口中选择“客人”，然后点按“连接”。

  6. 传输文件。

  7. 文件传输完成后，在另一台 Mac 上推出宗卷。

查看恢复日志

恢复日志包含 macOS 恢复记录的信息。您可以指定显示的信息类型，过滤信息以搜索特定文本，以及将恢复日志存储到文件。

  1. 在“恢复” App 中，选取“窗口”>“恢复日志”。

  2. 点按“详细级别”弹出式菜单，然后选取要显示的信息：

    • 只显示错误：选取此选项以仅显示错误信息。

    • 显示错误和进程：选取此选项以显示错误信息和进程信息。

    • 显示所有日志：选取此选项以显示所有信息。

  3. 若要过滤信息，请在“过滤”栏中键入想要搜索的文本。

  4. 若要存储恢复日志，请点按“存储”按钮，选取文件的位置，然后点按“存储”。

  5. 查看完恢复日志后，请点按窗口左上角的“关闭”按钮 。

创建恢复诊断文件

如果您使用“反馈助理”提交问题报告，可能要求您提供恢复诊断文件。您可以创建该文件并将其存储到外置储存设备。

  1. 将外置储存设备连接到 Mac。

  2. 按下 Control-Option-Shift-Command-句号键。

  3. 屏幕闪烁后，等待片刻直到出现对话框。

  4. 使用该对话框将恢复诊断文件存储到外置储存设备上的位置。

退出 macOS 恢复

从时间机器恢复系统或重新安装 macOS 后，您的 Mac 会自动重新启动。在其他情况下，您需要手动退出 macOS 恢复（例如，如果想要在重新安装 macOS 前备份文件，或者您刚刚使用“启动安全性实用工具”更改了某项设置）。

在 Mac 上，执行以下一项操作：

  • 重新启动 Mac：选取苹果菜单 >“重新启动”。

  • 在重新启动 Mac 前，选取不同的启动磁盘：选取苹果菜单 >“启动磁盘”，选择一个启动磁盘，然后点按“重新启动”。

  • 将 Mac 关机：选取苹果菜单 >“关机”。
