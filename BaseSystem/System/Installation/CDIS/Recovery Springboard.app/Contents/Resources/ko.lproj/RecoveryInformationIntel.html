Intel 기반의 Mac에서 macOS 복구 사용하기

macOS 복구는 Mac의 내장 복구 시스템입니다.

Intel 기반 Mac의 macOS 복구에서 앱을 사용하여 내장 디스크 복구, macOS 재설치, Time Machine 백업으로 파일 복원, 보안 옵션 설정 등의 작업을 수행할 수 있습니다.

macOS를 다시 설치하려면 인터넷이 연결되어 있어야 합니다. 인터넷을 연결하려면 무선 및 유선 네트워크 연결을 사용해야 합니다. Wi-Fi를 사용하여 인터넷에 연결하기를 참조하십시오.

참고: 종속 포털(예: 커피숍, 도서관, 호텔 등) 또는 기업용 네트워크에 연결하려는 경우 macOS 복구에서 인터넷을 사용하지 못할 수도 있습니다.

Intel 기반 Mac의 macOS 복구에서 사용 가능한 앱

Intel 기반 Mac의 macOS 복구에서 다음 앱을 사용할 수 있습니다.

  • 복구: 복구 앱은 macOS 복구에서 컴퓨터를 시동할 때 열립니다. macOS 복구에서 Intel 기반의 Mac 시동하기 및 macOS 복구 종료하기를 참조하십시오.

  • Time Machine 시스템 복원: Time Machine 백업으로 데이터를 복원합니다. 복구 앱에서 ‘Time Machine으로부터 복원’을 선택한 다음 계속을 클릭하십시오. 복구 앱으로 돌아가려면 Time Machine 시스템 복원 > Time Machine 시스템 복원 종료를 선택하십시오. Time Machine을 사용하여 시스템 복원하기를 참조하십시오.

  • macOS 설치: macOS를 컴퓨터에서 다시 설치합니다. 복구 앱에서 ‘macOS 다시 설치’를 선택한 다음 계속을 클릭하십시오. 복구 앱으로 돌아가려면 macOS Big Sur 설치 > macOS 설치 종료를 선택하십시오. macOS 다시 설치하기를 참조하십시오.

  • Safari: 복구 앱에서 Safari를 선택한 다음 계속을 클릭하십시오. 복구 앱으로 돌아가려면 Safari > Safari 종료를 선택하십시오. 인터넷에 연결되어 있는 경우 Safari를 사용하여 Apple 지원 웹 사이트에 있는 더 많은 정보를 검색할 수 있습니다. Safari 브라우저 플러그인 및 확장 프로그램은 macOS 복구에서 비활성화됩니다.

  • 디스크 유틸리티: 디스크를 복구하거나 지웁니다. 복구 앱에서 디스크 유틸리티를 선택한 다음 계속을 클릭하십시오. 복구 앱으로 돌아가려면 디스크 유틸리티 > 디스크 유틸리티 종료를 선택하십시오. macOS 지우고 다시 설치하기를 참조하십시오.

  • 시동 보안 유틸리티: Mac에 대한 보안 정책을 설정합니다. 복구 앱에서 유틸리티 > 시동 보안 유틸리티를 선택하십시오. 복구 앱으로 돌아가려면 시동 보안 유틸리티 > 시동 보안 유틸리티 종료를 선택하십시오. 시동 보안 유틸리티를 사용하여 펌웨어 암호 설정하기 및 시동 보안 유틸리티를 사용하여 보안 시동 옵션 설정하기를 참조하십시오.

    시동 보안 유틸리티에서 사용 가능한 옵션은 사용자의 Mac 모델에 따라 다릅니다. Apple 지원 문서 시동 보안 유틸리티에 관하여 및 Apple T2 보안 칩이 탑재된 Mac 모델을 참조하십시오.

  • 시동 디스크: Mac에 대한 시동 디스크를 설정합니다. Apple 메뉴 > 시동 디스크를 선택하십시오. 복구 앱으로 돌아가려면 시동 디스크 > 시동 디스크 종료를 선택하십시오.

  • 터미널: 복구 앱에서 유틸리티 > 터미널을 선택하십시오. 복구 앱으로 돌아가려면 터미널 > 터미널 종료를 선택하십시오.

macOS 복구를 사용하여 다음 작업을 수행할 수도 있습니다.

  • 복구 진단 파일 생성하기

macOS 복구에서 Intel 기반의 Mac 시동하기

  1. Mac을 재시동하십시오.

  2. 시동 화면이 나타날 때까지 다음의 키 조합 중 하나를 즉시 길게 누르십시오.

    • Command-R: 내장된 macOS 복구 시스템에서 시동합니다. 이 키 조합을 사용하여 시스템에 설치된 최신 macOS를 다시 설치하거나 macOS 복구에서 다른 앱을 사용합니다.

    • Option-Command-R: 인터넷을 통해 macOS 복구에서 시동합니다. 이 키 조합을 사용하여 macOS를 다시 설치하고 사용자의 Mac과 호환되는 최신 버전의 macOS로 업그레이드하십시오.

    • Option-Shift-Command-R: 인터넷을 통해 macOS 복구에서 시동합니다. 이 키 조합을 사용하여 사용자의 Mac과 함께 제공되는 macOS 버전 또는 계속 사용할 수 있는 가장 가까운 버전을 다시 설치하십시오.

  3. 자물쇠가 보이는 경우 Mac의 암호를 입력하십시오.

  4. 디스크에 여러 개의 볼륨이 있는 경우 복구할 볼륨을 선택한 후 다음을 클릭하십시오.

  5. 필요한 경우 관리자 계정을 선택하고 다음을 클릭하여 계정 암호를 입력한 다음, 계속을 클릭하십시오.

Wi-Fi를 사용하여 인터넷에 연결하기

Wi-Fi 메뉴 아이콘(화면 오른쪽 상단 모서리)에 하나 이상의 막대가 있는 경우, Wi-Fi가 켜져 있고 무선 네트워크에 연결된 것입니다. Wi-Fi 메뉴 아이콘에 막대가 없는 경우 Wi-Fi가 켜져 있지만 무선 네트워크에 연결되지 않은 것입니다. Wi-Fi 메뉴 아이콘이 슬래시로 표시되어 있는 경우 Wi-Fi가 꺼진 것입니다. 무선 네트워크에 연결하려면 먼저 Wi-Fi를 켜야 합니다.

Mac에서 다음 중 하나를 수행하십시오.

  • Wi-Fi 켜기: Wi-Fi 메뉴 아이콘을 클릭한 다음, Wi-Fi 켜기를 선택하십시오.

  • Wi-Fi 끄기: Wi-Fi 메뉴 아이콘을 클릭한 다음, Wi-Fi 끄기를 선택하십시오.

  • 네트워크에 연결하기: Wi-Fi 메뉴 아이콘을 클릭한 다음, 네트워크를 선택하십시오. 필요한 경우에는 네트워크 암호를 입력하십시오.

  • 폐쇄형 네트워크에 연결하기: Wi-Fi 메뉴 아이콘을 클릭한 다음 ‘다른 네트워크에 연결’을 선택하십시오. 네트워크 이름을 입력하고 필요한 경우에는 네트워크 암호도 입력하십시오.

내장 저장 장치 복구하기

컴퓨터에 문제가 있거나 컴퓨터를 시작한 후에 macOS 복구 윈도우가 나타나는 경우 컴퓨터의 내장 저장 장치(솔리드 스테이트 드라이브(SSD) 등)를 복구해야 할 수 있습니다.

  1. 복구 앱에서 디스크 유틸리티를 선택한 다음, 계속을 클릭하십시오.

  2. 디스크 유틸리티 앱에서 보기 > 모든 기기 보기를 선택하십시오.

  3. 사이드바에서 볼륨을 선택한 다음, 검사/복구 버튼을 클릭하십시오.

  4. 검사/복구 대화상자에서 실행을 클릭한 다음, 화면상의 지침을 따르십시오.

  5. 검사/복구 작업이 완료되면 완료를 클릭하십시오.

  6. 저장 장치의 각 볼륨에서 3~5단계를 반복하십시오.

  7. 사이드바에서 컨테이너를 선택한 다음, 검사/복구 버튼을 클릭하십시오.

  8. 검사/복구 대화상자에서 실행을 클릭한 다음, 화면상의 지침을 따르십시오.

  9. 검사/복구 작업이 완료되면 완료를 클릭하십시오.

  10. 저장 장치의 각 컨테이너에서 7~9단계를 반복하십시오.

  11. 사이드바에서 저장 장치를 선택한 다음, 검사/복구 버튼을 클릭하십시오.

  12. 검사/복구 대화상자에서 실행을 클릭한 다음, 화면상의 지침을 따르십시오.

  13. 검사/복구 작업이 완료되면 완료를 클릭하십시오.

디스크 유틸리티가 디스크를 복구할 수 없는 경우 가능한 많은 정보를 백업한 다음, macOS를 지우고 다시 설치하기의 지침을 따르십시오.

macOS 복구 앱으로 돌아가려면 디스크 유틸리티 > 디스크 유틸리티 종료를 선택하십시오.

Time Machine을 사용하여 시스템 복원하기

이전에 Time Machine 백업을 생성한 경우 ‘Time Machine 백업으로부터 복원’을 사용하여 컴퓨터에 있는 모든 것을 이전 상태로 복원할 수 있습니다.

중요: 백업의 소스인 컴퓨터로 정보를 복원하려는 경우에만 Time Machine 백업을 사용하십시오. 이 Mac에서 새로운 컴퓨터로 정보를 전송하려는 경우 macOS 복구를 종료하고 Mac을 재시동한 다음 macOS에서 마이그레이션 지원 앱을 사용하십시오.

  1. 사용자의 백업이 Time Capsule에 있을 경우 인터넷에 연결되어 있는지 확인하십시오.

  2. 복구 앱에서 ‘Time Machine 백업으로부터 복원’을 선택한 다음, 계속을 클릭하십시오.

  3. Time Machine 시스템 복원 앱에서 화면상의 지침을 따르십시오.

시동 보안 유틸리티를 사용하여 펌웨어 암호 설정하기

펌웨어 암호를 설정하면, 펌웨어 암호를 입력하지 않은 상태로 사용자가 선택한 시동 디스크가 아닌 다른 내장 또는 외장 저장 장치에서 Mac을 시동할 수 없습니다. 시동 보안 유틸리티를 사용하여 펌웨어 암호를 켜고 끄거나 변경할 수 있습니다.

펌웨어 암호가 켜져 있으면 macOS 복구에서 시동하거나 시동 디스크 환경설정에서 선택된 저장 장치가 아닌 다른 저장 장치에서 시동할 때마다 펌웨어 암호를 입력해야 합니다.

Mac을 재시동한 후 펌웨어 암호에 대한 변경 사항이 적용됩니다.

  1. 복구 앱에서 유틸리티 > 시동 보안 유틸리티를 선택하십시오.

  2. 필요한 경우 관리자의 사용자 이름 및 암호를 입력하십시오.

  3. 다음 중 하나를 수행하십시오.

    • 펌웨어 암호 켜기: 펌웨어 암호 켜기를 클릭하여 새로운 암호를 입력하고 다시 입력하여 해당 암호를 확인한 다음, 암호 설정을 클릭하십시오.

    • 펌웨어 암호 끄기: 펌웨어 암호 끄기를 클릭하고 암호를 입력한 다음, 암호 끄기를 클릭하십시오.

    • 펌웨어 암호 변경하기: 암호 변경을 클릭하고 이전 암호를 입력한 후 새로운 암호를 입력하고 이를 확인한 다음, 암호 변경을 클릭하십시오.

  4. 시동 보안 유틸리티 종료를 클릭하십시오.

시동 보안 유틸리티를 사용하여 보안 시동 옵션 설정하기

보안 시동 옵션은 Apple T2 보안 칩이 탑재된 Mac 컴퓨터에서만 사용 가능합니다.

Mac을 재시동한 후 보안 시동 옵션에 대한 변경 사항이 적용됩니다.

  1. 복구 앱에서 유틸리티 > 시동 보안 유틸리티를 선택하십시오.

  2. 관리자의 사용자 이름 및 암호를 입력하십시오.

  3. 보안 시동 패널에서 다음의 보안 옵션 중 하나를 선택하십시오.

    • 완전 보안: 현재 OS 또는 Apple이 현재 신뢰하는 운영 체제에서 서명된 소프트웨어만 실행할 수 있습니다. 이 모드는 소프트웨어를 설치할 때 네트워크 연결이 필요합니다.

    • 중간 보안: Apple이 신뢰하는 운영 체제에서 서명된 소프트웨어의 모든 버전을 실행하도록 허용합니다.

    • 보안 없음: 시동 가능한 운영 체제에 어떤 요구 사항도 강제 적용하지 않습니다.

  4. 허용된 시동 미디어 패널에서 다음의 시동 옵션 중 하나를 선택하십시오.

    • 외부 또는 제거 가능한 미디어에서 시동 허용 안 함: USB 및 Thunderbolt 드라이브 또는 내부에서 연결된 PCIe 또는 SATA 드라이브와 같은 T2로 보호되지 않는 기기에서 시동하는 것을 제한합니다.

    • 외부 또는 제거 가능한 미디어에서 시동 허용: 어떤 기기에서도 시동에 제한이 없습니다.

  5. 시동 보안 유틸리티 > 시동 보안 유틸리티 종료를 선택하십시오.

macOS 다시 설치하기

일부의 경우 macOS를 다시 설치해야 할 수 있으며 사용자의 파일 및 사용자 설정을 그대로 유지한 상태로 macOS를 다시 설치할 수 있습니다.

  1. 인터넷에 연결되어 있는지 확인하십시오.

  2. 복구 앱에서 macOS Big Sur 다시 설치를 클릭한 다음, 계속을 클릭하십시오.

    중요: 디스크를 선택하는 패널에서 현재 macOS 디스크를 선택하십시오(대부분의 경우 사용 가능한 디스크는 하나뿐임).

  3. 화면상의 지침을 따르십시오.

macOS 지우고 다시 설치하기

macOS를 지우고 다시 설치하면 사용자 계정, 네트워크 설정, 사용자의 Mac에 있는 모든 파일 및 폴더가 삭제됩니다. macOS를 지우기 전에 시스템에 로그인한 다음, 보관하려는 모든 파일을 다른 저장 장치로 복사하여 백업하십시오. 또한, 네트워크 환경설정에 있는 네트워크 설정 정보를 메모해 두면 macOS를 다시 설치한 후에 네트워크에 더 쉽게 다시 연결할 수 있습니다.

  1. 인터넷에 연결되어 있는지 확인하십시오.

  2. 복구 앱에서 디스크 유틸리티를 클릭한 다음, 계속을 클릭하십시오.

  3. 디스크 유틸리티의 사이드바에서 지우려는 볼륨을 선택한 다음, 도구 막대에서 지우기 버튼을 클릭하십시오.

  4. 이름 필드에 볼륨의 이름을 입력하십시오.

  5. 포맷 팝업 메뉴를 클릭하고 APFS를 선택한 다음, 볼륨 그룹 지우기를 클릭하십시오.

  6. 지우기 작업이 완료되면 완료를 클릭한 다음, 디스크 유틸리티 > 디스크 유틸리티 종료를 선택하십시오.

  7. 복구 앱에서 macOS Big Sur 다시 설치를 선택하고 계속을 클릭한 다음, 화면상의 지침을 따르십시오.

복구 로그 보기

복구 로그는 macOS 복구에서 기록된 메시지를 포함합니다. 표시할 메시지 유형을 지정하고 메시지를 필터링하여 특정 텍스트를 검색하고 복구 로그를 파일에 저장할 수 있습니다.

  1. 복구 앱에서 윈도우 > 복구 로그를 선택하십시오.

  2. ‘자세한 레벨’ 팝업 메뉴를 클릭한 다음, 표시할 메시지를 선택하십시오.

    • 오류만 보기: 오류 메시지만 표시하려면 이 옵션을 선택하십시오.

    • 오류 및 진행 과정 보기: 오류 메시지와 진행 과정 메시지를 표시하려면 이 옵션을 선택하십시오.

    • 모든 로그 보기: 모든 메시지를 표시하려면 이 옵션을 선택하십시오.

  3. 메시지를 필터링하려면 검색할 내용을 필터 필드에 입력하십시오.

  4. 복구 로그를 저장하려면 저장 버튼을 클릭하고 파일 위치를 선택한 다음, 저장을 클릭하십시오.

  5. 복구 로그 보기를 마치면 윈도우의 왼쪽 상단 모서리에 있는 닫기 버튼을 클릭하십시오.

복구 진단 파일 생성하기

피드백 지원을 사용하여 문제 리포트를 제출하는 경우, 복구 진단 파일을 제공하도록 요청을 받을 수 있습니다. 파일을 생성하여 외장 저장 장치에 저장할 수 있습니다.

  1. Mac에 외장 저장 장치를 연결하십시오.

  2. Control-Option-Shift-Command-마침표를 누르십시오.

  3. 화면이 깜박인 다음 대화상자가 나타날 때까지 잠시 기다리십시오.

  4. 대화상자를 사용하여 복구 진단 파일을 외장 저장 장치의 위치에 저장하십시오.

macOS 복구 종료하기

Time Machine으로 시스템을 복원하거나 macOS를 다시 설치하는 경우, Mac은 자동으로 재시동됩니다. 다른 경우에 macOS 복구를 수동으로 종료해야 할 수 있습니다(예: macOS를 다시 설치하기 전에 파일을 백업하려는 경우 또는 시동 보안 유틸리티를 사용하여 설정을 방금 변경한 경우).

Mac에서 다음 중 하나를 수행하십시오.

  • Mac 재시동하기: Apple 메뉴 > 재시동을 선택하십시오.

  • Mac을 재시동하기 전에 다른 시동 디스크 선택하기: Apple 메뉴 > 시동 디스크를 선택하고 원하는 시동 디스크를 선택한 다음, 재시동을 클릭하십시오.

  • Mac 종료하기: Apple 메뉴 > 시스템 종료를 선택하십시오.
