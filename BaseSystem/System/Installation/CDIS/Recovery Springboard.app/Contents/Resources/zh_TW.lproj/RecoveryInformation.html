在配備 Apple 晶片的 Mac 上使用「macOS 復原」

「macOS 復原」為 Mac 的內建復原系統。

您可以使用在配備 Apple 晶片的 Mac 上「macOS 復原」中的 App 來修復內部磁碟、重新安裝 macOS、從「時光機」備份回復檔案、設定不同磁碟的安全性規則、在兩部 Mac 電腦之間傳送檔案、以安全模式啟動等等。

重新安裝 macOS 需要 Internet 連線。若要連接 Internet，您可以使用無線或有線網路連線。請參閱：使用 Wi-Fi 連接 Internet。

注意：若您嘗試連接網頁驗證入口（例如在咖啡廳、圖書館或飯店）或企業級網路，您可能無法在「macOS 復原」中連接 Internet。

在配備 Apple 晶片的 Mac 上可於「macOS 復原」中使用的 App

下列 App 可在配備 Apple 晶片的 Mac 上於「macOS 復原」中使用：

  • 復原：「復原」App 可取用「macOS 復原」中的其他 App。請參閱：在「macOS 復原」中啟動電腦、在「macOS 復原」中啟動電腦和配對藍牙輸入裝置，以及結束「macOS 復原」。

  • 時光機系統回復：從「時光機」備份回復您的資料。在「復原」App 中，選取「從時光機回復」，然後按一下「繼續」。若要返回「復原」App，請選擇「時光機系統回復」>「結束時光機系統回復」。請參閱：使用「時光機」來回復系統。

  • 安裝 macOS：在電腦上重新安裝 macOS。在「復原」App 中，選取「重新安裝 macOS」，然後按一下「繼續」。若要返回「復原」App，請選擇「安裝 macOS Big Sur」>「結束安裝 macOS」。請參閱：重新安裝 macOS。

  • Safari：在「復原」App 中，選取 Safari，然後按一下「繼續」。若要返回「復原」App，請選擇 Safari >「結束 Safari」。若您已連接 Internet，便可使用 Safari 在 Apple 支援網站上搜尋更多資訊。請注意 Safari 瀏覽器外掛模組和延伸功能會在「macOS 復原」中停用。

  • 磁碟工具程式：修復或清除磁碟。在「復原」App 中，選取「磁碟工具程式」，然後按一下「繼續」。若要返回「復原」App，請選擇「磁碟工具程式」>「結束磁碟工具程式」。請參閱：清除並重新安裝 macOS 和 修復內部儲存裝置。

  • 開機安全性工具程式：設定 Mac 的安全性規則。在「復原」App 中，選擇「工具程式」>「開機安全性工具程式」。若要返回「復原」App，請選擇「開機安全性工具程式」>「結束開機安全性工具程式」。請參閱：設定安全性規則。

  • 終端機：透過命令列更改設定。在「復原」App 中，選擇「工具程式」>「終端機」。若要返回「復原」App，請選擇「終端機」>「結束終端機」。

  • 共享磁碟：共享在「macOS 復原」中啟動的 Mac 磁碟。在「復原」App 中，選擇「工具程式」>「共享磁碟」。請參閱：使用「共享磁碟」來在兩部 Mac 電腦之間傳送檔案。

  • 啟動磁碟：設定 Mac 的啟動磁碟。選擇「蘋果」選單 >「啟動磁碟」。若要結束 App，請選擇「啟動磁碟」>「結束啟動磁碟」。

您也可以使用「macOS 復原」來執行以下作業：

  • 以安全模式啟動電腦

  • 設定預設啟動卷宗並重新啟動電腦

  • 檢視復原記錄

  • 製作復原診斷檔案

在「macOS 復原」中啟動電腦

  1. 選擇「蘋果」選單  >「關機⋯」。

  2. 按住 Mac 上的電源按鈕直到您看見「正在載入啟動選項」。

  3. 按一下「選項」，然後按一下「繼續」。

  4. 若有要求，請輸入管理者帳號的密碼。

在「macOS 復原」中啟動電腦和配對藍牙輸入裝置

若您在「macOS 復原」中啟動時需將藍牙鍵盤、觸控式軌跡版或滑鼠與 Mac 配對，請依照下列指示操作。

  1. 選擇「蘋果」選單  >「關機⋯」。

  2. 按住 Mac 上的電源按鈕直到您看見「正在載入啟動選項」。

  3. 當系統卷宗和「選項」按鈕顯示，請按下電源按鈕三次。

    注意：每次按下電源按鈕的間隔與前一次需在 1 秒以內。若您電源按鈕按得太快也不算數，因為快速按下另有用途，例如「旁白」。

  4. 使用「藍牙設定輔助程式」來將藍牙配件與 Mac 配對。

    注意：若您已經有已連線的輸入裝置，便不會看見「藍牙設定輔助程式」。

  5. 輸入管理者帳號的密碼。

以安全模式啟動電腦

  1. 選擇「蘋果」選單  >「關機⋯」。

  2. 按住 Mac 上的電源按鈕直到您看見「正在載入啟動選項」。

  3. 選取卷宗。

  4. 按住 Shift 鍵，然後按一下安全模式中的「繼續」。

    電腦會自動重新啟動。

設定預設啟動卷宗並重新啟動電腦

  1. 選擇「蘋果」選單  >「關機⋯」。

  2. 按住 Mac 上的電源按鈕直到您看見「正在載入啟動選項」。

  3. 選取卷宗。

  4. 按住 Option 鍵，然後按一下「總是使用」。

    電腦會自動重新啟動。

使用 Wi-Fi 連接 Internet

若 Wi-Fi 選單圖像（在螢幕右上角）包含一格或多格訊號格數 ，表示 Wi-Fi 已開啟並連接無線網路。若 Wi-Fi 選單圖像未包含訊號格數 ，表示 Wi-Fi 已開啟但未連接無線網路。若 Wi-Fi 選單圖像有斜線穿過 ，表示 Wi-Fi 已關閉。需先開啟 Wi-Fi 才能加入無線網路。

在 Mac 上，執行以下任一操作：

  • 開啟 Wi-Fi：按一下 Wi-Fi 選單圖像，然後選擇「開啟 Wi-Fi」。

  • 關閉 Wi-Fi：按一下 Wi-Fi 選單圖像，然後選擇「關閉 Wi-Fi」。

  • 加入網路：按一下 Wi-Fi 選單圖像，然後選擇一個網路。如有要求，請輸入網路的密碼。

  • 加入封閉網路：按一下 Wi-Fi 選單圖像，然後選擇「加入其他網路」。輸入網路的名稱和網路的密碼（如有要求）。

修復內部儲存裝置

若您的電腦發生問題或啟動電腦後看見「macOS 復原」視窗，您可能需要修復電腦的內部儲存裝置（例如固態硬碟（SSD））。

  1. 在「復原」App 中，選取「工具程式」，然後按一下「繼續」。

  2. 在「磁碟工具程式」App 中，選擇「顯示方式」>「顯示所有裝置」。

  3. 在側邊欄中，選取卷宗，然後按一下「修理工具」按鈕 。

  4. 在「修理工具」對話框中，按一下「執行」，然後依照螢幕上的指示操作。

  5. 當「修理工具」程序完成，請按一下「完成」。

  6. 對儲存裝置上的每個卷宗重複步驟 3 到 5 。

  7. 在側邊欄中，選取容器，然後按一下「修理工具」按鈕 。

  8. 在「修理工具」對話框中，按一下「執行」，然後依照螢幕上的指示操作。

  9. 當「修理工具」程序完成，請按一下「完成」。

  10. 對儲存裝置上的每個容器重複步驟 7 到 9。

  11. 在側邊欄中，選取儲存裝置，然後按一下「修理工具」按鈕 。

  12. 在「修理工具」對話框中，按一下「執行」，然後依照螢幕上的指示操作。

  13. 當「修理工具」程序完成，請按一下「完成」。

若「磁碟工具程式」無法修復磁碟，請盡可能備份您的資訊，然後依照清除並重新安裝 macOS 中的指示操作。

若要返回「復原」App，請選擇「磁碟工具程式」>「結束磁碟工具程式」。

使用「時光機」來回復系統

若您先前已製作「時光機」備份，即可使用「從時光機備份回復」來將電腦上的所有資料回復到先前的狀態。

重要事項：請僅使用「時光機」備份將資料回復到原本備份的來源電腦上。若您要從這部 Mac 傳送資訊到新的電腦上，請結束「macOS 復原」，重新啟動 Mac，然後使用 macOS 中的「系統移轉輔助程式」App。

  1. 如果備份位於「時光囊」，請確定已連接 Internet。

  2. 在「復原」App 中，請選取「從時光機備份回復」，然後按一下「繼續」。

  3. 在「時光機系統回復」App 中，依照螢幕上的指示操作。

重新安裝 macOS

在部分情況下，您可能需要重新安裝 macOS。重新安裝 macOS 時，您仍可以保留原有的檔案和使用者設定。

  1. 請確定已連接 Internet。

  2. 在「復原」App 中，按一下「重新安裝 macOS Big Sur」，然後按一下「繼續」。

    重要事項：請在選取磁碟的面板中選擇現用的 macOS 磁碟（在大部分情況下，這是唯一可選的磁碟）。

  3. 依照螢幕上的指示操作。

清除並重新安裝 macOS

若您清除並重新安裝 macOS，您 Mac 上的使用者帳號、網路設定、所有檔案和檔案夾都將刪除。清除 macOS 前請嘗試重新登入您的系統，然後將想要保留的檔案拷貝到其他儲存裝置來進行備份。也請記下「網路」偏好設定中的網路設定，以便重新安裝 macOS 後可以再次輕鬆連線。

  1. 確定已連接 Internet。

  2. 在「復原」App 中，按一下「磁碟工具程式」，然後按一下「繼續」。

  3. 在「磁碟工具程式」中，在側邊欄中選取您要清除的卷宗，然後按一下工具列中的「清除」按鈕。

  4. 在「名稱」欄位中輸入卷宗的名稱。

  5. 按一下「格式」彈出式選單，選擇 APFS，然後按一下「清除卷宗群組」。

  6. 當清除程序完成，請按一下「完成」，然後選擇「磁碟工具程式」>「結束磁碟工具程式」。

  7. 在「復原」App 中，按一下「重新安裝 macOS Big Sur」，按一下「繼續」，然後依照螢幕上的指示操作。

設定安全性規則

  1. 在「復原」App 中，選擇「工具程式」>「開機安全性工具程式」。

  2. 選取您要用來設定安全性規則的系統。

  3. 若磁碟已使用「檔案保險箱」加密，請按一下「解鎖」，輸入密碼，然後按一下「解鎖」。

  4. 按一下「安全性規則」。

  5. 檢閱以下安全性選項：

    • 完整安全性：確保只會執行目前的作業系統，或是 Apple 目前信任的已簽署作業系統軟體。此模式在軟體安裝期間需要網路連線。

    • 較低安全性：允許執行任何 Apple 所信任的已簽署作業系統軟體版本。

  6. 如有需要，請選取「較低安全性」，輸入您的管理者使用者名稱和密碼，然後執行下列其中一項操作：

    • 選取「允許來自已識別開發者的核心延伸功能使用者管理」註記框以允許安裝會使用舊式核心延伸功能的軟體。

    • 選取「允許核心延伸功能遠端管理和自動軟體更新」註記框來使用行動裝置管理（MDM）授權舊式核心延伸功能和軟體更新項目的遠端管理。

  7. 按一下「好」。

  8. 重新啟動 Mac 來讓變更生效。

使用「共享磁碟」來在兩部 Mac 電腦之間傳送檔案

您可以共享卷宗和使用 USB、USB-C 或 Thunderbolt 連接線來連接兩部電腦，藉此在開機進入「macOS 復原」的 Mac 和另一部 Mac 之間傳送檔案。

  1. 使用 USB、USB-C 或 Thunderbolt 連接線來連接兩部電腦。

  2. 在「復原」App 中，選擇「工具程式」>「共享磁碟」。

  3. 選取您要共享的卷宗，然後按一下「開始共享」。

  4. 在另一部 Mac 上，打開 Finder 視窗，然後在側邊欄中的「位置」下方按一下「網路」。

  5. 在「網路」視窗中，按兩下包含共享磁碟的 Mac，按一下「連接身分」，選取「連接身分」視窗中的「訪客」，然後按一下「連接」。

  6. 傳送檔案。

  7. 傳送檔案完成時，請退出另一部 Mac 上的卷宗。

檢視復原記錄

復原記錄包含「macOS 復原」記錄的訊息。您可以指定要顯示的訊息類型，過濾訊息來搜尋特定文字，以及將復原記錄儲存到檔案。

  1. 在「復原」App 中，選擇「視窗」>「復原記錄」。

  2. 按一下「詳細程度」彈出式選單，然後選擇要顯示的訊息：

    • 只顯示錯誤：選擇此選項來只顯示錯誤訊息。

    • 顯示錯誤與過程：選擇此選項來顯示錯誤訊息和過程訊息。

    • 顯示全部記錄：選擇此選項來顯示所有訊息。

  3. 若要過濾訊息，請在「過濾條件」欄位中輸入您要搜尋的文字。

  4. 若要儲存復原記錄，請按一下「儲存」按鈕，選擇檔案的位置，然後按一下「儲存」。

  5. 當您完成檢視復原記錄，請按一下視窗左上角的「關閉」按鈕 。

製作復原診斷檔案

若您透過「使用者意見程式」提交問題報告，系統可能會要求您提供復原診斷檔案。您可以製作檔案並儲存到外接儲存裝置。

  1. 將外接儲存裝置連接到 Mac。

  2. 按下 Control + Option + Shift + Command + 句號 (.)。

  3. 在螢幕閃爍後，請稍候片刻直到對話框顯示。

  4. 使用對話框來將復原診斷檔案儲存到外接儲存裝置上的位置。

結束「macOS 復原」

從「時光機」回復系統或重新安裝 macOS 時，您的 Mac 會自動重新開機。在其他情況下，您需要手動結束「macOS 復原」（例如，若您要在重新安裝 macOS 前備份檔案，或是若您僅使用「開機安全性工具程式」更改了設定）。

在 Mac 上，執行下列其中一項操作：

  • 重新啟動 Mac：選擇「蘋果」選單  >「重新開機⋯」。

  • 在重新啟動 Mac 前選擇其他啟動磁碟：選擇「蘋果」選單  >「啟動磁碟」，選取一個啟動磁碟，然後按一下「重新開機」。

  • 將 Mac 關機：選擇「蘋果」選單  >「關機⋯」。
