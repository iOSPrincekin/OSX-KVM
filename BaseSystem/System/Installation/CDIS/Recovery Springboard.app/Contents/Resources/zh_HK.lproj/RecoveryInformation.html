在配備 Apple 晶片的 Mac 上使用 macOS 還原

「macOS 還原」是 Mac 的內置還原系統。

你可以在配備 Apple 晶片的 Mac 上使用「macOS 還原」中的 App 來修復內置磁碟、重新安裝 macOS、從「時光機」備份還原檔案、為不同磁碟設定保安政策、在兩部 Mac 電腦之間傳送檔案，以及在安全模式開機等等。

重新安裝 macOS 需要互聯網連線。如要連接互聯網，你可以使用無線或有線網絡連線。請參閱：使用 Wi-Fi 連接到互聯網。

注意：如你正在嘗試連接到限制入口網站（例如咖啡店、圖書館和酒店）或企業網絡，你可能無法在「macOS 還原」中取用互聯網。

在配備 Apple 晶片的 Mac 上「macOS 還原」中可用的 App

以下的 App 可在配備 Apple 晶片的 Mac 上的「macOS 還原」中使用：

  • 還原：「還原」App 在「macOS 還原」中提供其他 App 的取用權。請參閱：在「macOS 還原」中開啟電腦、在「macOS 還原」中開啟電腦並配對藍牙輸入裝置和結束「macOS 還原」。

  • 時光機系統還原：從「時光機」備份還原資料。在「還原」App 視窗中，選擇「從時光機還原」，然後按一下「繼續」。如要返回「還原」App， 選擇「時光機系統還原」>「結束時光機系統還原」。請參閱：使用「時光機」來還原你的系統。

  • 安裝 macOS：在你的電腦上重新安裝 macOS。在「還原」App 視窗中，選擇「重新安裝 macOS」，然後按一下「繼續」。如要返回「還原」App， 選擇「安裝 macOS Big Sur」>「結束安裝 macOS」。請參閱：重新安裝 macOS。

  • Safari：在「還原」App 中，選擇 Safari，然後按一下「繼續」。 如要返回「還原」App，選擇 Safari >「結束 Safari」。如你已連接到互聯網，你可以使用 Safari 來在 Apple 支援網站上搜尋更多資料。注意，Safari 瀏覽器外掛模組和延伸功能在「macOS 還原」中停用。

  • 磁碟工具程式：修復或清除磁碟。在「還原」App 中，選擇「磁碟工具程式」，然後按一下「繼續」。如要返回「還原」App， 選擇「磁碟工具程式」>「結束磁碟工具程式」。請參閱：清除並重新安裝 macOS 和修復你的內置儲存裝置。

  • 開機保安工具程式：為你的 Mac 設定保安政策。在「還原」App 中，選擇「工具程式」>「開機保安工具程式」。如要返回「還原」App， 選擇「開機保安工具程式」>「結束開機保安工具程式」。請參閱：設定保安政策。

  • 終端機：透過命令列變更設定。在「還原」App 中，選擇「工具程式」>「終端機」。如要返回「還原」App， 選擇「終端機」>「結束終端機」。

  • 分享磁碟：分享在「macOS 還原」中開機的 Mac 磁碟。在「還原」App 中，選擇「工具程式」>「分享磁碟」。請參閱：使用「分享磁碟」來在兩部 Mac 電腦之間傳送檔案。

  • 開機磁碟：為 Mac 設定開機磁碟。選擇「蘋果」選單 >「開機磁碟」。如要結束 App，選擇「開機磁碟」>「結束開機磁碟」。

你也可以使用「macOS 還原」來執行以下操作：

  • 在安全模式開機

  • 設定預設開機卷宗並重新啟動電腦

  • 檢視還原記錄

  • 製作還原診斷檔案

在「macOS 還原」中開啟電腦

  1. 選擇「蘋果」選單 >「關機」。

  2. 在 Mac 上按住電源按鈕，直到看到「正在載入開機選項」。

  3. 按一下「選項」，然後按一下「繼續」。

  4. 輸入管理員帳户密碼。

在「macOS 還原」中開啟電腦並配對藍牙輸入裝置

如你需要在「macOS 還原」開機時將藍牙鍵盤、觸控板或滑鼠與 Mac 配對，請遵循以下指示執行。

  1. 選擇「蘋果」選單 >「關機」。

  2. 在 Mac 上按住電源按鈕，直至看到「正在載入開機選項」。

  3. 在系統卷宗和「選項」按鈕出現時，按下電源按鈕三次。

    注意：按下電源按鈕的每一下，需要在上一下後的一秒內完成。如你太快按下電源按鈕，將不會被記錄，因為快速按下已保留作其他用途，例如「旁白」。

  4. 使用「藍牙設定輔助程式」來將藍牙配件與 Mac 配對

    注意：如你已連接輸入裝置，就不會看到「藍牙設定輔助程式」

  5. 輸入管理員帳户密碼。

在安全模式開機

  1. 選擇「蘋果」選單 >「關機」。

  2. 在 Mac 上按住電源按鈕，直至看到「正在載入開機選項」。

  3. 選擇一個卷宗。

  4. 按住 Shift 鍵，然後在安全模式中按一下「繼續」。

    電腦會自動重新啟動。

設定預設開機卷宗並重新啟動電腦

  1. 選擇「蘋果」選單 >「關機」。

  2. 按住 Mac 上的電源按鈕，直至看到「正在載入開機選項」。

  3. 選擇一個卷宗。

  4. 按住 Option 鍵，然後按一下「總是使用」。

    電腦會自動重新啟動。

使用 Wi-Fi 連接到互聯網

如螢幕右上角的 Wi-Fi 選單圖像包含一或多個訊號格 ，，即 Wi-Fi 已開啟並已連接到無線網絡。如 Wi-Fi 選單圖像並不包含任何訊號格 ，即 Wi-Fi 已開啟但未連接到無線網絡。如有斜線穿過 Wi-Fi 選單圖像上 ，即 Wi-Fi 已關閉。需要開啟 Wi-Fi 才能加入無線網絡。

在你的 Mac 上，執行以下任何操作：

  • 開啟 Wi-Fi：按一下 Wi-Fi 選單圖像，然後選擇「開啟 Wi-Fi」。

  • 關閉 Wi-Fi：按一下 Wi-Fi 選單圖像，然後選擇「關閉 Wi-Fi」。

  • 加入網絡：按一下 Wi-Fi 選單圖像，然後選擇網絡。如系統要求，請輸入網絡的密碼。

  • 加入封閉式網絡：按一下 Wi-Fi 選單圖像，然後選擇「加入其他網絡」。輸入網絡的名稱，並在系統要求時（如有），輸入網絡密碼。

修復你的內置儲存裝置

如你的電腦發生問題或如你開機後看到「macOS 還原」視窗，你可能需要修復電腦的內置儲存裝置（例如硬碟（SSD））。

  1. 在「還原」App 中，選擇「磁碟工具程式」，然後按一下「繼續」。

  2. 在「磁碟工具程式」App 中，選擇「顯示方式」>「顯示所有裝置」。

  3. 在側邊欄中選擇一個卷宗，然後按一下「修理工具」按鈕 。

  4. 在「修理工具」對話框中，按一下「執行」，然後按照螢幕上的指示操作。

  5. 在「修理工具」程序完成後，按一下「完成」。

  6. 為儲存裝置上的每個卷宗重複步驟 3 至 5。

  7. 在側邊欄中選擇一個容器，然後按一下「修理工具」按鈕 。

  8. 在「修理工具」對話框中，按一下「執行」，然後按照螢幕上的指示操作。

  9. 在「修理工具」程序完成後，按一下「完成」。

  10. 為儲存裝置上的每個容器重複步驟 7 至 9。

  11. 在側邊欄中選擇一個儲存裝置，然後按一下「修理工具」按鈕 。

  12. 在「修理工具」對話框中，按一下「執行」，然後按照螢幕上的指示操作。

  13. 在「修理工具」程序完成後，按一下「完成」。

如「磁碟工具程式」」無法修復你的磁碟，請盡量備份你的資料，然後按照清除並重新安裝 macOS 中的指示操作。

如要返回「還原」App， 選擇「磁碟工具程式」>「結束磁碟工具程式」。

使用「時光機」來還原你的系統

如你之前已製作「時光機」備份，可使用「從時光機備份還原」以將電腦上的所有資料還原為之前的狀態。

重要事項：只能使用「時光機」備份將資料還原至備份的來源電腦上。如你要從這部 Mac 傳送資料到新的電腦上，請結束「macOS 還原」，重新啟動 Mac，然後使用 macOS 的「系統轉移輔助程式」App。

  1. 如果備份位於「時間囊」，請確定已連接互聯網。

  2. 在「還原」App 中，選擇「從時光機備份還原」，然後按一下「繼續」。

  3. 在「時光機系統還原」App 中，遵循螢幕上的指示操作。

重新安裝 macOS

在部份情況下，你可能需要重新安裝 macOS。你可以重新安裝 macOS 同時完整保留你的檔案和用户設定。

  1. 確定已連接互聯網。

  2. 在「還原」App 中，按一下「重新安裝 macOS Big Sur」，然後按一下「繼續」。

    重要事項：在你選擇磁碟的面板上，選擇你目前的 macOS 磁碟（在大部份情況下，這是唯一可用的選項）。

  3. 按照螢幕上的指示操作。

清除並重新安裝 macOS

如你清除並重新安裝 macOS，Mac 上的資料（你的用户帳户、網絡設定、所有檔案和資料夾）都將刪除。清除 macOS 前，嘗試登入返回你的系統，將要保留的檔案複製到其他儲存裝置來備份。此外，請記下「網絡」偏好設定中的網絡設定，以便重新安裝 macOS 後可以再次輕鬆連線。

  1. 確定已連接互聯網。

  2. 在「還原」App 中，選擇「磁碟工具程式」，然後按一下「繼續」。

  3. 在「磁碟工具程式」中，於側邊欄選擇你要清除的卷宗，然後按一下工具列上的「清除」按鈕。

  4. 在「名稱」欄位為卷宗輸入名稱。

  5. 按一下「格式」彈出式選單，選擇 APFS，然後按一下「清除卷宗群組」。

  6. 在清除程序完成後，按一下「完成」，然後選擇「磁碟工具程式」>「結束磁碟工具程式」。

  7. 在「還原」App 中，按一下「重新安裝 macOS Big Sur」，按一下「繼續」，然後按照螢幕上的指示操作。

設定保安政策

  1. 在「還原」App 中，選擇「工具程式」>「開機保安工具程式」。

  2. 選擇你要用於設定保安政策的系統。

  3. 如磁碟已使用「檔案保險箱」加密，按一下「解鎖」，輸入密碼，然後按一下「解鎖」。

  4. 按一下「保安政策」。

  5. 檢視以下保安選項：

    • 完整保安：確保只有你目前的作業系統或 Apple 目前信任的已簽署作業系統軟件可以執行。此模式需要在軟件安裝時連接網絡。

    • 較低保安：允許執行 Apple 曾經信任的任何版本的已簽署作業系統軟件。

  6. 如有必要，選擇「寬鬆保安」，輸入你的管理員用户名稱和密碼，然後執行以下一項操作：

    • 選擇「允許用户管理來自已識別開發者的核心延伸功能」剔選框以允許安裝使用舊版核心延伸功能的軟件。

    • 選擇「允許遙距管理核心延伸功能和自動軟件更新」剔選框來授權使用流動裝置管理（MDM）解決方案來遙距管理舊版核心延伸功能和軟件更新。

  7. 按一下「好」。

  8. 重新啟動 Mac 以讓變更生效。

使用「分享磁碟」在兩部 Mac 電腦之間傳送檔案

通過分享卷宗並使用 USB，USB-C 或 Thunderbolt 連接線連接兩部電腦，你可以在啟動到「macOS 還原」的 Mac 和另一部 Mac 之間傳送檔案。

  1. 使用 USB，USB-C 或 Thunderbolt 連接線連接兩部電腦。

  2. 在「還原」App 中，選擇「磁碟工具程式」>「分享磁碟」。

  3. 選擇你要分享的卷宗，然後按一下「開始分享」。

  4. 在另一部 Mac 上，開啟 Finder 視窗，然後在側邊欄中的「位置」下按一下「網絡」。

  5. 在「網絡」視窗中，按兩下已分享磁碟的 Mac，按一下「連接身份」，在「連接身份」視窗中選擇「訪客」，然後按一下「連線」。

  6. 傳送檔案。

  7. 當你完成傳送檔案後，退出另一部 Mac 上的卷宗。

檢視還原記錄

還原記錄包含「macOS 還原」記錄的訊息。你可以指定要顯示的訊息類型、過濾訊息以搜尋指定文字，以及將還原記錄儲存到檔案中。

  1. 在「還原」App 中，選擇「視窗」>「還原記錄」。

  2. 按一下「詳細層級」彈出式選單，然後選擇要顯示的訊息：

    • 只顯示錯誤：選擇此選項以只顯示錯誤訊息。

    • 顯示錯誤與過程：選擇此選項以顯示錯誤訊息和過程訊息。

    • 顯示全部記錄：選擇此選項以顯示全部訊息。

  3. 如要過濾訊息，請在「過濾」欄位輸入你要搜尋的文字。

  4. 如要儲存還原記錄，按一下「儲存」按鈕，選擇檔案位置，然後按一下「儲存」。

  5. 在你完成檢視還原記錄後，按一下視窗左上角的「關閉」按鈕 。

製作還原診斷檔案

如你使用「意見輔助程式」提交問題報告，你可能需要提供還原診斷檔案。你可以製作檔案並將檔案儲存在外置儲存裝置。

  1. 將外置儲存裝置連接至你的 Mac。

  2. 按下 Control＋Option＋Shift＋Command＋句號鍵。

  3. 在螢幕閃爍後，稍等直至對話框出現。

  4. 使用對話框將還原診斷檔案儲存至外置儲存裝置的位置中。

結束「macOS 還原」

從「時光機」還原系統或重新安裝 macOS 時，你的 Mac 會自動重新啟動。在其他情況下，你需要手動結束「macOS 還原」（例如，如果要在重新安裝 macOS 之前備份檔案，或只是使用「開機保安工具程式」更改了設定）。

在 Mac 上，執行以下其中一項操作：

  • 重新啟動 Mac：選擇「蘋果」選單 >「重新開機」。

  • 重新啟動 Mac 前，選擇一個不同的開機磁碟：選擇「蘋果」選單 >「開機磁碟」，選擇開機磁碟，然後按一下「重新開機」。

  • 關閉 Mac：選擇「蘋果」選單 > 「關機」。
