Восстановление macOS на Mac с процессором Intel

Восстановление macOS — это встроенная система, которая переводит компьютер Mac в режим восстановления.

Приложения, доступные в режиме восстановления macOS, можно использовать, чтобы исправить внутренний диск, переустановить macOS, восстановить файлы из резервной копии Time Machine, настроить параметры безопасности и выполнить другие действия на компьютере Mac с процессором Intel.

Для переустановки macOS требуется подключение к интернету. Можно использовать как проводное подключение к интернету, так и беспроводное. См. раздел Подключение к интернету по Wi-Fi.

Примечание. При попытке подключения к порталу, требующему авторизации (например, в кофейне, библиотеке или отеле), или к корпоративной сети, возможно, не удастся получить доступ к интернету в режиме восстановления macOS.

Приложения, доступные в режиме восстановления macOS на Mac с процессором Intel

В режиме восстановления macOS на Mac с процессором Intel доступны следующие приложения.

  • Восстановление. Приложение «Восстановление» открывается при загрузке компьютера в режиме восстановления macOS. См. разделы Загрузка компьютера Mac с процессором Intel в режиме восстановления macOS и Завершение сеанса в режиме восстановления macOS.

  • Восстановление системы из Time Machine. Вы можете восстановить данные из резервной копии Time Machine. В приложении «Восстановление» выберите восстановление из резервной копии Time Machine, затем нажмите «Продолжить». Для возврата в приложение «Восстановление» выберите «Восстановление системы из Time Machine» > «Завершить Восстановление системы из Time Machine». См. раздел Восстановление системы с помощью Time Machine.

  • Установка macOS. Вы можете переустановить macOS на своем компьютере. В приложении «Восстановление» выберите «Переустановить macOS», затем нажмите «Продолжить». Для возврата в приложение «Восстановление» выберите «Установка macOS Big Sur» > «Завершить Установку macOS». См. раздел Переустановка macOS.

  • Safari. В приложении «Восстановление» выберите Safari, затем нажмите «Продолжить». Для возврата в приложение «Восстановление» выберите «Safari» > «Завершить Safari». При наличии подключения к интернету можно использовать Safari для поиска дополнительной информации на веб-сайте службы поддержки Apple. В режиме восстановления macOS плагины и расширения браузера Safari выключены.

  • Дисковая утилита. Вы можете исправить или стереть диск. В приложении «Восстановление» выберите «Дисковая утилита», затем нажмите «Продолжить». Для возврата в приложение «Восстановление» выберите «Дисковая утилита» > «Завершить Дисковую утилиту». См. раздел Стирание и переустановка macOS.

  • Утилита безопасной загрузки. Вы можете настроить политики безопасности для компьютера Mac. В приложении «Восстановление» выберите «Утилиты» > «Утилита безопасной загрузки». Для возврата в приложение «Восстановление» выберите «Утилита безопасной загрузки» > «Завершить Утилиту безопасной загрузки». См. разделы Настройка пароля прошивки с помощью Утилиты безопасной загрузки и Настройка параметров безопасной загрузки с помощью Утилиты безопасной загрузки.

    Параметры, доступные в Утилите безопасной загрузки, зависят от модели Mac. См. статьи службы поддержки Apple Утилита безопасной загрузки и Модели компьютеров Mac с процессором безопасности Apple T2 Security Chip.

  • Загрузочный диск. Вы можете настроить загрузочный диск для компьютера Mac. Выберите меню Apple  > «Загрузочный диск». Для возврата в приложение «Восстановление» выберите «Загрузочный диск» > «Завершить Загрузочный диск».

  • Терминал. В приложении «Восстановление» выберите «Утилиты» > «Терминал». Для возврата в приложение «Восстановление» выберите «Терминал» > «Завершить Терминал».

Режим восстановления macOS также можно использовать для следующих задач:

  • Просмотр журнала восстановления

  • Создание файла диагностики восстановления

Загрузка компьютера Mac с процессором Intel в режиме восстановления macOS

  1. Перезагрузите Mac.

  2. Быстро нажмите и удерживайте одну из следующих комбинаций клавиш, пока не отобразится экран загрузки.

    • Command-R. Загрузка будет выполнена из встроенной системы восстановления macOS. Используйте эту комбинацию клавиш, чтобы переустановить новейшую версию macOS, которая была установлена в Вашей системе, или использовать другие приложения, доступные в режиме восстановления macOS.

    • Option-Command-R. Загрузка будет выполнена из системы восстановления macOS, загружаемой из интернета. Используйте эту комбинацию клавиш, чтобы переустановить macOS и перейти на новейшую версию macOS, которая совместима с Вашим компьютером Mac.

    • Option-Shift-Command-R. Загрузка будет выполнена из системы восстановления macOS, загружаемой из интернета. Используйте эту комбинацию клавиш, чтобы переустановить версию macOS, которая была предустановлена на Вашем компьютере Mac, или наиболее близкую к ней версию, которая доступна на данный момент.

  3. Если отображается замок, введите пароль для входа в систему на компьютере Mac.

  4. Если на диске есть несколько томов, выберите том, который хотите восстановить, затем нажмите «Далее».

  5. При появлении запроса выберите учетную запись с правами администратора, нажмите «Далее», введите пароль для выбранной учетной записи, затем нажмите «Продолжить».

Подключение к интернету по Wi-Fi

Если значок меню Wi-Fi (в правом верхнем углу экрана) содержит одну или несколько полос , функция Wi-Fi включена, а компьютер подключен к беспроводной сети. Если значок меню Wi-Fi не содержит полос , функция Wi-Fi включена, но компьютер не подключен к беспроводной сети. Если значок меню Wi-Fi перечеркнут , функция Wi-Fi выключена. Перед подключением к беспроводной сети необходимо включить Wi-Fi.

На Mac выполните любое из следующих действий.

  • Включение Wi-Fi. Нажмите на значок меню Wi-Fi, затем выберите «Включить Wi-Fi».

  • Выключение Wi-Fi. Нажмите на значок меню Wi-Fi, затем выберите «Выключить Wi-Fi».

  • Подключение к сети. Нажмите на значок меню Wi-Fi, затем выберите сеть. При необходимости введите пароль сети.

  • Подключение к закрытой сети. Нажмите значок меню Wi-Fi, затем выберите «Подключиться к другой сети». Введите имя сети и, если необходимо, ее пароль.

Исправление внутреннего устройства хранения

Если с компьютером возникли проблемы или если при его загрузке отобразилось окно «Восстановление macOS», возможно, необходимо исправить внутреннее устройство хранения компьютера (такое как твердотельный накопитель).

  1. В приложении «Восстановление» выберите «Дисковая утилита», затем нажмите «Продолжить».

  2. В приложении «Дисковая утилита» выберите «Вид» > «Показывать все устройства».

  3. В боковой панели выберите том, затем нажмите кнопку «Первая помощь» .

  4. В диалоговом окне «Первая помощь» нажмите «Запустить», затем следуйте инструкциям на экране.

  5. Когда процесс «Первая помощь» завершится, нажмите «Готово».

  6. Повторите шаги с 3-го по 5-й для каждого тома на устройстве хранения.

  7. В боковой панели выберите контейнер, затем нажмите кнопку «Первая помощь» .

  8. В диалоговом окне «Первая помощь» нажмите «Запустить», затем следуйте инструкциям на экране.

  9. Когда процесс «Первая помощь» завершится, нажмите «Готово».

  10. Повторите шаги с 7-го по 9-й для каждого контейнера на устройстве хранения.

  11. В боковой панели выберите устройство хранения, затем нажмите кнопку «Первая помощь» .

  12. В диалоговом окне «Первая помощь» нажмите «Запустить», затем следуйте инструкциям на экране.

  13. Когда процесс «Первая помощь» завершится, нажмите «Готово».

Если диск не удается исправить с помощью Дисковой утилиты, по возможности создайте резервные копии всей своей информации, затем следуйте инструкциям в разделе Стирание и переустановка macOS.

Для возврата в приложение «Восстановление» для macOS выберите «Дисковая утилита» > «Завершить Дисковую утилиту».

Восстановление системы с помощью Time Machine

Если ранее была создана резервная копия Time Machine, можно использовать восстановление из резервной копии Time Machine для восстановления всей информации на компьютере до предыдущего состояния.

Важно! Используйте резервную копию Time Machine только для восстановления информации на компьютере, с которого была создана эта резервная копия. Если Вы хотите перенести информацию с этого Mac на новый компьютер, завершите сеанс в режиме восстановления macOS, перезагрузите Mac и используйте приложение «Ассистент миграции» для macOS.

  1. Если резервная копия находится на станции Time Capsule, убедитесь, что компьютер подключен к интернету.

  2. В приложении «Восстановление» выберите восстановление из резервной копии Time Machine, затем нажмите «Продолжить».

  3. Следуйте инструкциям на экране в приложении «Восстановление системы из Time Machine».

Настройка пароля прошивки с помощью Утилиты безопасной загрузки

Если настроен пароль прошивки, то становится невозможно выполнить загрузку компьютера Mac с любого внутреннего или внешнего устройства хранения, кроме выбранного Вами загрузочного диска, если пароль прошивки не введен или введен неправильно. Используя Утилиту безопасной загрузки, Вы можете включить, выключить или изменить пароль прошивки.

Когда пароль прошивки включен, необходимо вводить этот пароль каждый раз при загрузке компьютера в режиме восстановления macOS или при загрузке с любого устройства хранения, кроме выбранного Вами в настройках Загрузочного диска.

Изменения пароля прошивки вступают в силу после перезагрузки Mac.

  1. В приложении «Восстановление» выберите «Утилиты» > «Утилита безопасной загрузки».

  2. В случае появления запроса введите имя пользователя с правами администратора и его пароль.

  3. Выполните любое из следующих действий.

    • Включение пароля прошивки. Нажмите «Включить пароль прошивки», введите новый пароль, повторно введите его для подтверждения, а затем нажмите «Задать пароль».

    • Выключение пароля прошивки. Нажмите «Выключить пароль прошивки», введите пароль, затем нажмите «Выключить пароль».

    • Изменение пароля прошивки. Нажмите «Изменить пароль», введите старый пароль, введите новый пароль и подтвердите его, затем нажмите «Изменить пароль».

  4. Нажмите «Завершить Утилиту безопасной загрузки».

Настройка параметров безопасной загрузки с помощью Утилиты безопасной загрузки

Параметры безопасной загрузки доступны только на компьютерах Mac с чипом безопасности Apple T2.

Изменение выбранного параметра безопасной загрузки вступает в силу после перезагрузки Mac.

  1. В приложении «Восстановление» выберите «Утилиты» > «Утилита безопасной загрузки».

  2. Введите имя пользователя с правами администратора и его пароль.

  3. В панели «Безопасная загрузка» выберите один из описанных ниже уровней безопасности.

    • Высший уровень безопасности. Можно запускать только текущую операционную систему или подписанное ПО операционной системы, которое на данный момент признано Apple надежным. Если выбрать этот вариант, для установки ПО потребуется подключение к интернету.

    • Средний уровень безопасности. Можно запускать любую версию подписанного ПО операционной системы, когда-либо признанную Apple надежной.

    • Функции безопасности отключены. Нет никаких требований к загружаемой операционной системе.

  4. В панели «Разрешенный носитель для загрузки» выберите один из описанных ниже параметров загрузки.

    • Запретить загрузку с внешних или съемных носителей. Запрещена загрузка с любых устройств, не защищенных чипом T2, таких как USB- и Thunderbolt-накопители или внутренние накопители PCIe или SATA.

    • Разрешить загрузку с внешних или съемных носителей. Разрешена загрузка с любых устройств.

  5. Выберите «Утилита безопасной загрузки» > «Завершить Утилиту безопасной загрузки».

Переустановка macOS

В некоторых случаях может потребоваться переустановить macOS. При этом можно сохранить файлы и пользовательские настройки.

  1. Убедитесь, что компьютер подключен к интернету.

  2. В приложении «Восстановление» нажмите «Переустановить macOS Big Sur», затем нажмите «Продолжить».

    Важно! В панели дисков выберите тот диск, который сейчас используется для запуска macOS (обычно это единственный доступный диск).

  3. Следуйте инструкциям на экране.

Стирание и переустановка macOS

При стирании и переустановке macOS удаляются Ваши учетные записи пользователя, сетевые настройки, а также все файлы и папки на Вашем Mac. Прежде чем приступать к стиранию macOS, попытайтесь снова войти в систему, затем создайте резервные копии всех файлов, которые Вы хотите сохранить, скопировав их на устройство хранения. Кроме того, запишите данные сетевых настроек из раздела настроек «Сеть», чтобы было проще подключиться к интернету и другим сетям после переустановки macOS.

  1. Убедитесь, что компьютер подключен к интернету.

  2. В приложении «Восстановление» выберите «Дисковая утилита», затем нажмите «Продолжить».

  3. В боковой панели приложения «Дисковая утилита» выберите том, который нужно стереть, затем нажмите кнопку «Стереть» в панели инструментов.

  4. Введите имя тома в поле «Имя».

  5. Во всплывающем меню «Формат» выберите «APFS», затем нажмите «Стереть группу томов».

  6. Когда стирание завершится, нажмите «Готово», затем выберите «Дисковая утилита» > «Завершить Дисковую утилиту».

  7. В приложении «Восстановление» нажмите «Переустановить macOS Big Sur», нажмите «Продолжить», затем следуйте инструкциям на экране.

Просмотр журнала восстановления

Журнал восстановления содержит сообщения, записанные в режиме восстановления macOS. Вы можете указать типы сообщений, которые хотите просмотреть, а также фильтровать сообщения, чтобы найти определенные слова, или сохранить журнал восстановления в виде файла.

  1. В приложении «Восстановление» выберите «Окно» > «Журнал Восстановления».

  2. Во всплывающем меню «Уровень детализации» выберите тип отображаемых сообщений.

    • Показать только ошибки. Выберите этот параметр, чтобы отобразились только сообщения об ошибках.

    • Показать ошибки и выполнение. Выберите этот параметр, чтобы отобразились сообщения об ошибках и о процессе выполнения.

    • Показать все сообщения. Выберите этот параметр, чтобы отобразились все сообщения.

  3. Чтобы отфильтровать сообщения, введите в поле «Фильтр» текст, который нужно найти.

  4. Чтобы сохранить журнал восстановления, нажмите кнопку «Сохранить», выберите место сохранения файла, затем нажмите «Сохранить».

  5. Когда закончите просматривать журнал восстановления, нажмите кнопку «Закрыть»  в левом верхнем углу окна.

Создание файла диагностики восстановления

Если Вы отправляли отчет об ошибке в приложении «Ассистент обратной связи», Вас могут попросить предоставить файл диагностики восстановления. Вы можете создать файл и сохранить его на внешнее устройство хранения.

  1. Подключите внешнее устройство хранения к своему компьютеру Mac.

  2. Нажмите Control-Option-Shift-Command-точка (.).

  3. Экран на мгновение вспыхнет. Подождите пару минут, пока не отобразится диалоговое окно.

  4. Используя диалоговое окно, можно сохранить файл диагностики восстановления на внешнем устройстве хранения.

Завершение сеанса в режиме восстановления macOS

Компьютер Mac перезагружается автоматически при восстановлении системы из Time Machine или переустановке macOS. Однако в некоторых случаях может понадобиться вручную завершить сеанс в режиме восстановления macOS (например, если Вы хотите сделать резервные копии файлов перед переустановкой macOS или если Вы только что изменили настройку через Утилиту безопасной загрузки).

На компьютере Mac выполните одно из следующих действий.

  • Перезагрузка Mac. Выберите меню Apple  > «Перезагрузить».

  • Выбор другого загрузочного диска перед перезагрузкой Mac. Выберите меню Apple  > «Загрузочный диск», выберите загрузочный диск, затем нажмите «Перезагрузить».

  • Выключение Mac. Выберите меню Apple  > «Выключить».
