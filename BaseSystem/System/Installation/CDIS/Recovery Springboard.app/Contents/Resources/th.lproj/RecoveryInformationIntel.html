ใช้การกู้คืน macOS บน Mac ที่ใช้ Intel

การกู้คืน macOS คือระบบการกู้คืนที่มาพร้อมกับ Mac ของคุณ

คุณสามารถใช้แอพในการกู้คืน macOS บน Mac ที่ใช้ Intel เพื่อซ่อมแซมดิสก์ภายในของคุณ, ติดตั้ง macOS อีกครั้ง, กู้คืนไฟล์ของคุณจากข้อมูลสำรอง Time Machine, ตั้งค่าตัวเลือกความปลอดภัย และอื่นๆ ได้

การติดตั้ง macOS อีกครั้งต้องใช้การเชื่อมต่อกับอินเทอร์เน็ต ในการเชื่อมต่อกับอินเทอร์เน็ต คุณสามารถใช้การเชื่อมต่อเครือข่ายแบบไร้สายและแบบผ่านสายได้ ให้ดูที่เชื่อมต่อกับอินเทอร์เน็ตโดยใช้ Wi-Fi

หมายเหตุ: ถ้าคุณพยายามเชื่อมต่อกับพอร์ทัลแบบ Captive (ตัวอย่างเช่น ที่ร้านกาแฟ ห้องสมุด หรือโรงแรม) หรือเครือข่ายขององค์กร คุณอาจไม่สามารถเข้าถึงอินเทอร์เน็ตในการกู้คืน macOS ได้

แอพที่มีให้ใช้งานในการกู้คืน macOS บน Mac ที่ใช้ Intel

แอพต่อไปนี้มีให้ใช้งานในการกู้คืน macOS บน Mac ที่ใช้ Intel:

  • การกู้คืน: แอพการกู้คืนจะเปิดขึ้นเมื่อคอมพิวเตอร์ของคุณเริ่มต้นระบบในการกู้คืน macOS ให้ดูที่เริ่มต้นระบบ Mac ที่ใช้ Intel ของคุณในการกู้คืน macOS และออกจากการกู้คืน macOS

  • การกู้คืนระบบ Time Machine: กู้คืนข้อมูลของคุณจากข้อมูลสำรอง Time Machine ในแอพการกู้คืน ให้เลือก กู้คืนจาก Time Machine แล้วคลิก ดำเนินการต่อ ในการย้อนกลับไปที่แอพการกู้คืน ให้เลือก การกู้คืนระบบ Time Machine > ออกจากการกู้คืนระบบ Time Machine ให้ดูที่ใช้ Time Machine เพื่อกู้คืนระบบของคุณ

  • ติดตั้ง macOS: ติดตั้ง macOS อีกครั้งบนคอมพิวเตอร์ของคุณ ในแอพการกู้คืน ให้เลือก ติดตั้ง macOS อีกครั้ง แล้วคลิก ดำเนินการต่อ ในการย้อนกลับไปที่แอพการกู้คืน ให้เลือก ติดตั้ง macOS Big Sur > ออกจากการติดตั้ง macOS ให้ดูที่ติดตั้ง macOS อีกครั้ง

  • Safari: ในแอพการกู้คืน ให้เลือก Safari แล้วคลิก ดำเนินการต่อ ในการย้อนกลับไปที่แอพการกู้คืน ให้เลือก Safari > ออกจาก Safari ถ้าคุณเชื่อมต่อกับอินเทอร์เน็ตแล้ว คุณสามารถใช้ Safari เพื่อค้นหาข้อมูลเพิ่มเติมบนเว็บไซต์บริการช่วยเหลือของ Apple ได้ โปรดทราบว่าปลั๊กอินเบราว์เซอร์และส่วนขยายของ Safari จะถูกปิดใช้งานในการกู้คืน macOS

  • ยูทิลิตี้ดิสก์: ซ่อมแซมหรือลบดิสก์โวลุ่มของคุณ ในแอพการกู้คืน ให้เลือก ยูทิลิตี้ดิสก์ แล้วคลิก ดำเนินการต่อ ในการย้อนกลับไปที่แอพการกู้คืน ให้เลือก ยูทิลิตี้ดิสก์ > ออกจากยูทิลิตี้ดิสก์ ให้ดูที่ลบและติดตั้ง macOS อีกครั้ง

  • ยูทิลิตี้ความปลอดภัยการเริ่มต้นระบบ: ตั้งค่านโยบายความปลอดภัยสำหรับ Mac ของคุณ ในแอพการกู้คืน ให้เลือก ยูทิลิตี้ > ยูทิลิตี้ความปลอดภัยการเริ่มต้นระบบ ในการย้อนกลับไปที่แอพการกู้คืน ให้เลือก ยูทิลิตี้ความปลอดภัยการเริ่มต้นระบบ > ออกจากยูทิลิตี้ความปลอดภัยการเริ่มต้นระบบ ให้ดูที่ใช้ยูทิลิตี้ความปลอดภัยการเริ่มต้นระบบเพื่อตั้งรหัสผ่านเฟิร์มแวร์และใช้ยูทิลิตี้ความปลอดภัยการเริ่มต้นระบบเพื่อตั้งค่าตัวเลือกการเริ่มต้นระบบอย่างปลอดภัย

    ตัวเลือกที่มีให้ใช้งานในยูทิลิตี้ความปลอดภัยการเริ่มต้นระบบจะแตกต่างกันตามรุ่นของ Mac ของคุณ ให้ดูที่บทความบริการช่วยเหลือของ Apple เกี่ยวกับยูทิลิตี้ความปลอดภัยการเริ่มต้นระบบและ Mac รุ่นที่มีชิพ Apple T2 Security

  • ดิสก์เริ่มต้นระบบ: ตั้งค่าดิสก์เริ่มต้นระบบสำหรับ Mac ของคุณ เลือกเมนู Apple > ดิสก์เริ่มต้นระบบ ในการย้อนกลับไปที่แอพการกู้คืน ให้เลือก ดิสก์เริ่มต้นระบบ > ออกจากดิสก์เริ่มต้นระบบ

  • เทอร์มินัล: ในแอพการกู้คืน ให้เลือก ยูทิลิตี้ > เทอร์มินัล ในการย้อนกลับไปที่แอพการกู้คืน ให้เลือก เทอร์มินัล > ออกจากเทอร์มินัล

คุณยังสามารถใช้การกู้คืน macOS เพื่อทำสิ่งต่างๆ ต่อไปนี้ได้อีกด้วย:

  • ดูบันทึกการใช้งานการกู้คืน

  • สร้างไฟล์การวินิจฉัยการกู้คืน

เริ่มต้นระบบ Mac ที่ใช้ Intel ของคุณในการกู้คืน macOS

  1. เริ่มการทำงาน Mac ของคุณใหม่

  2. กดปุ่มแบบผสมแบบใดแบบหนึ่งต่อไปนี้ค้างไว้ในทันที จนกระทั่งคุณเห็นหน้าจอเริ่มต้นระบบ:

    • Command-R: เริ่มต้นระบบจากระบบการกู้คืน macOS ที่มาพร้อมเครื่อง ใช้การกดปุ่มแบบผสมนี้เพื่อติดตั้ง macOS เวอร์ชั่นล่าสุดที่ได้รับการติดตั้งบนระบบของคุณแล้วอีกครั้ง หรือเพื่อใช้แอพอื่นในการกู้คืน macOS

    • Option-Command-R: เริ่มต้นระบบจากการกู้คืน macOS ผ่านอินเทอร์เน็ต ใช้การกดปุ่มแบบผสมนี้เพื่อติดตั้ง macOS อีกครั้งและอัพเกรดเป็น macOS เวอร์ชั่นล่าสุดที่ใช้งานร่วมกับ Mac ของคุณได้

    • Option-Shift-Command-R: เริ่มต้นระบบจากการกู้คืน macOS ผ่านอินเทอร์เน็ต ใช้การกดปุ่มแบบผสมนี้เพื่อติดตั้ง macOS เวอร์ชั่นที่มาพร้อมกับ Mac ของคุณอีกครั้งหรือเวอร์ชั่นใกล้เคียงที่สุดที่ยังมีให้ใช้งาน

  3. ถ้าคุณเห็นแม่กุญแจ ให้ป้อนรหัสผ่านสำหรับ Mac ของคุณ

  4. ถ้าคุณมีหลายดิสก์โวลุ่มบนดิสก์ของคุณ ให้เลือกดิสก์โวลุ่มที่คุณต้องการกู้คืน แล้วคลิก ถัดไป

  5. ถ้าระบบร้องขอ ให้เลือกบัญชีผู้ดูแลระบบ แล้วคลิก ถัดไป ป้อนรหัสผ่านของบัญชี จากนั้นคลิก ดำเนินการต่อ

เชื่อมต่อกับอินเทอร์เน็ตโดยใช้ Wi-Fi

ถ้าไอคอนเมนู Wi-Fi (ที่มุมขวาบนของหน้าจอ) มีอย่างน้อยหนึ่งแถบ แสดงว่า Wi-Fi เปิดอยู่และเชื่อมต่อกับเครือข่ายไร้สายแล้ว ถ้าไอคอนเมนู Wi-Fi ไม่มีแถบ แสดงว่า Wi-Fi เปิดอยู่แต่ไม่ได้เชื่อมต่อกับเครือข่ายไร้สาย ถ้าไอคอนเมนู Wi-Fi มีเครื่องหมายทับอยู่ แสดงว่า Wi-Fi ปิดอยู่ คุณต้องเปิด Wi-Fi ก่อนจึงจะสามารถเข้าร่วมเครือข่ายไร้สายได้

บน Mac ของคุณ ให้ปฏิบัติตามวิธีใดๆ ต่อไปนี้:

  • เปิด Wi-Fi: คลิกไอคอนเมนู Wi-Fi แล้วเลือก เปิด Wi-Fi

  • ปิด Wi-Fi: คลิกไอคอนเมนู Wi-Fi แล้วเลือก ปิด Wi-Fi

  • เข้าร่วมเครือข่าย: คลิกไอคอนเมนู Wi-Fi แล้วเลือกเครือข่าย ป้อนรหัสผ่านของเครือข่ายหากระบบร้องขอ

  • เข้าร่วมเครือข่ายปิด: คลิกไอคอนเมนู Wi-Fi แล้วเลือก เข้าร่วมเครือข่ายอื่น ป้อนชื่อของเครือข่ายและรหัสผ่านของเครือข่าย หากระบบร้องขอ

ซ่อมแซมอุปกรณ์จัดเก็บข้อมูลภายในของคุณ

ถ้าคอมพิวเตอร์ของคุณมีปัญหา หรือถ้าคุณเริ่มการทำงานของคอมพิวเตอร์แล้วเห็นหน้าต่างการกู้คืน macOS คุณอาจต้องซ่อมแซมอุปกรณ์จัดเก็บข้อมูลภายในของคอมพิวเตอร์ของคุณ (เช่น ไดรฟ์โซลิดสเตท (SSD))

  1. ในแอพการกู้คืน ให้เลือก ยูทิลิตี้ดิสก์ แล้วคลิก ดำเนินการต่อ

  2. ในแอพยูทิลิตี้ดิสก์ ให้เลือก มุมมอง > แสดงอุปกรณ์ทั้งหมด

  3. ในแถบด้านข้าง ให้เลือกดิสก์โวลุ่ม จากนั้นคลิกปุ่มการช่วยเหลือเบื้องต้น

  4. ในหน้าต่างโต้ตอบการช่วยเหลือเบื้องต้น ให้คลิก สั่งทำงาน แล้วทำตามขั้นตอนบนหน้าจอ

  5. เมื่อกระบวนการการช่วยเหลือเบื้องต้นสิ้นสุด ให้คลิก เสร็จสิ้น

  6. ทำซ้ำขั้นตอนที่ 3 ถึง 5 สำหรับดิสก์โวลุ่มแต่ละดิสก์บนอุปกรณ์จัดเก็บข้อมูล

  7. ในแถบด้านข้าง ให้เลือกตัวบรรจุ จากนั้นคลิกปุ่มการช่วยเหลือเบื้องต้น

  8. ในหน้าต่างโต้ตอบการช่วยเหลือเบื้องต้น ให้คลิก สั่งทำงาน แล้วทำตามขั้นตอนบนหน้าจอ

  9. เมื่อกระบวนการการช่วยเหลือเบื้องต้นสิ้นสุด ให้คลิก เสร็จสิ้น

  10. ทำซ้ำขั้นตอนที่ 7 ถึง 9 สำหรับตัวบรรจุแต่ละตัวบนอุปกรณ์จัดเก็บข้อมูล

  11. ในแถบด้านข้าง ให้เลือกอุปกรณ์จัดเก็บข้อมูล จากนั้นคลิกปุ่มการช่วยเหลือเบื้องต้น

  12. ในหน้าต่างโต้ตอบการช่วยเหลือเบื้องต้น ให้คลิก สั่งทำงาน แล้วทำตามขั้นตอนบนหน้าจอ

  13. เมื่อกระบวนการการช่วยเหลือเบื้องต้นสิ้นสุด ให้คลิก เสร็จสิ้น

ถ้ายูทิลิตี้ดิสก์ไม่สามารถซ่อมแซมดิสก์ของคุณได้ ให้สำรองข้อมูลของคุณให้มากที่สุดเท่าที่จะทำได้ แล้วปฏิบัติตามคำแนะนำในลบและติดตั้ง macOS อีกครั้ง

ในการกลับไปยังแอพการกู้คืน macOS ให้เลือก ยูทิลิตี้ดิสก์ > ออกจากยูทิลิตี้ดิสก์

ใช้ Time Machine เพื่อกู้คืนระบบของคุณ

ถ้าคุณเคยสร้างข้อมูลสำรอง Time Machine ให้ใช้ กู้คืนจากการสำรองข้อมูลของ Time Machine เพื่อกู้คืนทุกสิ่งทุกอย่างบนคอมพิวเตอร์ของคุณกลับเป็นสภาวะก่อนหน้านี้

สิ่งสำคัญ: ใช้เฉพาะข้อมูลสำรอง Time Machine ของคุณเพื่อกู้คืนข้อมูลไปที่คอมพิวเตอร์ที่เป็นแหล่งสำรองข้อมูล ถ้าคุณต้องการถ่ายโอนข้อมูลจาก Mac เครื่องนี้ไปที่คอมพิวเตอร์เครื่องใหม่ ให้ออกจากการกู้คืน macOS แล้วเริ่มการทำงาน Mac ของคุณใหม่ จากนั้นใช้แอพผู้ช่วยการโยกย้ายใน macOS

  1. ถ้าข้อมูลสำรองของคุณอยู่บน Time Capsule โปรดตรวจสอบให้แน่ใจว่าคุณได้เชื่อมต่อกับอินเทอร์เน็ตแล้ว

  2. ในแอพการกู้คืน ให้เลือก กู้คืนจากการสำรองข้อมูลของ Time Machine แล้วคลิก ดำเนินการต่อ

  3. ในแอพการกู้คืนระบบ Time Machine ให้ทำตามขั้นตอนบนหน้าจอ

ใช้ยูทิลิตี้ความปลอดภัยการเริ่มต้นระบบเพื่อตั้งค่ารหัสผ่านเฟิร์มแวร์

การตั้งค่ารหัสผ่านเฟิร์มแวร์จะป้องกันไม่ให้ Mac ของคุณเริ่มต้นระบบจากอุปกรณ์จัดเก็บข้อมูลภายในหรือภายนอกใดๆ นอกเหนือจากดิสก์เริ่มต้นระบบที่คุณได้เลือกไว้ นอกเสียจากว่าจะมีการป้อนรหัสผ่านเฟิร์มแวร์ คุณสามารถเปิดใช้ ปิดใช้ หรือเปลี่ยนรหัสผ่านเฟิร์มแวร์ได้โดยใช้ยูทิลิตี้ความปลอดภัยการเริ่มต้นระบบ

เมื่อเปิดใช้รหัสผ่านเฟิร์มแวร์ ทุกครั้งที่คุณเริ่มต้นระบบจากการกู้คืน macOS หรือจากอุปกรณ์จัดเก็บข้อมูลที่ไม่ใช่อุปกรณ์ที่คุณได้เลือกไว้ในการตั้งค่าดิสก์เริ่มต้นระบบ คุณจะต้องป้อนรหัสผ่านเฟิร์มแวร์

หลังจากที่คุณเริ่มการทำงาน Mac ของคุณใหม่ การเปลี่ยนแปลงไปยังรหัสผ่านเฟิร์มแวร์จะมีผล

  1. ในแอพการกู้คืน ให้เลือก ยูทิลิตี้ > ยูทิลิตี้ความปลอดภัยการเริ่มต้นระบบ

  2. ป้อนชื่อผู้ใช้และรหัสผ่านของผู้ดูแลระบบหากระบบร้องขอ

  3. ปฏิบัติตามวิธีการใดๆ ต่อไปนี้:

    • เปิดใช้รหัสผ่านเฟิร์มแวร์: คลิก เปิดใช้รหัสผ่านเฟิร์มแวร์ แล้วป้อนรหัสผ่านใหม่ ป้อนอีกครั้งเพื่อตรวจสอบยืนยันรหัสผ่าน จากนั้นคลิก ตั้งรหัสผ่าน

    • ปิดใช้รหัสผ่านเฟิร์มแวร์: คลิก ปิดใช้รหัสผ่านเฟิร์มแวร์ แล้วป้อนรหัสผ่าน จากนั้นคลิก ปิดรหัสผ่าน

    • เปลี่ยนรหัสผ่านเฟิร์มแวร์: คลิก เปลี่ยนรหัสผ่าน แล้วป้อนรหัสผ่านเก่า ป้อนรหัสผ่านใหม่แล้วตรวจสอบยืนยันรหัสผ่าน จากนั้นคลิก เปลี่ยนรหัสผ่าน

  4. คลิก ออกจากยูทิลิตี้ความปลอดภัยการเริ่มต้นระบบ

ใช้ยูทิลิตี้ความปลอดภัยการเริ่มต้นระบบเพื่อตั้งค่าตัวเลือกการเริ่มต้นระบบอย่างปลอดภัย

ตัวเลือกการเริ่มต้นระบบอย่างปลอดภัยมีให้ใช้งานเฉพาะบนคอมพิวเตอร์ Mac ที่ใช้ชิพ Apple T2 Security เท่านั้น

หลังจากที่คุณเริ่มการทำงาน Mac ของคุณใหม่ การเปลี่ยนแปลงไปยังตัวเลือกการเริ่มต้นระบบอย่างปลอดภัยจะมีผล

  1. ในแอพการกู้คืน ให้เลือก ยูทิลิตี้ > ยูทิลิตี้ความปลอดภัยการเริ่มต้นระบบ

  2. ป้อนชื่อผู้ใช้และรหัสผ่านของผู้ดูแลระบบ

  3. ในบานหน้าต่างเริ่มต้นระบบอย่างปลอดภัย ให้เลือกหนึ่งในตัวเลือกความปลอดภัยต่อไปนี้:

    • ความปลอดภัยแบบเต็ม: ช่วยให้มั่นใจว่าสามารถเรียกใช้ได้เฉพาะ OS ปัจจุบันของคุณหรือซอฟต์แวร์ระบบปฏิบัติการที่มีการลงนามและ Apple เชื่อถือในขณะนี้เท่านั้น โหมดนี้ต้องใช้การเชื่อมต่อกับเครือข่ายเมื่อติดตั้งซอฟต์แวร์

    • ความปลอดภัยปานกลาง: อนุญาตให้เรียกใช้ซอฟต์แวร์ระบบปฏิบัติการเวอร์ชั่นใดๆ ที่มีการลงนามและ Apple เชื่อถือ

    • ไม่มีความปลอดภัย: ไม่ได้บังคับใช้ข้อกำหนดใดๆ บนระบบปฏิบัติการที่เริ่มต้นระบบได้

  4. ในบานหน้าต่างสื่อเริ่มต้นระบบที่อนุญาต ให้เลือกตัวเลือกการเริ่มต้นระบบแบบใดแบบหนึ่งต่อไปนี้:

    • ไม่อนุญาตให้เริ่มต้นระบบจากสื่อภายนอกหรือสื่อแบบถอดออกได้: จำกัดความสามารถในการเริ่มต้นระบบจากอุปกรณ์ใดๆ ที่ไม่ได้รับการปกป้องด้วย T2 เช่น ไดรฟ์ USB และ Thunderbolt หรือไดรฟ์ PCIe หรือ SATA ที่มีการเชื่อมต่อภายใน

    • อนุญาตให้เริ่มต้นระบบจากสื่อภายนอกหรือสื่อแบบถอดออกได้: ไม่ได้จำกัดความสามารถในการเริ่มต้นระบบจากอุปกรณ์ใดๆ

  5. เลือก ยูทิลิตี้ความปลอดภัยการเริ่มต้นระบบ > ออกจากยูทิลิตี้ความปลอดภัยการเริ่มต้นระบบ

ติดตั้ง macOS อีกครั้ง

ในบางสถานการณ์ คุณอาจต้องติดตั้ง macOS อีกครั้ง คุณสามารถติดตั้ง macOS อีกครั้งได้ในขณะที่ทำให้ไฟล์และการตั้งค่าผู้ใช้ของคุณคงเดิมอยู่เสมอ

  1. ตรวจสอบให้แน่ใจว่าคุณเชื่อมต่ออยู่กับอินเทอร์เน็ต

  2. ในแอพการกู้คืน ให้คลิก ติดตั้ง macOS Big Sur อีกครั้ง แล้วคลิก ดำเนินการต่อ

    สิ่งสำคัญ: ในบานหน้าต่างที่คุณเลือกดิสก์ ให้เลือกดิสก์ macOS ปัจจุบันของคุณ (ซึ่งส่วนใหญ่แล้ว จะมีเพียงหนึ่งดิสก์เท่านั้นที่ใช้ได้)

  3. ทำตามขั้นตอนบนหน้าจอ

ลบและติดตั้ง macOS อีกครั้ง

ถ้าคุณลบและติดตั้ง macOS อีกครั้ง บัญชีผู้ใช้ของคุณ การตั้งค่าเครือข่าย รวมทั้งไฟล์และโฟลเดอร์ทั้งหมดบน Mac ของคุณจะถูกลบ ก่อนที่คุณจะลบ macOS ให้พยายามกลับเข้าสู่ระบบของคุณ จากนั้นสำรองข้อมูลไฟล์ใดๆ ที่คุณต้องการเก็บไว้โดยคัดลอกไฟล์เหล่านั้นไปยังอุปกรณ์จัดเก็บข้อมูลอื่น แล้วจดการตั้งค่าเครือข่ายของคุณจากการตั้งค่าเครือข่ายเพื่อให้เชื่อมต่ออีกครั้งหลังจากติดตั้ง macOS อีกครั้งง่ายยิ่งขึ้น

  1. ตรวจสอบให้แน่ใจว่าคุณเชื่อมต่ออยู่กับอินเทอร์เน็ต

  2. ในแอพการกู้คืน ให้คลิก ยูทิลิตี้ดิสก์ แล้วคลิก ดำเนินการต่อ

  3. ในยูทิลิตี้ดิสก์ ให้เลือกดิสก์โวลุ่มที่คุณต้องการลบในแถบด้านข้าง จากนั้นคลิกปุ่มลบในแถบเครื่องมือ

  4. ป้อนชื่อของดิสก์โวลุ่มในช่องชื่อ

  5. คลิกเมนูรูปแบบที่แสดงขึ้น เลือก APFS แล้วคลิก ลบกลุ่มดิสก์โวลุ่ม

  6. เมื่อกระบวนการลบสิ้นสุดแล้ว ให้คลิก เสร็จสิ้น แล้วเลือก ยูทิลิตี้ดิสก์ > ออกจากยูทิลิตี้ดิสก์

  7. ในแอพการกู้คืน ให้คลิก ติดตั้ง macOS Big Sur อีกครั้ง แล้วคลิก ดำเนินการต่อ จากนั้นทำตามขั้นตอนบนหน้าจอ

ดูบันทึกการใช้งานการกู้คืน

บันทึกการใช้งานการกู้คืนประกอบด้วยข้อความที่เก็บบันทึกการใช้งานโดยการกู้คืน macOS คุณสามารถระบุประเภทของข้อความที่จะดู ฟิลเตอร์ข้อความเพื่อค้นหาบางข้อความ และบันทึกบันทึกการใช้งานการกู้คืนไปยังไฟล์ได้

  1. ในแอพการกู้คืน ให้เลือก หน้าต่าง > บันทึกการใช้งานการกู้คืน

  2. คลิกเมนูระดับรายละเอียดที่แสดงขึ้น จากนั้นเลือกข้อความที่จะแสดง:

    • แสดงเฉพาะข้อผิดพลาด: เลือกตัวเลือกนี้เพื่อแสดงข้อความแสดงข้อผิดพลาดเท่านั้น

    • แสดงข้อผิดพลาดและความคืบหน้า: เลือกตัวเลือกนี้เพื่อแสดงข้อความแสดงข้อผิดพลาดและข้อความแสดงความคืบหน้า

    • แสดงบันทึกการใช้งานทั้งหมด: เลือกตัวเลือกนี้เพื่อแสดงข้อความทั้งหมด

  3. ในการฟิลเตอร์ข้อความ ให้ป้อนข้อความที่คุณต้องการค้นหาในช่องฟิลเตอร์

  4. ในการบันทึกบันทึกการใช้งานการกู้คืน ให้คลิกปุ่มบันทึก แล้วเลือกตำแหน่งสำหรับไฟล์ จากนั้นคลิก บันทึก

  5. เมื่อคุณดูบันทึกการใช้งานการกู้คืนเสร็จแล้ว ให้คลิกปุ่มปิด ที่มุมซ้ายบนสุดของหน้าต่าง

สร้างไฟล์การวินิจฉัยการกู้คืน

ถ้าคุณส่งรายงานปัญหาโดยใช้ผู้ช่วยผลตอบรับ คุณอาจถูกขอให้ส่งไฟล์การวินิจฉัยการกู้คืนคุณสามารถสร้างไฟล์และบันทึกไปยังอุปกรณ์จัดเก็บภายนอกได้

  1. เชื่อมต่ออุปกรณ์จัดเก็บข้อมูลภายนอกเข้ากับ Mac ของคุณ

  2. กดปุ่ม Control-Option-Shift-Command-จุด

  3. หลังจากที่หน้าจอกะพริบ ให้รอสักครู่จนกว่าหน้าต่างโต้ตอบจะแสดงขึ้น

  4. ใช้หน้าต่างโต้ตอบเพื่อบันทึกไฟล์การวินิจฉัยการกู้คืนไปยังตำแหน่งบนอุปกรณ์จัดเก็บข้อมูลภายนอกของคุณ

ออกจากการกู้คืน macOS

Mac ของคุณจะเริ่มการทำงานเครื่องใหม่โดยอัตโนมัติในขณะที่กู้คืนระบบของคุณจาก Time Machine หรือติดตั้ง macOS อีกครั้ง ในกรณีอื่นๆ คุณจะต้องออกจากการกู้คืน macOS ด้วยตัวเอง (ตัวอย่างเช่น ถ้าคุณต้องการสำรองข้อมูลไฟล์ของคุณก่อนที่จะติดตั้ง macOS อีกครั้ง หรือถ้าคุณเพิ่งเปลี่ยนการตั้งค่าโดยใช้ยูทิลิตี้ความปลอดภัยการเริ่มต้นระบบ)

บน Mac ของคุณ ให้ปฏิบัติตามวิธีใดวิธีหนึ่งต่อไปนี้:

  • เริ่มการทำงาน Mac ของคุณใหม่: เลือกเมนู Apple > เริ่มการทำงานเครื่องใหม่

  • เลือกดิสก์เริ่มต้นระบบดิสก์อื่นก่อนเริ่มการทำงาน Mac ของคุณใหม่: เลือกเมนู Apple > ดิสก์เริ่มต้นระบบ เลือกดิสก์เริ่มต้นระบบ จากนั้นคลิก เริ่มการทำงานใหม่

  • ปิดเครื่อง Mac ของคุณ: เลือกเมนู Apple > ปิดเครื่อง
