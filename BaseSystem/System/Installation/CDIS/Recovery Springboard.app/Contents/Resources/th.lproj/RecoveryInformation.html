ใช้การกู้คืน macOS บน Mac ที่ใช้ Apple Silicon

การกู้คืน macOS คือระบบการกู้คืนที่มาพร้อมกับ Mac ของคุณ

คุณสามารถใช้แอพในการกู้คืน macOS บน Mac ที่ใช้ Apple Silicon เพื่อซ่อมแซมดิสก์ภายในของคุณ, ติดตั้ง macOS อีกครั้ง, กู้คืนไฟล์ของคุณจากข้อมูลสำรอง Time Machine, ตั้งค่านโยบายความปลอดภัยสำหรับดิสก์โวลุ่มอื่น, ถ่ายโอนไฟล์ระหว่างคอมพิวเตอร์ Mac สองเครื่อง, เริ่มต้นระบบในเซฟโหมด และอื่นๆ ได้

การติดตั้ง macOS อีกครั้งต้องใช้การเชื่อมต่อกับอินเทอร์เน็ต ในการเชื่อมต่อกับอินเทอร์เน็ต คุณสามารถใช้การเชื่อมต่อเครือข่ายแบบไร้สายและแบบผ่านสายได้ ให้ดูที่เชื่อมต่อกับอินเทอร์เน็ตโดยใช้ Wi-Fi

หมายเหตุ: ถ้าคุณพยายามเชื่อมต่อกับพอร์ทัลแบบ Captive (ตัวอย่างเช่น ที่ร้านกาแฟ ห้องสมุด หรือโรงแรม) หรือเครือข่ายขององค์กร คุณอาจไม่สามารถเข้าถึงอินเทอร์เน็ตในการกู้คืน macOS ได้

แอพที่มีให้ใช้งานในการกู้คืน macOS บน Mac ที่ใช้ Apple Silicon

แอพต่อไปนี้มีให้ใช้งานในการกู้คืน macOS บน Mac ที่ใช้ Apple Silicon:

  • การกู้คืน: แอพการกู้คืนทำให้สามารถเข้าถึงแอพเพิ่มเติมในการกู้คืน macOS ได้ ให้ดูที่เริ่มต้นระบบคอมพิวเตอร์ของคุณในการกู้คืน macOS, เริ่มต้นระบบคอมพิวเตอร์ของคุณในการกู้คืน macOS และจับคู่อุปกรณ์ป้อนข้อมูลบลูทูธ และออกจากการกู้คืน macOS

  • การกู้คืนระบบ Time Machine: กู้คืนข้อมูลของคุณจากข้อมูลสำรอง Time Machine ในหน้าต่างแอพการกู้คืน ให้เลือก กู้คืนจาก Time Machine แล้วคลิก ดำเนินการต่อ ในการย้อนกลับไปที่แอพการกู้คืน ให้เลือก การกู้คืนระบบ Time Machine > ออกจากการกู้คืนระบบ Time Machine ให้ดูที่ใช้ Time Machine เพื่อกู้คืนระบบของคุณ

  • ติดตั้ง macOS: ติดตั้ง macOS อีกครั้งบนคอมพิวเตอร์ของคุณ ในหน้าต่างแอพการกู้คืน ให้เลือก ติดตั้ง macOS อีกครั้ง แล้วคลิก ดำเนินการต่อ ในการย้อนกลับไปที่แอพการกู้คืน ให้เลือก ติดตั้ง macOS Big Sur > ออกจากการติดตั้ง macOS ให้ดูที่ติดตั้ง macOS อีกครั้ง

  • Safari: ในหน้าต่างแอพการกู้คืน ให้เลือก Safari แล้วคลิก ดำเนินการต่อ ในการย้อนกลับไปที่แอพการกู้คืน ให้เลือก Safari > ออกจาก Safari ถ้าคุณเชื่อมต่อกับอินเทอร์เน็ตแล้ว คุณสามารถใช้ Safari เพื่อค้นหาข้อมูลเพิ่มเติมบนเว็บไซต์บริการช่วยเหลือของ Apple ได้ โปรดทราบว่าปลั๊กอินเบราว์เซอร์และส่วนขยายของ Safari จะถูกปิดใช้งานในการกู้คืน macOS

  • ยูทิลิตี้ดิสก์: ซ่อมแซมหรือลบดิสก์โวลุ่มของคุณ ในหน้าต่างแอพการกู้คืน ให้เลือก ยูทิลิตี้ดิสก์ แล้วคลิก ดำเนินการต่อ ในการย้อนกลับไปที่แอพการกู้คืน ให้เลือก ยูทิลิตี้ดิสก์ > ออกจากยูทิลิตี้ดิสก์ ให้ดูที่ลบและติดตั้ง macOS อีกครั้งและซ่อมแซมอุปกรณ์จัดเก็บข้อมูลภายในของคุณ

  • ยูทิลิตี้ความปลอดภัยการเริ่มต้นระบบ: ตั้งค่านโยบายความปลอดภัยสำหรับ Mac ของคุณ ในแอพการกู้คืน ให้เลือก ยูทิลิตี้ > ยูทิลิตี้ความปลอดภัยการเริ่มต้นระบบ ในการย้อนกลับไปที่แอพการกู้คืน ให้เลือก ยูทิลิตี้ความปลอดภัยการเริ่มต้นระบบ > ออกจากยูทิลิตี้ความปลอดภัยการเริ่มต้นระบบ ให้ดูที่ตั้งค่านโยบายความเป็นส่วนตัว

  • เทอร์มินัล: เปลี่ยนการตั้งค่าผ่านบรรทัดคำสั่ง ในหน้าต่างแอพการกู้คืน ให้เลือก ยูทิลิตี้ > เทอร์มินัล ในการย้อนกลับไปที่แอพการกู้คืน ให้เลือก เทอร์มินัล > ออกจากเทอร์มินัล

  • แชร์ดิสก์: แชร์ดิสก์โวลุ่มของ Mac ที่เริ่มต้นระบบในการกู้คืน macOS ในแอพการกู้คืน ให้เลือก ยูทิลิตี้ > แชร์ดิสก์ ให้ดูที่ใช้การแชร์ดิสก์เพื่อถ่ายโอนไฟล์ระหว่างคอมพิวเตอร์ Mac สองเครื่อง

  • ดิสก์เริ่มต้นระบบ: ตั้งค่าดิสก์เริ่มต้นระบบสำหรับ Mac เลือกเมนู Apple > ดิสก์เริ่มต้นระบบ ในการออกจากแอพ ให้เลือก ดิสก์เริ่มต้นระบบ > ออกจากดิสก์เริ่มต้นระบบ

คุณยังสามารถใช้การกู้คืน macOS เพื่อดำเนินการทำงานต่อไปนี้ได้อีกด้วย:

  • เริ่มต้นระบบคอมพิวเตอร์ของคุณในเซฟโหมด

  • ตั้งค่าดิสก์โวลุ่มเริ่มต้นระบบตามค่าเริ่มต้นและเริ่มการทำงานคอมพิวเตอร์ของคุณใหม่

  • ดูบันทึกการใช้งานการกู้คืน

  • สร้างไฟล์การวินิจฉัยการกู้คืน

เริ่มต้นระบบคอมพิวเตอร์ของคุณในการกู้คืน macOS

  1. เลือกเมนู Apple > ปิดเครื่อง

  2. กดปุ่มเปิด/ปิดค้างไว้บน Mac ของคุณ จนกระทั่งคุณเห็น “กำลังโหลดตัวเลือกการเริ่มต้นระบบ”

  3. คลิก ตัวเลือก แล้วคลิก ดำเนินการต่อ

  4. ป้อนรหัสผ่านสำหรับบัญชีผู้ดูแลระบบ

เริ่มต้นระบบคอมพิวเตอร์ของคุณในการกู้คืน macOS และจับคู่อุปกรณ์ป้อนข้อมูลบลูทูธ

ปฏิบัติตามคำแนะนำเหล่านี้หากคุณจำเป็นต้องจับคู่แป้นพิมพ์ แทร็คแพด หรือเมาส์แบบบลูทูธกับ Mac ของคุณเมื่อคุณเริ่มต้นระบบในการกู้คืน macOS

  1. เลือกเมนู Apple > ปิดเครื่อง

  2. กดปุ่มเปิด/ปิดค้างไว้บน Mac ของคุณ จนกระทั่งคุณเห็น “กำลังโหลดตัวเลือกการเริ่มต้นระบบ”

  3. เมื่อดิสก์โวลุ่มระบบและปุ่มตัวเลือกแสดงขึ้น ให้กดปุ่มเปิด/ปิดสามครั้ง

    หมายเหตุ: การกดปุ่มเปิด/ปิดแต่ละครั้งจะต้องทำภายใน 1 วินาทีหลังจากการกดครั้งก่อน ถ้าคุณกดปุ่มเปิด/ปิดเร็วเกินไป ปุ่มจะไม่ตอบสนอง เนื่องจากการกดอย่างรวดเร็วมีไว้เพื่อจุดประสงค์อื่น เช่น VoiceOver

  4. ใช้ผู้ช่วยตั้งค่าบลูทูธเพื่อจับคู่อุปกรณ์เสริมบลูทูธกับ Mac ของคุณ

    หมายเหตุ: คุณจะไม่เห็นผู้ช่วยตั้งค่าบลูทูธหากคุณมีอุปกรณ์ป้อนข้อมูลที่เชื่อมต่ออยู่แล้ว

  5. ป้อนรหัสผ่านของบัญชีผู้ดูแลระบบ

เริ่มต้นระบบคอมพิวเตอร์ของคุณในเซฟโหมด

  1. เลือกเมนู Apple > ปิดเครื่อง

  2. กดปุ่มเปิด/ปิดค้างไว้บน Mac ของคุณ จนกระทั่งคุณเห็น “กำลังโหลดตัวเลือกการเริ่มต้นระบบ”

  3. เลือกดิสก์โวลุ่ม

  4. กดปุ่ม Shift ค้างไว้ แล้วคลิก ดำเนินการต่อในเซฟโหมด

    คอมพิวเตอร์จะเริ่มการทำงานเครื่องใหม่โดยอัตโนมัติ

ตั้งค่าดิสก์โวลุ่มเริ่มต้นระบบตามค่าเริ่มต้นและเริ่มการทำงานคอมพิวเตอร์ของคุณใหม่

  1. เลือกเมนู Apple > ปิดเครื่อง

  2. กดปุ่มเปิด/ปิดค้างไว้บน Mac ของคุณ จนกระทั่งคุณเห็น “กำลังโหลดตัวเลือกการเริ่มต้นระบบ”

  3. เลือกดิสก์โวลุ่ม

  4. กดปุ่ม Option ค้างไว้ แล้วคลิก ใช้เสมอ

    คอมพิวเตอร์จะเริ่มการทำงานเครื่องใหม่โดยอัตโนมัติ

เชื่อมต่อกับอินเทอร์เน็ตโดยใช้ Wi-Fi

ถ้าไอคอนเมนู Wi-Fi (ที่มุมขวาบนของหน้าจอ) มีอย่างน้อยหนึ่งแถบ แสดงว่า Wi-Fi เปิดอยู่และเชื่อมต่อกับเครือข่ายไร้สายแล้ว ถ้าไอคอนเมนู Wi-Fi ไม่มีแถบ แสดงว่า Wi-Fi เปิดอยู่แต่ไม่ได้เชื่อมต่อกับเครือข่ายไร้สาย ถ้าไอคอนเมนู Wi-Fi มีเครื่องหมายทับอยู่ แสดงว่า Wi-Fi ปิดอยู่ คุณต้องเปิด Wi-Fi ก่อนจึงจะสามารถเข้าร่วมเครือข่ายไร้สายได้

บน Mac ของคุณ ให้ปฏิบัติตามวิธีใดๆ ต่อไปนี้:

  • เปิด Wi-Fi: คลิกไอคอนเมนู Wi-Fi แล้วเลือก เปิด Wi-Fi

  • ปิด Wi-Fi: คลิกไอคอนเมนู Wi-Fi แล้วเลือก ปิด Wi-Fi

  • เข้าร่วมเครือข่าย: คลิกไอคอนเมนู Wi-Fi แล้วเลือกเครือข่าย ถ้าระบบร้องขอ ให้ป้อนรหัสผ่านของเครือข่าย

  • เข้าร่วมเครือข่ายปิด: คลิกไอคอนเมนู Wi-Fi แล้วเลือก เข้าร่วมเครือข่ายอื่น ป้อนชื่อของเครือข่ายและรหัสผ่านของเครือข่าย หากระบบร้องขอ

ซ่อมแซมอุปกรณ์จัดเก็บข้อมูลภายในของคุณ

ถ้าคุณมีปัญหาเกี่ยวกับคอมพิวเตอร์ของคุณ หรือถ้าคุณเปิดคอมพิวเตอร์แล้วเห็นหน้าต่างการกู้คืน macOS แสดงว่าคุณอาจต้องซ่อมแซมอุปกรณ์จัดเก็บข้อมูลภายในของคอมพิวเตอร์ของคุณ (เช่น ไดรฟ์โซลิดสเตท (SSD))

  1. ในหน้าต่างแอพการกู้คืน ให้เลือก ยูทิลิตี้ดิสก์ แล้วคลิก ดำเนินการต่อ

  2. ในแอพยูทิลิตี้ดิสก์ ให้เลือก มุมมอง > แสดงอุปกรณ์ทั้งหมด

  3. ในแถบด้านข้าง ให้เลือก ดิสก์โวลุ่ม จากนั้นคลิกปุ่มการช่วยเหลือเบื้องต้น

  4. ในหน้าต่างโต้ตอบการช่วยเหลือเบื้องต้น ให้คลิก สั่งทำงาน แล้วทำตามขั้นตอนบนหน้าจอ

  5. เมื่อกระบวนการการช่วยเหลือเบื้องต้นสิ้นสุด ให้คลิก เสร็จสิ้น

  6. ทำซ้ำขั้นตอนที่ 3 ถึง 5 สำหรับดิสก์โวลุ่มแต่ละดิสก์บนอุปกรณ์จัดเก็บข้อมูล

  7. ในแถบด้านข้าง ให้เลือกตัวบรรจุ จากนั้นคลิกปุ่มการช่วยเหลือเบื้องต้น

  8. ในหน้าต่างโต้ตอบการช่วยเหลือเบื้องต้น ให้คลิก สั่งทำงาน แล้วทำตามขั้นตอนบนหน้าจอ

  9. เมื่อกระบวนการการช่วยเหลือเบื้องต้นสิ้นสุด ให้คลิก เสร็จสิ้น

  10. ทำซ้ำขั้นตอนที่ 7 ถึง 9 สำหรับตัวบรรจุแต่ละตัวบนอุปกรณ์จัดเก็บข้อมูล

  11. ในแถบด้านข้าง ให้เลือกอุปกรณ์จัดเก็บข้อมูล จากนั้นคลิกปุ่มการช่วยเหลือเบื้องต้น

  12. ในหน้าต่างโต้ตอบการช่วยเหลือเบื้องต้น ให้คลิก สั่งทำงาน แล้วทำตามขั้นตอนบนหน้าจอ

  13. เมื่อกระบวนการการช่วยเหลือเบื้องต้นสิ้นสุด ให้คลิก เสร็จสิ้น

ถ้ายูทิลิตี้ดิสก์ไม่สามารถซ่อมแซมดิสก์โวลุ่มของคุณได้ ให้สำรองข้อมูลของคุณให้มากที่สุดเท่าที่ทำได้ แล้วทำตามขั้นตอนในลบและติดตั้ง macOS อีกครั้ง

ในการย้อนกลับไปที่แอพการกู้คืน ให้เลือก ยูทิลิตี้ดิสก์ > ออกจากยูทิลิตี้ดิสก์

ใช้ Time Machine เพื่อกู้คืนระบบของคุณ

ถ้าคุณเคยสร้างข้อมูลสำรอง Time Machine ให้ใช้กู้คืนจากการสำรองข้อมูลของ Time Machine เพื่อกู้คืนทุกสิ่งทุกอย่างบนคอมพิวเตอร์ของคุณกลับเป็นสภาวะก่อนหน้านี้

สิ่งสำคัญ: ใช้เฉพาะข้อมูลสำรอง Time Machine ของคุณเพื่อกู้คืนข้อมูลไปที่คอมพิวเตอร์ที่เป็นแหล่งสำรองข้อมูล ถ้าคุณต้องการถ่ายโอนข้อมูลจาก Mac เครื่องนี้ไปที่คอมพิวเตอร์เครื่องใหม่ ให้ออกจากการกู้คืน macOS แล้วเริ่มการทำงาน Mac ของคุณใหม่ จากนั้นใช้แอพผู้ช่วยการโยกย้ายใน macOS

  1. ถ้าข้อมูลสำรองของคุณอยู่บน Time Capsule โปรดตรวจสอบให้แน่ใจว่าคุณได้เชื่อมต่อกับอินเทอร์เน็ตแล้ว

  2. ในแอพการกู้คืน ให้เลือก กู้คืนจากการสำรองข้อมูลของ Time Machine แล้วคลิก ดำเนินการต่อ

  3. ในแอพการกู้คืนระบบ Time Machine ให้ทำตามขั้นตอนบนหน้าจอ

ติดตั้ง macOS อีกครั้ง

ในบางสถานการณ์ คุณอาจต้องติดตั้ง macOS อีกครั้ง คุณสามารถติดตั้ง macOS อีกครั้งได้ในขณะที่ทำให้ไฟล์และการตั้งค่าผู้ใช้ของคุณคงเดิมอยู่เสมอ

  1. ตรวจสอบให้แน่ใจว่าคุณเชื่อมต่ออยู่กับอินเทอร์เน็ต

  2. ในแอพการกู้คืน ให้คลิก ติดตั้ง macOS Big Sur อีกครั้ง แล้วคลิก ดำเนินการต่อ

    สิ่งสำคัญ: ในบานหน้าต่างที่คุณเลือกดิสก์โวลุ่ม ให้เลือกดิสก์โวลุ่ม macOS ปัจจุบันของคุณ (ซึ่งส่วนใหญ่แล้ว จะมีดิสก์โวลุ่มให้เลือกเพียงดิสก์เดียว)

  3. ทำตามขั้นตอนบนหน้าจอ

ลบและติดตั้ง macOS อีกครั้ง

ถ้าคุณลบและติดตั้ง macOS อีกครั้ง บัญชีผู้ใช้ของคุณ การตั้งค่าเครือข่าย รวมทั้งไฟล์และโฟลเดอร์ทั้งหมดบน Mac ของคุณจะถูกลบ ก่อนที่คุณจะลบ macOS ให้พยายามกลับเข้าสู่ระบบของคุณ จากนั้นสำรองข้อมูลไฟล์ใดๆ ที่คุณต้องการเก็บไว้โดยคัดลอกไฟล์เหล่านั้นลงในอุปกรณ์จัดเก็บข้อมูลอื่น แล้วจดการตั้งค่าเครือข่ายของคุณจากการตั้งค่าเครือข่ายเพื่อทำให้การเชื่อมต่ออีกครั้งหลังจากติดตั้ง macOS อีกครั้งง่ายยิ่งขึ้น

  1. ตรวจสอบให้แน่ใจว่าคุณเชื่อมต่ออยู่กับอินเทอร์เน็ต

  2. ในแอพการกู้คืน ให้คลิก ยูทิลิตี้ดิสก์ แล้วคลิก ดำเนินการต่อ

  3. ในยูทิลิตี้ดิสก์ ให้เลือกดิสก์โวลุ่มที่คุณต้องการลบในแถบด้านข้าง จากนั้นคลิกปุ่มลบในแถบเครื่องมือ

  4. ป้อนชื่อของดิสก์โวลุ่มในช่องชื่อ

  5. คลิกเมนูรูปแบบที่แสดงขึ้น เลือก APFS แล้วคลิก ลบกลุ่มดิสก์โวลุ่ม

  6. เมื่อกระบวนการลบสิ้นสุดแล้ว ให้คลิก เสร็จสิ้น แล้วเลือก ยูทิลิตี้ดิสก์ > ออกจากยูทิลิตี้ดิสก์

  7. ในแอพการกู้คืน ให้คลิก ติดตั้ง macOS Big Sur อีกครั้ง แล้วคลิก ดำเนินการต่อ จากนั้นทำตามขั้นตอนบนหน้าจอ

ตั้งค่านโยบายความปลอดภัย

  1. ในแอพการกู้คืน ให้เลือก ยูทิลิตี้ > ยูทิลิตี้ความปลอดภัยการเริ่มต้นระบบ

  2. เลือกระบบที่คุณต้องการใช้เพื่อตั้งค่านโยบายความเป็นส่วนตัว

  3. ถ้าดิสก์โวลุ่มถูกเข้ารหัสด้วย FileVault ให้คลิก ปลดล็อค แล้วป้อนรหัสผ่าน จากนั้นคลิก ปลดล็อค

  4. คลิก นโยบายความปลอดภัย

  5. ทบทวนตัวเลือกความปลอดภัยแบบใดแบบหนึ่งต่อไปนี้:

    • ความปลอดภัยแบบเต็ม: ช่วยให้มั่นใจว่าสามารถเรียกใช้ได้เฉพาะ OS ปัจจุบันของคุณหรือซอฟต์แวร์ระบบปฏิบัติการที่มีการลงนามและ Apple เชื่อถือในขณะนี้เท่านั้น โหมดนี้ต้องใช้การเชื่อมต่อกับเครือข่ายเมื่อติดตั้งซอฟต์แวร์

    • ความปลอดภัยแบบลดลง: อนุญาตให้เรียกใช้ซอฟต์แวร์ระบบปฏิบัติการเวอร์ชั่นใดๆ ที่มีการลงนามและ Apple เชื่อถือ

  6. ถ้าจำเป็น ให้เลือก ความปลอดภัยแบบลดลง แล้วป้อนชื่อผู้ใช้และรหัสผ่านผู้ดูแลระบบของคุณ จากนั้นปฏิบัติตามวิธีใดวิธีหนึ่งต่อไปนี้:

    • เลือกกล่องกาเครื่องหมาย “อนุญาตการจัดการผู้ใช้ของส่วนขยายเคอร์เนลจากนักพัฒนาที่ระบุ” เพื่ออนุญาตให้ติดตั้งซอฟต์แวร์ที่ใช้ส่วนขยายเคอร์เนลรุ่นดั้งเดิม

    • เลือกกล่องกาเครื่องหมาย “อนุญาตการจัดการระยะไกลของส่วนขยายเคอร์เนลและรายการอัพเดทซอฟต์แวร์อัตโนมัติ” เพื่ออนุญาตการจัดการระยะไกลของส่วนขยายเคอร์เนลรุ่นดั้งเดิมและรายการอัพเดทซอฟต์แวร์โดยใช้โซลูชั่นการจัดการอุปกรณ์เคลื่อนที่ (MDM)

  7. คลิก ตกลง

  8. เริ่มการทำงาน Mac ของคุณใหม่เพื่อให้การเปลี่ยนแปลงมีผล

ใช้การแชร์ดิสก์เพื่อถ่ายโอนไฟล์ระหว่างคอมพิวเตอร์ Mac สองเครื่อง

คุณสามารถถ่ายโอนไฟล์ระหว่าง Mac ที่เริ่มต้นระบบเป็นการกู้คืน macOS กับ Mac เครื่องอื่นได้ด้วยการแชร์ดิสก์โวลุ่มและการเชื่อมต่อคอมพิวเตอร์สองเครื่องด้วยสาย USB, USB-C หรือ Thunderbolt

  1. เชื่อมต่อคอมพิวเตอร์สองเครื่องด้วยสาย USB, USB-C หรือ Thunderbolt

  2. ในแอพการกู้คืน ให้เลือก ยูทิลิตี้ > แชร์ดิสก์

  3. เลือกดิสก์โวลุ่มที่คุณต้องการแชร์ จากนั้นคลิก เริ่มแชร์

  4. บน Mac เครื่องอื่น ให้เปิดหน้าต่าง Finder จากนั้นคลิก เครือข่าย ในแถบด้านข้างที่อยู่ใต้ตำแหน่งที่ตั้ง

  5. ในหน้าต่างเครือข่าย ให้คลิกสองครั้งที่ Mac ที่มีดิสก์ที่แชร์ แล้วคลิก เชื่อมต่อเป็น เลือก ผู้ใช้ทั่วไป ในหน้าต่างเชื่อมต่อเป็น จากนั้นคลิก เชื่อมต่อ

  6. ถ่ายโอนไฟล์

  7. เมื่อคุณถ่ายโอนไฟล์เสร็จแล้ว ให้เลิกต่อเชื่อมกับดิสก์โวลุ่มบน Mac เครื่องอื่น

ดูบันทึกการใช้งานการกู้คืน

บันทึกการใช้งานการกู้คืนประกอบด้วยข้อความที่เก็บบันทึกการใช้งานโดยการกู้คืน macOS คุณสามารถระบุประเภทของข้อความที่จะแสดง ฟิลเตอร์ข้อความเพื่อค้นหาบางข้อความ และบันทึกบันทึกการใช้งานการกู้คืนไปยังไฟล์ได้

  1. ในแอพการกู้คืน ให้เลือก หน้าต่าง > บันทึกการใช้งานการกู้คืน

  2. คลิกเมนูระดับรายละเอียดที่แสดงขึ้น จากนั้นเลือกข้อความที่จะแสดง:

    • แสดงเฉพาะข้อผิดพลาด: เลือกตัวเลือกนี้เพื่อแสดงข้อความแสดงข้อผิดพลาดเท่านั้น

    • แสดงข้อผิดพลาดและความคืบหน้า: เลือกตัวเลือกนี้เพื่อแสดงข้อความแสดงข้อผิดพลาดและข้อความแสดงความคืบหน้า

    • แสดงบันทึกการใช้งานทั้งหมด: เลือกตัวเลือกนี้เพื่อแสดงข้อความทั้งหมด

  3. ในการฟิลเตอร์ข้อความ ให้ป้อนข้อความที่คุณต้องการค้นหาในช่องฟิลเตอร์

  4. ในการบันทึกบันทึกการใช้งานการกู้คืน ให้คลิกปุ่มบันทึก แล้วเลือกตำแหน่งสำหรับไฟล์ จากนั้นคลิก บันทึก

  5. เมื่อคุณดูบันทึกการใช้งานการกู้คืนเสร็จแล้ว ให้คลิกปุ่มปิด ที่มุมซ้ายบนสุดของหน้าต่าง

สร้างไฟล์การวินิจฉัยการกู้คืน

ถ้าคุณส่งรายงานปัญหาโดยใช้ผู้ช่วยผลตอบรับ คุณอาจถูกขอให้ส่งไฟล์การวินิจฉัยการกู้คืน คุณสามารถสร้างไฟล์และบันทึกไปยังอุปกรณ์จัดเก็บภายนอกได้

  1. เชื่อมต่ออุปกรณ์จัดเก็บข้อมูลภายนอกเข้ากับ Mac ของคุณ

  2. กดปุ่ม Control-Option-Shift-Command-จุด

  3. หลังจากที่หน้าจอกะพริบ ให้รอสักครู่จนกว่าหน้าต่างโต้ตอบจะแสดงขึ้น

  4. ใช้หน้าต่างโต้ตอบเพื่อบันทึกไฟล์การวินิจฉัยการกู้คืนไปยังตำแหน่งบนอุปกรณ์จัดเก็บข้อมูลภายนอกของคุณ

ออกจากการกู้คืน macOS

Mac ของคุณจะเริ่มการทำงานเครื่องใหม่โดยอัตโนมัติขณะกู้คืนระบบของคุณจาก Time Machine หรือติดตั้ง macOS อีกครั้ง ในกรณีอื่น คุณต้องออกจากการกู้คืน macOS ด้วยตนเอง (ตัวอย่างเช่น ถ้าคุณต้องการสำรองข้อมูลไฟล์ของคุณก่อนติดตั้ง macOS อีกครั้ง หรือหากคุณเพิ่งเปลี่ยนการตั้งค่าโดยใช้ยูทิลิตี้ความปลอดภัยการเริ่มต้นระบบ)

บน Mac ของคุณ ให้ปฏิบัติตามวิธีใดวิธีหนึ่งต่อไปนี้:

  • เริ่มการทำงาน Mac ของคุณใหม่: เลือกเมนู Apple > เริ่มการทำงานเครื่องใหม่

  • เลือกดิสก์เริ่มต้นระบบดิสก์อื่นก่อนเริ่มการทำงาน Mac ของคุณใหม่: เลือกเมนู Apple > ดิสก์เริ่มต้นระบบ แล้วเลือกดิสก์เริ่มต้นระบบ จากนั้นคลิก เริ่มการทำงานใหม่

  • ปิดเครื่อง Mac ของคุณ: เลือกเมนู Apple > ปิดเครื่อง
